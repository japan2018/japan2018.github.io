[Python] Use linter to reduce code review costs

May 23, 2017 Python ATOM lint 

-**lint**: A program that checks source code more strictly than a compiler or interpreter
  • linter: One that points out the bad part of writing code that gets caught by lint in the editor
  • Code formatter: One that automatically formats code according to lint

In software development projects with multiple people, coding conventions may be set in order to improve the quality and reduce the review cost by writing in a unified manner. Since it is difficult to share it by considering the specification of the detailed writing too much, it is better to use linter published by the language developer or a large company for the part to be unified at least. For example, the source code can be kept in a minimum unified state simply by specifying that the code should be formatted before committing so that there is no linter error display, so the review cost is concentrated on the processing content, not the writing style. be able to.

Even for small-scale development at the individual level, you can easily adjust the writing style from the state where the indentation and line breaks are different depending on the file or method, and even if the source code inherited from another person is different from your own writing style At a minimum, you can bring it into a readable state.

Most major languages have linters and code formatters, and IDEs and text editors have formatting features by default, or you can easily extend the formatting features, so if you haven’t done so already Let’s try.

The following describes how to use Python’s linter formatter with atom.

Environment

  • Mac OS 10.12.4
  • Installed python 3.x installed by homebrew
$python3 -V
Python 3.6.1

atom package to use

  • atom-beautify -Automatically formats source code in various languages
  • linter
  • linter-python-pep8

Procedure

Installation of pep8 and autopep8

$ pip3 install pep8
$ pip3 install autopep8

install atom package

apm install atom-beautify
apm install linter
apm install linter-python-pep8
