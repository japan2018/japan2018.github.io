[Python] Switch and use multiple models with M5StickV

Nov 2, 2019 Python M5stickV 

This article is the 3rd day of M5 Stack Advent Calendar 2019

Currently, for those who are blind, there are various apps (https://www.tokyo-itcenter.com/700link/sm-iphon4.html#iphone-10-00200).

However, there seems to be no one that can be used, for example, to register (the pattern of) the plastic bottle of tea that you want to drink and recognize it. To check what you have registered, you can use hardware such as Oakham My Eye 2, but the price is hundreds of thousands. It is expensive as yen, and it seems that it is difficult for students to use it.

So, I made a thing to register and recognize an image using Get it for about 3000 yenM5StickV.

The recognition model uses the V-Training web service provided by M5Stack. V-Training creates a model that can distinguish up to 10 types of objects, so I created multiple models and switched them to use.

There are still some troubles such as V-Training itself and the process of making voice, ・Allow padding with few images when recognizing a flat object such as a logo (added) ・Simplify the process of creating voice I would like to reduce the hassle with such things.

The program and how to create it is Publishing on GitHub.
