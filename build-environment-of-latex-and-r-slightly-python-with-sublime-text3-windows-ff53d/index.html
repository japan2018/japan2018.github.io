# Sublime Text3 makes LaTeX and R (a little Python) easier to use

Introduction

For your own notes. Unexpectedly, I thought that I wouldn’t see the site about TeX in Sublime, I thought I would write about the environment I set up (Friend J’s Maru Pakuri).

In SublimeText3, I will describe a series of setting procedures to make LaTeX and R (a little Python) easy to use. Make sure to restart Sublime accordingly.

*This article is for Windows users.

Software to prepare

  • Sublime Text 3
  • TeXLive 2019
  • SumatraPDF

Setup steps

First, open Sublime Text and press Ctrl+Shift+P to open the Command Palette, Enter install Package installer and Palette and confirm. This completes the installation of Package Control.

Next, install the following 3 packages.

  • LaTeXTools
  • R-Box
  • SendCode

LaTeXTools

Open the Command Palette by pressing Ctrl+Shift+P as in the procedure of Package Control. Next, enter Package Control: Install Package and Palette and confirm. Enter LaTeXTools in the Palette that appears, and if you decide, LaTeXTools will be installed. (When the installation is completed, a message will be displayed on Sublime Text.)

Next, set LaTeX Tools. Sublime Text’s upper menu, From the second [Preferences] from the right [Preferences]-> [Package Settings] -> [LaTeXTools] -> [Settings-User] Select.

If \Sublime Text 3\Packages\User\LaTeXTools.sublime-settings does not exist,

You do not currently have a personalized
LaTeXTools.sublime-setting file.

Create a copy of the default settings file in your
User directory?

The message above will be displayed. Click [OK].

Press Ctrl + F for LaTeXTools.sublime-settings displayed on Sublime Text, Enter builder_settings in the search field that appears and find “builder_settings” in (398th line).

And From (line 398) "builder_settings" Add the following contents between (line 406) // General settings:.


398 "builder_settings" :{
        "command" :["latexmk", "-cd",
                "-e", "$latex ='uplatex %O -no-guess-input-enc -kanji=utf8 -interaction=nonstopmode -synctex=1 %S'",
                "-e", "$biber ='biber %O --bblencoding=utf8 -u -U --output_safechars %B'",
                "-e", "$bibtex ='upbibtex %O %B'",
                "-e", "$makeindex ='upmendex %O -o %D %S'",
                "-e", "$dvipdf ='dvipdfmx %O -o %D %S'",
                "-f", "-%E", "-norc", "-gg", "-pdfdvi"],
406 // General settings:

After adding the above contents, save with Ctrl + S.

Check if you can compile LaTeX. Of the following contents After creating text.tex on Sublime Text, press Ctrl + B. At the first time, you will be asked how to build, so select Build With LaTeX-Traditional. You can also select the build method from [Tool]->[Build With...].

After the building operation (the operation status is displayed in the bar at the bottom of the screen) is completed, SumatraPDF is automatically started, and if the contents are displayed normally, the confirmation is complete.

Make it convenient through environment variables!

It seems to be the most confusing part of this article.

If you double-click the text on SumatraPDF by passing environment variables, You can open the source code of the corresponding part with Sublime Text.

Press the Windows key + R and type control to open the control panel. Enter environment variable in the search field in the upper right corner, and from the list displayed, Select [System-Edit system environment variables] -> Environment variables (N). Select [Path] from the [System environment variables] list and double-click or press [Edit].Add the following Path from [Edit environment variable name] -> [New(N)].

C:\Program Files\Sublime Text 3\

Start SumatraPDF, click the list in the upper left, and select [Settings]->[Options(O)]. Reverse order search command line settings are as follows.

sublime_text %f:%l

This completes environment variable setting.

To confirm, open test.tex with SumatraPDf, Then double-click the [1 Introduction] part in the text above SumatraPDF, It is successful if you open the (10th line) \section{Introduction} in Sublime Text.

**! ! ! Note! ! ! ** Please do not put a * space in the folder name where the .tex file is located! ! ** Example You can find out by checking the above with test.tex placed in the hoge fuga folder.

R-Box

Similar to the procedure of LaTeX Tool, Install R-Box from Package Control: Install Package.

There is no need to make any special settings for R-Box.

SendCode

Similar to the procedure of LaTeX Tool, Install SendCode from Package Control: Install Package.

Set Send Code. This is also the same as the procedure of LaTeX Tool settings, Select [Preferences] -> [Package Settings] -> [SendCode] -> [Settings].

After that, the setting will be displayed. Enter the following contents on the right side of the setting column.

{
    "prog": "r",
    "r": {
        "prog": "r"
    },
}

By adding the above contents, you can send the R code written on Sublime Text to RGui by Ctrl + Enter. However, please note that it is necessary to start RGui beforehand.

Check if it works properly.

curve(dnorm(x) ,-5,5)

If you press Ctrl+Enter with this line entered, it will be sent to RGui. If the graph of normal distribution appears on RGui, it is successful.

Syntax configuration for Python

About Python indentation

In the Python code, indentation has a role to distinguish blocks, Mainly there are the styles of 4 half-width spaces, Tab, 2 half-width spaces. (although I rarely see two spaces) In Sublime Text’s Python syntax settings, if you set Tab = 4 spaces, Don’t confuse using tab with the 4 space code. (Because confusion is not recommended)

First, open any .py file in Sublime Text as you can do anything. (Make notepad properly and open it with the extension changed to .py)

After that, do the same as for settings such as LaTeX Tool. Select [Preferences] -> [Settings-Syntax Specific].

After that, the setting will be displayed. Enter the following contents on the right side of the setting column.

// These settings override both User and Default settings for the Python syntax
{
    "tab_size": 4,
    "translate_tabs_to_spaces": true
}

If you press Tab and 4 half-width spaces are output, the setting is complete.

What seems to be useful if put in Sublime

This completes the basic environment configuration, Other packages that may be useful are listed below.

  • Japanize Sublime Text will be translated into Japanese. I thought it would be a problem if the detailed settings were in English, so I put it in.

  • ConvertToUTF8 This is a package that seems to solve problems related to character codes. If you use UTF-8 as a standard, I think that there is something wrong with it.

  • Alignment It is a very convenient package.

For example, the following code.

int x = 0;
double y = 1;
int z = 10;

By selecting the entire range of this code and typing Ctrl + Alt + A

int x = 0;
double y = 1;
int z = 10;

It adjusts the indent based on =. Alignment setting Settings-User

{
"alignment_space_chars": ["=","-"],
}

By setting as above, - will also be adjusted according to the standard, It will be convenient to use R. Also, set the Alignment setting Key Bindings-User. The command can be changed by setting. (I have changed to Ctrl + Shift + A.)
