Install dlib for Python (Windows)

Jan 5, 2020 Python Image Processing OpenCV Beginner dlib 

I'm stumbled upon soberly, so I will explain with a lot of images.

There may be unnecessary steps, but I will introduce the method that will definitely be tried after various trials. I would like to gradually edit to reduce waste.

0. Assumption

The environment is as follows. ãƒ»Windows 10 Home 64 bit

1. Install Python

You can download it from the following. https://www.python.org/downloads/windows/

This time, install “Python 3.8.1 64bit”

On the installation screen, check “Add Python 3.8 to PATH”. (You can save the trouble of passing a pass)

After installation, enter the following command in the command prompt.

python -V

If the version is displayed as shown below, the installation is complete.

Python 3.8.1

If you see no response or not installed, we recommend restarting the command prompt! If that doesn’t work, restart your PC and try again!

2. Install CMake

Download the executable file of Visual Studio 2019 Community from the following and start installation https://visualstudio.microsoft.com/en/downloads/

In this screen that appears later, select “Desktop development with C++”. Select if “C++ CMake tool for Winsows” is not checked.

After the installation is complete, start Visual Studio 2019. Select “Continue without code”.

Launch Command Prompt from “Tools> Command Line> Developer Command Prompt”

Enter the following in the started command prompt.

cmake -version

If the version is displayed as shown below, the installation is complete.

cmake version 3.15.19101501-MSVC_2

If there is no response or it is not installed, we recommend restarting your PC!

3. Install dlib

Enter the following in the command prompt you just started.

pip install dlib

Once the installation is complete, you should be able to use dlib! If not, we recommend restarting the command prompt or PC!
