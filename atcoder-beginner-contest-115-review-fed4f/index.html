#C problem

First, select trees that are close to each other, sort them by length, and subtract the shortest one from the longest one of the K trees. Simply h[i+k-1]-h[i] You can ask in. Also note that the smallest of n-k+1 ways is output as the answer.

n,k=map(int,input().split())
h=[int(input()) for i in range(n)]
h.sort()
ans=[]
for i in range(n-k+1):
    ans.append(h[i+k-1]-h[i])
print(min(ans))

#D problem

I looked at the relationship between the burger with level k and the burger with level k+1, but I couldn’t get it to work. I didn’t understand it, so I looked at the answer, but I found that the habit of doing puzzles properly was the cause.

Below, I will explain while paying attention to **where I should have known.

First, if the thickness of the level i burger is $a_i$ and the total number of patties is $p_i$, then $a_0=1,a_i=2a_{i-1}+3,a_0=1,p_i=2p_{i- 1}+1(i \geqq 1)$ holds. So it turns out that it is possible to ask for this $a_i$ and $p_i$ up to Level N Burger**.

Here, it is assumed that f(N-1,Y) holds for any Y with N>=1, and the number of patty layers included in the X layer from the bottom of the level N burger is f(N,X). Indicates. (**Because I’ve seen before, the technique of keeping what you want as a function to clarify the subject ** is remembered.)

Therefore, if we consider **whether a recurrence formula holds between f(N,X) and f(N-1,Y), it becomes as follows (this **recurrence pattern is frequent ** So don’t forget it.) (1) When $x=1$, then $f(N,X)=0$ ② When $1 \leqq x \leqq 1+a_{N-1}$, then $f(N,X)=f(N-1,X-1)$ ③ When $x=2+a_{N-1}$ → $f(N,X)=p_{N-1}+1$ ④ When $2+a_{N-1} <x \leqq 2+2a_{N-1}$ → $f(N,X)=p_{N-1}+1+f(N-1,X- 2-a_{N-1})$ ⑤ When $x=3+2a_{N-1}$ → $f(N,X)=2p_{N-1}+1$

Also, the above recurrence formulas ① to ⑤ are considered in order from the layers below the figure below, but I feel that it is not difficult to consider. You can implement this as it is, but be careful** when N=0, X=1 and f(N,X)=1.

n_,x_=map(int,input().split())
a=[1]
p=[1]
for i in range(n_):
    a.append(2*a[-1]+3)
    p.append(2*p[-1]+1)

def f(n,x):
    if n==0:
        return 1
    elif x==1:
        return 0
    elif x<=1+a[n-1]:
        return f(n-1,x-1)
    elif x==2+a[n-1]:
        return p[n-1]+1
    elif x<=2+2*a[n-1]:
        return p[n-1]+1+f(n-1,x-2-a[n-1])
    else:
        return 2*p[n-1]+1

print(f(n_,x_))
