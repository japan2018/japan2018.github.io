#Summary of modules that help install WebDriver with Python

When scraping with Selenium etc., it may not work if the version of WebDriver is different, so I searched for a module that arbitrarily matches the version and summarized it.

#For Chrome

chromedriver-autoinstaller

  • License: MIT
  • Compatible OS: Widows/Mac/Linux

How to install

pip install chromedriver-autoinstaller

How to use

Add to the code to do the following

from selenium import webdriver
import chromedriver_autoinstaller

chromedriver_autoinstaller.install()

chromedriver-py

It seems to automatically download and install the latest chromedriver binary version

  • License: MIT
  • Compatible OS: Windows/Linux/macOS

How to install

 pip install chromedriver-py

How to use

Adding from chromedriver_py import binary_path'' substitutes Chrome driver path for binary_path’’ It seems that it will be automatically updated to the latest version

from selenium import webdriver
from chromedriver_py import binary_path # this will get you the path variable

driver = webdriver.Chrome(executable_path=binary_path)

driver.get("http://www.python.org")
assert "Python" in driver.title

Add import geckodriver_autoinstaller'' to the code you want to use and add geckodriver_autoinstaller.install()’’ Add

pygeckodriver

A script that automatically checks every hour for geckodriver updates and installs them

  • License: Apache-2.0
  • Compatible OS: windows/Mac/linux

Install

pip install pygeckodriver

How to use

Adding pygeckodriver import geckodriver_path'' puts geckodriver path in geckodriver_path’’

from selenium import webdriver
from pygeckodriver import geckodriver_path

bs = webdriver.Firefox(executable_path=geckodriver_path)
bs.get('https://www.pypi.org')

#Comprehensive

Pyderman

  • License: MIT
  • Compatible OS: Windows/Linux/macOS
  • Python 3.5 or above
  • Compatible browser Google Chrome (chromeDriver) Firefox (geckoDriver) PhantomJS Opera (operaDriver) Edge (edgeDriver)

install

pip install pyderman

How to use (Firefox)

import pyderman as driver
# Install FIrefox driver
path = driver.install(browser=driver.firefox)
print('The installed geckodriver driver path is: %s' %path)

Explanation

Download the driver for the specified browser and return the save path

def install(browser=None, file_directory='./lib/', verbose=True, chmod=True, overwrite=False, version='latest', filename=None, return_info=False):
"""
Downloads the given browser driver, and returns the path it was saved to.
:param browser: The Driver to download. Pass as `pyderman.chrome/firefox`. Default Chrome.
:param file_directory: The directory to save the driver.
:param verbose: If printouts are okay during downloading.
:param chmod: If True, attempt to make the downloaded driver executable.
:param overwrite: If true, overwrite existing drivers of the same version.
:param version: The version to download. Default'latest'.
:param filename: The filename to save the driver to.Defaults to driver-specific.
:param return_info: If True, return an Object with more download information.
:return: The absolute path of the downloaded driver, or None if something failed.
"""

  • License: Apache-2.0
  • Compatible OS: Windows/Linux/macOS
  • Compatible browser Google Chrome (chromeDriver) Firefox (geckoDriver) Opera (operaDriver) Edge (edgeDriver)

install

pip install webdriver_manager

How to use

Until now, every time I updated the driver, I downloaded it manually and in Chrome, I set the path etc. with the following feeling.

webdriver.Chrome('/home/user/drivers/chromedriver')
ChromeDriverManager(path=custom_path).install()

Instead

from webdriver_manager.chrome import ChromeDriverManager

webdriver.Chrome(ChromeDriverManager().install())

It will look like this Firefox

from webdriver_manager.firefox import GeckoDriverManager

driver = webdriver.Firefox(executable_path=GeckoDriverManager().install())

IE

from webdriver_manager.microsoft import IEDriverManager

driver = webdriver.Ie(IEDriverManager().install())

Opera

from webdriver_manager.opera import OperaDriverManager

driver = webdriver.Opera(executable_path=OperaDriverManager().install())

Webdriver Controller

CLI tool to manage web drivers?

  • License: MIT
  • Compatible OS: Linux/macOS
  • Python 3.5+
  • Compatible browser Google Chrome (chromeDriver) Firefox (geckoDriver)

install

pip install webdriver_controller

How to use

webdriver_controller [-h] {download,cleanup,list,start} ...
Webdriver controller
positional arguments:
  {download,cleanup,list,start}
    download Webdriver binary download
    cleanup Webdriver binaries removed
    list Display the list of downloaded Webdrivers
    start Selenium start

optional arguments:
  -h, --help Show this help message

webdriverdownloader

  • License: MIT
  • Compatible OS: Windows/Linux/macOS
  • python: 2.7.16 or 3.8
  • Compatible browser Google Chrome (chromeDriver) Firefox (geckoDriver) Opera (operaDriver)

install

pip install webdriverdownloader

How to use

Firefox

from webdriverdownloader import GeckoDriverDownloader
# Install latest version
gdd = GeckoDriverDownloader()
# Install v0.20.0 version
gdd.download_and_install("v0.20.0")

Chrome

from webdriverdownloader import ChromeDriverDownloader
# Install latest version
gdd = ChromeDriverDownloader()
# Install v0.20.0 version
gdd.download_and_install("v0.20.0")

Opera (based on Chromium)

from webdriverdownloader import OperaChromiumDriverDownloader
# Install latest version
gdd = OperaChromiumDriverDownloader()
# Install v0.20.0 version
gdd.download_and_install("v0.20.0")

Usage as CLI tool

webdriverdownloader browser: version

If you install Chrome 2.38, Firefox latest version, opera v2.35

webdriverdownloader chrome:2.38 firefox opera:v.2.35
