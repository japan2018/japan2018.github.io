How to insert NEologd dictionary in Windows relatively easily-system dictionary

Feb 11, 2020 Python Windows R mecab neologd 

#Introduction

I used WSL (Windows Subsystem for Linux)+Ubuntu to install the NEologd dictionary for MeCab morphological analysis, but it was relatively easy to install with git for Windows and 7-zip.

Since I wrote about user dictionaries in the previous article â†“, this time it is the system dictionary. https://qiita.com/zincjp/items/c61c441426b9482b5a48

#environment Windows10 64bit Language: Japanese MeCab 0.996-32bit

What to introduce

git for Windows 2.20.1 64-bit 7-Zip 18.06 64-bit #Introduction procedure

pass PATH through MeCab

Set the environment variables and pass PATH to the following folder that contains the MeCab executable file. C:\Program Files (x86)\MeCab\bin ##7-Install zip and set environment variables ###7-zip installation The NEologd dictionary I downloaded is compressed in xz format, so I use 7-zip to extract it. Download and install 7-zip 64bit from the following site. https://sevenzip.osdn.jp/ ###7-pass PATH to zip Set the following in environment variables C:\Program Files\7-Zip

Install git for Windows

Install git for Windows 64bit referring to the following site https://qiita.com/taiponrock/items/632c117220e57d555099

Download NEologd dictionary

Download dictionary from git (create clone)

Start the command prompt as an administrator and move to the work folder with the following command

cd %homepath%

Then download the NEologd dictionary with the following command

git clone --depth 1 https://github.com/neologd/mecab-ipadic-neologd.git

Confirm NEologd dictionary file

From the command prompt, use the following command to move to C:\Users(user name)\mecab-ipadic-neologd\seed and check the file.

cd mecab-ipadic-neologd\seed
dir

Extract NEologd dictionary file

Extract these .csv.xz files with 7-zip with the following command.

7z X *.xz

Copy the ###.csv file Copy the csv file to the dic\ipadic folder of Mecab with the following command. However, since the mecab-user-dict-seed.(date).csv file is large and frequently updated, I want to use it as a user dictionary. Deleted in this work.

copy *.csv "c:\Program Files (x86)\MeCab\dic\ipadic"
del "c:\Program Files (x86)\MeCab\dic\ipadic\mecab-user-dict-seed.*"

Convert SHIFT-JIS format file to UTF-8 format.

Change the character code of all .csv files in c:\Program Files (x86)\MeCab\dic\ipadic and unk.def file. Convert from SHIFT-JIS (line feed CR+LF) to UTF-8 (line feed LF) with an editor.

I converted the character code with EmEditor (https://jp.emeditor.com/).

This is because the csv file of the neologd dictionary is UTF-8 format, the csv file of the ipa dictionary included by default is SHIFT-JIS format, If you compile a dictionary under the condition that different codes are mixed, some morphemes will be displayed with “??”

Compile the dictionary file

###SHIFT-Create JIS dictionary Create a SHIFT-JIS system dictionary in the ipadic folder with the following command.

cd "c:\Program Files (x86)\MeCab\dic\ipadic"
mecab-dict-index -f utf-8 -t shift-jis

Creating UTF-8 dictionary

Continuing on, UTF-8 systems such as Python require a UTF-8 dictionary, so use the following command to create a UTF-8 system dictionary. Copy all the files in the ipadic folder to the newly created ipadic-UTF8 folder with the following command.

mkdir "c:\Program Files (x86)\MeCab\dic\ipadic-UTF8"
copy * "c:\Program Files (x86)\MeCab\dic\ipadic-UTF8"

Create a UTF-8 system dictionary from the files in c:\Program Files (x86)\MeCab\dic\ipadic-UTF8 with the following command.

cd "c:\Program Files (x86)\MeCab\dic\ipadic-UTF8"
mecab-dict-index -f utf-8 -t utf-8
