#Which is better, python or R for time series data?

  • Both python and R have abundant library packages of data analysis, and the speed from paper to implementation is fast
  • There are many similar things that both python and R can do visualization and functions
  • Python is advantageous for deep learning
  • Machine learning is led by python with excellent scikit-learn
  • R also has attractive auxiliary libraries such as tidymodels and recipes
  • R is the lead because there is only a “statistical language” for “easiness of execution” of tests and visualization
  • Stan for MCMC is available in R or python
  • ** R leads the time series modeling **

From the above, it is recommended to select R language if you want to learn time series.

#Methods and types of time series forecast

The methods used for forecasting time series data, including time series modeling, are (* arbitrarily) divided into 4 groups.

  1. Autoregressive system
  2. State space system
  3. Machine learning regression
  4. Deep learning

###1. Autoregressive system AR, ARMA, ARIMA, SARIMA, ARIMAX, SARIMAX, ARCH, GARCH, VAR

It is a group of time-series modeling methods that have been around for a long time, and they are not exhausted enough to be used in modern times.

Time-series data is expressed as a function by decomposing it into several components such as “trend”, “autocorrelation (cycle)”, “season (regular cycle)”, and “error”. Method.

↑ Break it down like this (the figure shows the output of prophet).

In order to decompose the data into these components, it is necessary to perform engineering such as auto-regression, unit root test, d-difference, AIC model selection, etc.

It may seem difficult, but in modern times, autoregressive modeling is basically automated, so the hurdle is low. However, theoretical knowledge is also necessary to “understand/consider” the modeling results.

If you don’t know “what kind of theory you use to calculate,” ・Whether the model is incorrect as a result of automatic determination ・Is it data that can be modeled by autoregression in the first place? You will overlook important parts such as.

###2. State space system

We will model time-series data based on the idea that it can be obtained by “probabilistically adding values” to numerical values called “states”. Consider two mathematical formulas, “observation equation” and “state equation”. When modeling the temperature in Japan, we can first think of a periodic waveform of hot summers and cold winters.

However, in the summer of one year, there was a lot of rain and the average temperature dropped.

When a periodic waveform is treated as a Japanese temperature model, such changes cannot be predicted, but If the waveform is corrected from the record “whether it rained”, it may reflect the decrease in temperature. It is a method of making a model based on the idea.

“Is it raining?” here is made up as an equation representing the “state.” Since the “state” is considered to affect the data, there is a merit that it is easier to explain the change by checking the value of the state when considering “why the value became that”.

The term “filter” in the state space model refers to the algorithm used to create and correct “states”. In the process of making a state space model, the probability density function corresponding to the posterior probability becomes complicated, and it becomes necessary to generate random numbers by MCMC in order to estimate its parameters.

###3. Machine learning regression

There is a regression model as a machine learning method. It is a method of predicting numerical values by using surrounding explanatory variables and past time values instead of time series modeling. Methods such as SVM (SVR) in the old days, GBDT system, LightGNM, etc. have been active in the competition of data analysis recently, and they have a wide range of applications such as “solving classification problems, fast calculation, handling multivariate”. It feels like it’s being used casually.

If you know time series modeling, theory, and it is easy to formulate, select time series modeling. If you feel it is multivariate or difficult to formulate, try regression using another machine learning model.
