Dictionary type

With the dictionary type, you can manage multiple elements by pairing key and value. To express a dictionary in Python, write it with a stove separator like {key:value}. It is used when you want to store a value for a specified key, such as “apple is 100” or “banana is 200”, as in the following example.

dic_data = {'apple':100,'banana':200}
print(dic_data['banana']) # Output: 200
dic_data["banana"] # Output: 200

If you want to add an element of the dictionary target dictionary [key] = element

dic_data ["orange"] = 300
print(dic_data) # Output: {'apple': 100,'banana': 200,'orange': 300}

When you want to delete a dictionary element, del target dictionary [key]

del dic_data["apple"]
print(dic_data) # Output: {'banana': 200,'orange': 300}

[Addition] Use .keys() or .values() when you want to extract only keys and values as a list. The acquired list is acquired as dict_keys and dict_values, respectively.

dic_data = {'apple':100,'banana':200}
dic_data.keys() # Output: dict_keys(['apple','banana'])
dic_data.values() # Output: dict_values([100, 200])

Similarly, in the following for sentence, it is retrieved using keys() and values(), but using the for sentence, it is possible to retrieve all keys and values in order rather than a list.

##function A mechanism that brings together a series of processes. As a way of writing, if there is a function name and argument after def, describe the argument name in (). This argument becomes an input, and the result is returned by return (return value), and this is the output.

def calc_multi(a,b):

calc_multi(3,10) # Output: 30

anonymous function

Some functions are called anonymous functions, which can be used to simplify code. To write an anonymous function, use the keyword lambda. Anonymous functions are often used when you want to perform a function on an element such as a list.

(lambda a,b: a*b)(3,10) # Output: 30

Here, lambda a, b: is the part corresponding to the function name (a, b). The basic way to write an anonymous function is to describe the processing of that function (here, return a * b) by separating it with :. ##map If you want to do something with an element, use the map function. A function that uses a function as an argument or a return value, and is used when you want to process or manipulate each element.

def calc_double(x):
