What is #echonest

The Echo Nest, a database of music information, seems to have been acquired by Spotify in 2014. The music information API is open to the public. http://the.echonest.com/

#echonest public library http://developer.echonest.com/client_libraries.html

Create a python file based on #Examples Display an artist similar to Duke Ellington instead of Bikini Kill.

vi echotest.py

#!/usr/bin/python
# -*- coding: utf-8 -*-

from pyechonest import config
config.ECHO_NEST_API_KEY="YOURAPIKEY"

from pyechonest import artist
de = artist.Artist('duke ellington')
print "Artists similar to: %s:" %(de.name,)
for similar_artist in de.similar: print "\t%s" %(similar_artist.name,)

#result

Shows where the #Duke Ellington artist is from and the tempo and length of the song Perdido

#!/usr/bin/python
# -*- coding: utf-8 -*-

from pyechonest import config
config.ECHO_NEST_API_KEY="YOURAPIKEY"

from pyechonest import song
de_results = song.search(artist='duke ellington', title='perdido')
perdido = de_results[0]
print perdido.artist_location
print'tempo:',perdido.audio_summary['tempo'],'duration:',perdido.audio_summary['duration']
