Mar 17, 2020 Python Selenium scraping chromedriver Mercari 

## Trigger

Do you sell anything you no longer need using Mercari? I also sell unnecessary books in Mercari, but since they are old reference books and study books, they are hard to sell. .. ..

Mercari will offer a “sellable price” when listing. However, if I set it to a high price, I can’t sell it, and if it’s too cheap, I feel like I’ve lost something.

Before setting the price of an exhibit, I conduct a search to find out what the real market price is. (Maybe it’s not just me…)

However, I was wondering if this process could be automated and somehow difficult. So I scraped Mercari using python and investigated how much it could sell!

Before implementation

What I wanted to do

  1. Get the price of an item sold for a search word in Mercari (scraping)
  2. Make scraped data visually easy to understand (Graphing)
  3. For a large number of search words, rotate 1 and 2 above (Batch processing)

Design overview

What we did to implement the above is described below. You just have to write the source code to make it happen.

  1. Put the product list to search by Mercari in csv
  2. Load 1 csv file with python
  3. Scraping with the loaded search word
  4. If the search results span multiple pages, scrape all pages
  5. After scraping, output the result to another csv file and save it Create a graph based on the csv file created in 1.5
  6. Save csv file and graph (.jpg) file for graph creation
  7. After that, loop 1 to 7 until scraping all product list csv

Implementation

import pandas as pd
from selenium import webdriverimport matplotlib.pyplot as plt
import time
import csv
import os


def search_mercari(search_words):

    # Evacuate temporarily because the search word is used as the directory name as it is
    org_search_words = search_words

    # If there are multiple search words, format them so that they are connected with "+".
    words = search_words.split("_")
    search_words = words[0]
    for i in range(1, len(words)):
        search_words = search_words + "+" + words[i]

    # URL to search by Mercari
    url = "https://www.mercari.com/jp/search/?keyword=" + search_words

    # Open browser
    #If chromeriver.exe is in the same directory as this python file,
    # Argument can be empty
    browser = webdriver.Chrome()

    # Sleep for 5 seconds because it takes time to start up
    time.sleep(5)

    #Display page
    page = 1
    # Create list
    columns = ["Name", "Price", "Sold", "Url"]
    # Specify array name
    df = pd.DataFrame(columns=columns)

    # Execute
    try:
        while(True):
            # Search in browser
            browser.get(url)
            # Get all HTML for each product
            posts = browser.find_elements_by_css_selector(".items-box")
            # Show how many pages are getting
            print(str(page) + "Getting page")

            # Get the name and price of each item, whether it has been purchased, and the URL
            for post in posts:
                # Product name
                title = post.find_element_by_css_selector(
                    "h3.items-box-name").text

                # Get price
                price = post.find_element_by_css_selector(
                    ".items-box-price").text
                #Excessive things will be acquired, so delete
                price = price.replace("\", "")
                price = price.replace(",", "")

                #Set to 1 if purchased and 0 if not purchased
                sold = 0
                if (len(post.find_elements_by_css_selector(".item-sold-out-badge"))> 0):
                    sold = 1

                # Get product URL
                Url = post.find_element_by_css_selector(
                    "a").get_attribute("href")

                # Add scraped information to the list
                se = pd.Series([title, price, sold, Url], columns)
                df = df.append(se, columns)

            # Increment page number
            page += 1
            #Get the URL to go to the next page
            url = browser.find_element_by_css_selector(
                "li.pager-next .pager-cell a").get_attribute("href")
            print("Moving to next page ...")
    except:
        print("Next page is nothing.")

    # Save the last obtained data as CSV
    filename = "mercari_scraping_" + org_search_words + ".csv"
    df.to_csv(org_search_words + "/" + filename, encoding="utf-8-sig")
    print("Finish!")


def make_graph(search_words, except_words, max_price, bins):
    #Open CSV file
    df = pd.read_csv(search_words + "/" +
                     "mercari_scraping_" + search_words + ".csv")

    # Excludes "except_words" in "Name"
    if(len(except_words) != 0):
        exc_words = except_words.split("_")
        for i in range(len(exc_words)):
            df = df[df["Name"].str.contains(exc_words[i]) == False]
    else:
        pass

    # Show only purchased (sold=1) products
    dfSold = df[df["Sold"] == 1]

    #Display only products with a price of 1500 yen or less
    dfSold = dfSold[dfSold["Price"] <max_price]

    # Specify column name "Price" "Number at that price" "Percent"
    columns = ["Price", "Num", "Percent"]

    # Specify array name
    all_num = len(dfSold)
    num = 0
    dfPercent = pd.DataFrame(columns=columns)

    for i in range(int(max_price/bins)):

        MIN = i * bins-1
        MAX = (i + 1) * bins

        List only the values between #MIN and MAX, and use len() to get the number
        df0 = dfSold[dfSold["Price"]> MIN]
        df0 = df0[df0["Price"] <MAX]
        sold = len(df0)

        # I want to make it cumulative, so add this number to num
        num += sold

        #Calculate percentage here
        percent = num / all_num * 100

        #The price is the median of MIN and MAX
        price = (MIN + MAX + 1) / 2
        se = pd.Series([price, num, percent], columns)
        dfPercent = dfPercent.append(se, columns)

    # Save to CSV
    filename = "mercari_histgram_" + search_words + ".csv"
    dfPercent.to_csv(search_words + "/" + filename, encoding="utf-8-sig")

    #Graph drawing
    """
    :param kind: specify the kind of graph
    :param y: specify the y-axis value
    :param bins: specify graph width
    :param alpha: Transparency of graph (0: transparent ~ 1: dark)
    :param figsize: specify the size of the graph
    :param color: Graph color
    :param secondary_y: Use 2 axes (when true)
    """
    ax1 = dfSold.plot(kind="hist", y="Price", bins=25,
                      secondary_y=True, alpha=0.9)
    dfPercent.plot(kind="area", x="Price", y=[
        "Percent"], alpha=0.5, ax=ax1, figsize=(20, 10), color="k")
    plt.savefig(search_words + "/" + "mercari_histgram_" +
                search_words + ".jpg")


def read_csv():
    # Read the csv file of the list for Mercari search
    with open("mercari_search.csv", encoding="utf-8") as f:

        # Prepare an empty list for storing search words
        csv_lists = []
        # counter to check which line of csv file to read
        counter = 0

        # read csv file line by line
        reader = csv.reader(f)
        for row in reader:
            counter += 1
            csv_lists.append(row)
            try:
                #Search word check
                # If empty, display an error message and exit
                if(len(row[0]) == 0):
                    print("File Error: No search word -> "+"mercari_search.csv "+ str(counter) + "line")
                    break
            except IndexError:
                If the # line is empty, display an error message and exit
                print("File Error: There is a problem with the CSV file. Please reduce the space between lines.")
                break
            try:
                if(len(row[2]) == 0):
                    # Check the highest value when drawing a graph
                    # If empty, display an error message and exit
                    print("File Error: No amount set -> "+
                          "mercari_search.csv "+ str(counter) + "line")
                    break
                else:
                    try:
                        int(row[2])
                    except ValueError:
                        # If the value does not appear as a number, display an error message and exit
                        print("File Error: Please enter a number for the amount -> "+
                              "mercari_search.csv "+ str(counter) + "line")
                        break
            except IndexError:
                # If the amount itself is not written, display an error message and exit.
                print("File Error: No amount set -> "+
                      "mercari_search.csv "+ str(counter) + "line")
                break
            try:
                if(len(row[3]) == 0):
                    # Check the highest value when drawing a graph
                    # If empty, display an error message and exit
                    print("File Error: Graph width is not set -> "+
                          "mercari_search.csv "+ str(counter) + "line")
                    break
                else:
                    try:
                        int(row[3])
                    except ValueError:
                        # If the value does not appear as a number, display an error message and exit
                        print("File Error: Please enter a number for the graph width ->" +
                              "mercari_search.csv "+ str(counter) + "line")
                        break
            except IndexError:
                # If the amount itself is not written, display an error message and exit.
                print("File Error: Graph width is not set -> "+
                      "mercari_search.csv "+ str(counter) + "line")
                break
        return csv_lists

# ------------------------------------------------- ----- #


# 0. Mercari search Prepare a box to store the list read from CSV file
"""
Read list from CSV file for search
:param csv_lists[i][0]: Search word
:param csv_lists[i][1]: Words to exclude from search results
:param csv_lists[i][2]: Maximum amount when displaying graph
:param csv_lists[i][3]: Graph width (bin)
"""
csv_lists = read_csv()

# Batch processing
for i in range(len(csv_lists)):
    # 1. Create directory
    os.mkdir(csv_lists[i][0])
    # 2. Scraping process
    search_mercari(csv_lists[i][0])
    # 3. Graph drawing
    make_graph(csv_lists[i][0], csv_lists[i][1],
               int(csv_lists[i][2]), int(csv_lists[i][3]))

How to use

1. Preparation of search word list

In mercari_search.csv, enter words you want to search, words you want to exclude, maximum amount, and graph width.

  • Search word (required): Enter the word you want to search on Mercari -If there are multiple search words, please connect with half-width underbar (_) -Example: Pokemon _ game -Be careful not to put a space between the word and the search word (the operation is not guaranteed).
  • Exclusion word (arbitrary): Enter when you want to exclude the products with that word when drawing a graph -If you do not enter the exclusion word, you do not need to enter anything.
  • Maximum amount (required): Enter the maximum amount that will be the horizontal axis when drawing the graph -Please enter in single-byte numbers (integers).
  • Graph width (required): Enter the graph width when drawing the graph -Please enter in single-byte numbers (integers).

When separating each word, it is a csv file, so separate each word with a comma (,).

Example:

Clock, digital, 10000,100
Wallet, cow, 3000,100
Pokemon_Game,Card_CD,3000,100
PC, 15000,500

2. Perform scraping

Confirm that chromedriver.exe and mercari_search.py exist in the same directory as this source (scraping_batch.py) file, and execute the following command.

python scraping_batch.py

During execution, the number of scraping pages is displayed as follows.

Moving to next page ...
Getting 2 pages
Moving to next page ...
Getting 3 pages
...
Getting 22 pages
Moving to next page ...
Getting 23 pages
Next page is nothing.
Finish!

3. Confirm the result

If you execute python in 2 above, a directory corresponding to the search word will be created. In that directory, a graph will be created as a result of scraping with Mercari, so please check the result.

If you are not satisfied with the result, review the structure of the csv file and execute scraping again!

Note

The same directory as the search word is created in the same directory as the python file (to prevent a large number of files from being created in the python file existence directory). If the same directory as the search word already exists, or the same search word exists in the search csv (mercari_search.csv), the process to create the directory (os.mkdir()) will not work properly, Scraping will stop halfway. Therefore, be careful when starting scraping that the same directory as the search word does not exist and that you do not enter the same search word in the csv file.
