Scheduler setup

Register Scheduler

This process requires credit card registration in advance. (Registration is required even in the free frame)

  1. Click Find more add-ons in the Resources> Add-ons section.

  2. Click Heroku Scheduler.

  3. Click Install Heroku Scheduler.

  4. Enter the Application name in App to provision to.

  5. Set the execution cycle in Schedule. Set the Python run command in Run Command.
