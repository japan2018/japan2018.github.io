Read emotion dictionary data

Now that we have confirmed that the dictionary has been successfully downloaded, we will load the data so that it can be handled by python. For that, I defined the following function. Converts and returns a group of words separated for each emotion into one dict format.

def get_emotional_words():
  emotions = ["aware", "haji", "ikari", "iya", "kowa", "odoroki", "suki", "takaburi", "yasu", "yorokobi"]
  emotional_words = {}
  for emotion in emotions:
    emotional_words[emotion] = []
    with open("emotions/" + emotion + "_uncoded.txt", "r") as f:
        for line in f:
          line = line.replace('\n','')
          emotional_words[emotion].append(line)
  return emotional_words

Acquisition of work data from Aozora Bunko

By the way, next we will pull the work data whose contents you want to check. The data of Aozora Bunko is also open to the public on git, but this time I’ve obtained the data from html. Taking advantage of my experience in my own article about 2 years ago, I am able to perform some preprocessing to obtain data.

def get_txt_from_aozorabunko(url):
  html = urllib.request.urlopen(url=url)
  soup = BeautifulSoup(html, "html.parser")
  # Get the body → <div class="main_text"> ~ body ~ </div>
  sentences = soup.find("div","main_text")
  # Extract only the character part
  sentences = sentences.get_text().replace("\r", "").replace("\n", "").replace("\u3000", "")

  sentences = re.sub(" (.*?)", "", sentences)
  return sentences

sentiment analysis

Now that we can read the dictionary and get the text data of the work, we will create a function for sentiment analysis. This time, we adopted a dictionary-based method and conducted a sentiment analysis using a relatively simple method. Specifically, the number of words contained in the emotion dictionary in the text data of the work is counted. Here is the actual code.

def count_emotional_words(sentences, emotional_words):
  count_emotions = [0] * len(emotional_words.keys())
  for idx, emotion in enumerate(emotional_words.keys()):
    for word in emotional_words[emotion]:
      count_emotions[idx] += sentences.count(word)
  return count_emotions

Make a radar chart

Create the code that creates the radar chart of the frequency of appearance of the last emotion expression word. This article was quite helpful in creating the code. Thank you very much. Enter a list of emotion names in labels and the frequency of emotion expression words received by the count_emotional_words function in values.

def plot_polar(labels, values, title):
  jp_font = {'fontname':'IPAGothic'}
  angles = np.linspace(0, 2 * np.pi, len(labels) + 1, endpoint=True)
  values = np.concatenate((values, [values[0]])) # make it a closed polygon
  fig = plt.figure()
  ax = fig.add_subplot(111, polar=True)ax.plot(angles, values,'o-') # outline
  ax.fill(angles, values, alpha=0.25) # fill
  ax.set_thetagrids(angles[:-1] * 180 / np.pi, labels, fontsize=15, **jp_font) # axis labels
  ax.set_rlim(0 ,max(values))
  ax.set_title("" "+ title + """, fontsize=15, **jp_font)

run

Finally, execute it with the main function. When the URL of Aozora Bunko and the work name are given to this main function, the radar chart at the beginning is output.

def main(url, title):
  sentences = get_txt_from_aozorabunko(url)
  emotional_words = get_emotional_words()
  count_emotions = count_emotional_words(sentences, emotional_words)
  emotional_kanji = ["sorrow", "shame", "anger", "dislike", "scary", "surprise", "good", "excite", "cheap", "joy"]
  labels = list(emotional_kanji)
  values = count_emotions
  plot_polar(labels, values, title)
