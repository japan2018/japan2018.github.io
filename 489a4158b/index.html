The goal is to get a general idea of the neural network. I don’t use mathematical formulas, so people who are allergic to mathematics should also check it.

#1.What is a neural network?

A neural network is something like the one below. When written in English, it is Neural Network, so it is often abbreviated as NN. So this time I will explain with NN

The round one is called “unit”, and the line connecting the units is called “synapse”. Since it is troublesome to call it a synapse, let’s call it a “line” and explain it.

The NN has a layered structure. Generally, counting from the left, the first layer, the second layer, and so on. The first layer is called “input layer”, the last layer is called “output layer”, and the spaces between them are collectively called “intermediate layer (hidden layer)”. Below is a diagram of a 4-layer NN.

#2.unit? weight? bias?

I will take a closer look. The round one is called a “unit”. The unit has one number.

Units are also called “neurons”. So Yuno can be recognized as “round guys”.

The unit number goes through the “line” (synapse) to the next unit. The numbers will be added to the next unit.

The value changes as you pass through the line. For example, as shown below, when the number “3” in the previous unit passes through the line, “x2” is calculated and “6” is added to the next unit. This “×2” is called “weight”. It is an image that the line has a number of × 2.

Each line has a different “weight” value. For example, if one unit is connected to three next units, as in the example below, the numbers will advance in three lines.

Consider the case where each of the three lines has a “weight” of x2, x-4, and x3. The “3” of the previous unit is multiplied by each “weight” and added to the next unit.

Also, the figure below shows when three front units are connected to one next unit. In this case as well, the results of “132”, “-2-4” and “-9*3” will be added more and more, and the value of the next unit will be “7”.

And again, it is a feeling that “7” advances to the next unit.

The NN is calculated as described above. Generally speaking, if you put a number in the “input layer” of the NN, various calculations will be performed through the NN and the results will appear in the “output layer”.

In the input layer, put the “data” that you want artificial intelligence to learn, but also put the “number other than data” in NN. This is called “bias”.

Bias is an image attached to each layer, as shown in the image below. To be precise, each unit has its own bias.

This “bias” number is also added at the timing when the number of the previous unit is added to the next unit. The bias line has no weight.

#3. Activation function

There is one more rule. The unit number changes value just before going to the line. Image-wise it looks like the image below.

The value of this changes according to the rule of “activation function”. There are many types of activation functions, but the most famous one is the “ReLU function”.

The ReLU function has the rule that if the unit value is “0 or less, change it to 0, and if it exceeds 0, pass it as it is.”

The value of the unit thus changed advances to the line.

Organize it. As a flow, The value of the unit changes according to the rule of the activation function → When the number passes through the synapse (line), it changes again → The number and bias are added to the next unit more and more.

By the way, changing the value by the activation function is called “activating”.

#4.NN overview

If you enter a number in the “input layer” of the NN, various calculations will be performed through the NN, and the result will appear in the “output layer”.

In detail, The value of the unit changes according to the rule of the activation function → When the number passes through the synapse (line), it changes again → The number and bias are added to the next unit more and more.

In addition, the “weight (× how many times)” and “bias” of the synapse are initially set at random values.

Updating this number is called “learning”. The goal of NN is to make the output feel good by updating the “weight” and “bias”.
