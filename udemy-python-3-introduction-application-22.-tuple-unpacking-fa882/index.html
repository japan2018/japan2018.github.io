■ Tuple unpacking

◆ Break tuples
num_tuple = (10, 20)
print(num_tuple)

x, y = num_tuple
print('x =', x, type(x))
print('y =', y, type(y))

(10, 20)
x = 10 <class'int'>
y = 20 <class'int'>

Elements of tuples can be separated.

◆ Actual substitution in parallel
X, Y = (0, 100)
print('X =', X, type(X))
print('Y =', Y, type(Y))

min, max = 0, 100
print('min =', min, type(min))
print('max =', max, type(max))

X = 0 <class'int'>
Y = 100 <class'int'>
min = 0 <class'int'>
max = 100 <class'int'>

The tuple could omit the (). To write in parallel like min and max means that Once 0, 100 is determined to be a tuple, and then it is unpacked, It has been assigned as an int type.

◆ Swapping objects using tuple unpacking
a, b = 1, 2
print(a, b)

a, b = b, a
print(a, b)

1 2
twenty one

Objects that have been assigned once can be replaced.
