I made a script to greet you at my Koshien

Feb 2, 2020 Python Selenium BeautifulSoup selenium-webdriver 

I tried to automate a certain task in a browser game called "My Koshien" using Python.

I don’t use Python at all, so it’s a beginner level code. Please note.

What is my Koshien

High school baseball simulation game. Although it is a baseball game, the actual game is played by the CPU, and the main player is coaching and coaching. Points are required for this instruction, and among some points there is a Passion PT, which can be increased by greeting another school (other player).

I have created a script that automates this greeting, so I would like to introduce it.

Library

The following libraries of Python are used for automation.

  • random
  • urllib
  • requests
  • Beautiful Soup
  • Selenium

To use Selenium, you need to download Chrome Driver separately.

#Script

import random, urllib, requests
from bs4 import BeautifulSoup

def getSchoolURL():
    # Randomly select a district
    area_num = random.randint(1, 49)

    #Get list of school links
    html = urllib.request.urlopen("http://orekou.net/profile/school_list/" + str(area_num)).read()
    soup = BeautifulSoup(html)
    link_elem = soup.select(".sub_content a")

    # Randomly select one from the list of schools
    school_num = random.randint(1, len(link_elem))
    school_url = "http://orekou.net" + link_elem[school_num-1].get("href")
    
    return school_url

from selenium import webdriver
from selenium.webdriver.common.keys import Keys

import orekou

#Load existing profile and generate Chrome driver
options = webdriver.ChromeOptions()
options.add_argument("--user-data-dir=C:\\Users\\<username>\\AppData\\Local\\Google\\Chrome\\User Data")
chrome = webdriver.Chrome("./chromedriver_win32/chromedriver.exe", options=options)

#Open Chrome browser
chrome.execute_script("window.open('','_brank');")

# Say hello to 100 randomly selected schools
for index in range(100):
    chrome.get(orekou.getSchoolURL())
    chrome.switch_to.window(chrome.window_handles[0])
    chrome.execute_script("document.getElementsByTagName('input')[2].click();")

chrome.quit()

Since it is necessary to log in to the browser game, we are including the process to acquire an existing profile. The above part is different for each user, so please change it by yourself. You can get points by greetings up to 100 times, so the loop is doing the same number of times.

#Result It takes about 3 seconds to greet one school, so it takes about 5 minutes to complete 100 schools. Much faster than doing it manually!
