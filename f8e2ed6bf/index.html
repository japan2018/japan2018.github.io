[Python] The simplest way to try PyQtGraph

Oct 7, 2016 Python PyQtGraph 

Before throwing it out, I definitely checked it.

PyQtGraph may not be available in the interpreter.

One line from command line (or terminal)

Typing python -m pyqtgraph.examples will bring up the example launcher in one line from the command line. (It was written in Documentation of README.md of PyQtGraph.)

python -m hoge.hogehoge is a command that executes the module hoge.hogehoge.py. (Python 2.7.x documentation) Since it is a module name, do not write the extension .py.

What can PyQtGraph do?

You can see what you can do with this launcher using PyQtGraph by clicking on the appropriate choice and pressing Run example at the bottom left.

How can I do it?

Not only what you can do, but what you can do “**” is shown on the right side of the launcher. In the right part of the launcher, the python code of the graph using the PyQtGraph you have just selected is displayed. This code display is prepared for PySide, PyQt4, PyQt5 (type of Qt Library), and can be changed from the Combo box on the right side written as Qt Library.

From the #python interpreter I tried embedding Matplotlib in a GUI made with Python PySide (Matplotlib version of the previous article) It was (It is the content of Matplotlib, but after all, it seems that I like PyQtGraph.)

import pyqtgraph.examples
pyqtgraph.examples.run()
