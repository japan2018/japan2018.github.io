# Jupyter Notebook Shortcut

The shortcut list that appears in h in Jupyter Notebook has been translated into Japanese for both Windows and Mac. I searched, but I didn’t have a list of all Windows and Mac in Japanese, so maybe it’s in demand?

Only the commands that you should remember are in bold.
