What is Face++

The name of the face recognition platform developed by Megvii in Beijing. Face++'s Detect API can authenticate a face and get various attributes. This time, we will use the attribute called Beauty Score to score it.

I also referred to the following articles for how to use Face++. Analysis of Pairs with python ~face photo version~ Introduction of Face++ API that detects the face when sending a photo and tells you the facial deviation value

Target

7 women’s contest participants from a team (Imamura Ishikawa Takahashi Yamashita Togo Naoe Yokogawa) The images used are those uploaded below. Instagram

Source

import requests
import json
import pprint

# API
# key: person name value: image URL declared as a dictionary
url_dict = {'Person name: Image URL'}

# Target person number loop
for img_data in url_dict.items():
    response = requests.post(
        'https://api-us.faceplusplus.com/facepp/v3/detect',
        {
            'api_key': "[APIKey obtained from Face++]",
            'api_secret': "[APISeacret obtained from Face++]",
            'image_url': img_data[1], # Image URL
            'return_attributes':'beauty' # attributes you want to get
        }
    )

    # json shaping
    json_dict = json.loads(response.text)
    print(img_data[0]) # output person name
    pprint.pprint(json_dict['faces'][0]['attributes']['beauty'])
