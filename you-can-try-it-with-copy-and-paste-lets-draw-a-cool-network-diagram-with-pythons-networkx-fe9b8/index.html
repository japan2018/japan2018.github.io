Today we will introduce a library called networkx that can draw network diagrams.

  • Time required for this work: about 10 minutes
  • What to prepare -Python 3.x -jupyterhub (please have pandas and matplotlib installed)

Install

(venv)$pip install networkx

First, load the data with pandas

import pandas as pd
df_links = pd.read_csv('https://microlearning.site/pydata/ch8/links.csv')
df_links.head(20)

Make sure you have a table loaded like this: The number of related networks is 1.

Finally draw a network diagram

The network diagram is shown below.

import networkx as nx
import matplotlib.pyplot as plt
G = nx.Graph()
NUM = len(df_links.index)
for i in range(1,NUM+1):
    node_no = df_links.columns[i].strip("Node")#Remove "Node" characters
    G.add_node(str(node_no))

for i in range(NUM):
    for j in range(NUM):
        if df_links.iloc[i][j] == 1:
            G.add_edge(str(i),str(j))

nx.draw_networkx(G,nide_color="k",edge_color="k",font_color="w")
plt.show()

Let me give you a brief explanation of what you are doing, first add a node Lines connect the parts where the nodes in the table are 1 (parts that are related).

Then, the following network diagram will be displayed (display is different every time) The numbers that have a strong relationship with each node come mainly. (4, 5 etc.)
