Thoughts It’s a long-standing problem. When I tried to solve this time, I didn’t understand why I was having a difficult time. Since the range of the solution is small, check all with for. I have placed a flag to check if the M conditions are met.

n, m = map(int,input().split())
sc = [list(map(int,input().split())) for _ in range(m)]

for i in range(10**n):
     i = str(i)
     flag = True
     if len(i) == n:
         for j in range(m):
             if i[sc[j][0]-1] != str(sc[j][1]):
                 flag = False
         if flag:
             print(i)
             quit()
