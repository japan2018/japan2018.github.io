④ book

I would like to introduce the books I read to learn machine learning.

Machine learning essence

Essence of Machine Learning This book covers the basics of Python, the mathematics necessary for machine learning, numerical calculation using Python, and machine learning algorithms. It seems like all of machine learning is in place. For those who have studied mathematics at university, it is recommended that you start reading from numerical calculation using Python.

Machine learning starts at work

[Machine learning starting at work](https://www.amazon.co.jp/%E4%BB%95%E4%BA%8B%E3%81%A7%E3%81%AF%E3%81%98%E3%82%81%E3%82%8B%E6%A9%9F%E6%A2%B0%E5%AD%A6%E7%BF%92-%E6%9C%89%E8%B3%80-(%E5%BA%B7%E9%A1%95/dp/4873118255) As its name suggests, it has an image of practice.
