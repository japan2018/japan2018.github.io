Number of cases

When you roll the dice, the number of rolls is 6, and if you make a coin toss, the number of sides In two ways, the number that can occur like this is called the number of cases __.

There are the following ways to find the number of cases.

  • Product law (multiplied if each probability can occur at the same time)
  • Law of sum (when each probability can occur independently, it is calculated by addition)
