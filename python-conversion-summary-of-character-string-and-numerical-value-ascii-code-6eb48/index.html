[Python] Conversion summary of character string and numerical value (ascii code)

Nov 25, 2019 Python Python3 

#Introduction

For myself, I have summarized how to calculate characters and ascii code in Python. It has been confirmed to work with Python 3.4.3 on AtCoder.

How to convert

You can convert between ord('character') and chr(number).

s ='A'

ord_s = ord(s)
print(ord_s) # 65

chr_s = chr(ord_s)
print(chr_s) #A

Conversion table

As a summary, the correspondence table of ascii codes and characters is shown. In addition, the item “ascii code” in the table below represents the decimal notation.

For example, uppercase and lowercase letters can be converted by converting to ascii code and then +32. (Of course you can also do it with .replace())

s ='A'
small_s = chr(ord(s)+32)
print(small_s) # a
