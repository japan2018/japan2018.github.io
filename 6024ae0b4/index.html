Overview

Get notified when Ringfit Adventures arrives at a list price on Amazon Ringfit adventure arrival bot I made. I will introduce the back side of this bot. Eradicate the resale yard.

#Background

Ringfit Adventure is very popular. The product has been in short supply since its release. This shortage is due to the mass purchase for resale by resellers. In fact, there are hundreds of resales at a price (about 12000 yen) that greatly exceeds the list price (8778 yen) on Amazon.

In order to buy at a fixed price without losing to resellers, it is necessary to click immediately after arrival. Therefore, we decided to create a function to notify the arrival of the ring fit at a fixed price. First of all, I made it for the purpose of getting the ring fit.

On the other hand, I also thought that by exposing this function, it would be possible to eradicate resellers. It is difficult to eliminate the existence of resale yard. However, you can use this feature to overcome the situation where you have to buy from a reseller. We thought that by providing a place where general buyers can fight on the same playing field as resellers, there is no need to buy from resellers, and as a result, resellers will be eradicated.

#Flow

This section describes the flow to eradicate resellers.

System job flow

  1. Every minute, cron runs a python script that does the following:
  2. selenium → chrome｜Access the product page of Amazon’s ring fit adventure.
  3. selenium｜Parse the obtained html and get the price of ring fit.
  4. python|Determines whether the price of the ring fit is the list price.
  5. selenium → chrome | If the price is fixed, log in to twitter and tweet.

User usage flow

  1. Follow the Ringfit Adventure Arrivals bot.
  2. Turn notifications on.
  3. Receive Tweet notifications to notify you of arrivals.
  4. Access Amazon first and click on the ring fit for a fixed price of ¥8778.
  5. You will receive the ring fit in a few days.

def tweet(message):
    # twitter account
    account ='ringfit_hoshii'
    password ='hogehoge'
    # start selenium
    options = webdriver.ChromeOptions()
    options.add_argument('--headless')
    driver = webdriver.Chrome(options=options)
    driver.set_window_size(height=877, width=1440)
    #Open login page
    driver.get('https://twitter.com/login/')
    time.sleep(3) # stop working
    Enter # account
    element_account = driver.find_element_by_class_name("js-username-field")
    element_account.send_keys(account)
    # Enter password
    element_pass = driver.find_element_by_class_name("js-password-field")
    element_pass.send_keys(password)
    # Click the login button
    element_login = driver.find_element_by_xpath('//*[@id="page-container"]/div/div[1]/form/div[2]/button')
    driver.execute_script("window.scrollTo(0, document.body.scrollHeight);")
    element_login.click()
    # Tweet
    time.sleep(3)
    driver.find_element_by_xpath('//*[@id="react-root"]/div/div/div/main/div/div/div/div[1]/div/div[2]/div[2]/ div[1]/div/div/div[2]/div[1]/div/div/div/div/div/div/div/div[1]/div[1]/div/div/div/div [2]/div/div/div/div').send_keys(message)
    # time.sleep(1)
    driver.find_element_by_xpath('//*[@id="react-root"]/div/div/div/main/div/div/div/div[1]/div/div[2]/div[2]/ div[1]/div/div/div[2]/div[2]/div/div/div[2]/div[3]').click()
    # Sleep to wait for the tweet to be sent
    time.sleep(2)
    driver.quit()
    return

# Function to get the price from the Amazon ring fit product page and determine the list price
def get_price(xpath):
    price1 = driver.find_element_by_xpath(xpath)
    p1str = price1.text[1:]
    p1int = int(p1str.replace(',',''))
    now = datetime.datetime.now(pytz.timezone('Asia/Tokyo'))
    message2 = "[Stock] Ring Fit Adventure is sold for + str(p1int) + "yen!" + "Amazon URL: "+ url +" datetime: "+ str(now)
    # Fixed price judgment
    if p1int> 9000 or p1int <5000:
        return
    tweet(message2)
    return

Why via selenium?

Amazon Since it is played when accessed from curl command or python, I started chrome with selenium and took the method of accessing with the same procedure as general users.

twitter I got a twitter developer account, but for some reason I didn’t have permission to create an app. There was a description that you can say that you can register your phone number in your twitter account, but this time I could not register the phone number for some reason, Give up and launch chrome with selenium, abbreviated below.

Is it a convenient function for resellers?

That’s right. However, I think the current situation is that resellers have such functions themselves. The significance of this bot is to ensure that the general buyer stands on the same playing field as the reseller.
