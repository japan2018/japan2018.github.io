Apr 11, 2020 Python Anaconda Miniconda 

# Miniconda

A lightweight version of Anaconda. While Anaconda includes many libraries and IDEs, Miniconda focuses on conda, Python, and the minimum required libraries.

What is good

Lightweight compared to Anaconda At the time of installation, since various Anaconda is included, it will be about several GB. In that respect Miniconda contains only the minimum required, so it will be several hundred MB.

conda can be used conda is a tool for managing Python packages and versioning Python itself. If you use pip to build the environment, you only need conda where you need pip+pyenv. In particular, the Python environment construction on MacOS is easy to clog up with the default Python, so it’s easy to use conda alone.

Miniconda has a cool name I feel that conda is cooler than pip and Miniconda is cooler than Anaconda. It’s just a sound.

How to use

Install it using the installer from https://docs.conda.io/en/latest/miniconda.html from the official Miniconda site.

Create a virtual environment

Create a virtual environment using conda.

conda create -n py36 python=3.6
conda activate py36

By doing, a Python 3.6 virtual environment py36 is created, and you can switch to that environment with activate.

At this time, kindly add (py36)$ to the environment name at the beginning of the prompt. If not needed,

conda config --set changeps1 False
source ~/.bash_profile

By doing so, it will be written to .condarc and will not be displayed.

How to get out of the virtual environment

conda deactivate

Will restore the default base.

Add Library

The library can be added with the following command.

conda install pandas

If you can’t find the library in the default channel, you can try searching with conda-forge.

conda search -c conda-forge pixiedust
conda install -c conda-forge pixiedust

If you can’t find it by default, you’ll want to use pip, but I’d like to avoid using conda and pip together as much as possible. If you can’t find it in conda-forge, you can try Anaconda Cloud.
