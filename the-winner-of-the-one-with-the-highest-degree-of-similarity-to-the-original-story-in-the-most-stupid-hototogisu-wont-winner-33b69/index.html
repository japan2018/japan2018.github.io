#Implementation Similarity calculation API.

The original story of Hototogisu is the following three works. Nobunaga: If it doesn’t ring, kill it. Hideyoshi: If it doesn’t ring, let’s ring it Hottogisu Ieyasu: If it doesn’t ring, wait until it rings.

First, we will compare each of the above three phrases with the “Kitchen’s Stinking Hototogisu Works” one by one, and create a ranking similar to each daimyo. After that, by summing up the respective similarity scores, the “winning stinking Hototogisu overall winning work” with the highest similarity to the original story is determined. (That is, the one with a high degree of similarity on average wins.)

Display the uploader and upload the above work list “hototogisu_utf8.csv”. Then read the file line by line as follows:


import csv

f = open('hototogisu_utf8.csv','r',encoding = "utf-8-sig")
reader = csv.reader(f)

sentence1 = "If you don't ring, kill me, Hottogisu"
d_nobunaga = {}
for row in reader:
    sentence2="".join(row)
    print(sentence2) # for displaying progress
    api_result = cotoha_api.callSimilarityApi(sentence1, sentence2)
    result_score = api_result['result']['score']
    d_nobunaga[sentence2]=result_score
f.close()
print(d_nobunaga) # for displaying results

For example, calculate the degree of similarity with each daimyo and store it in the dictionary. When sorting and displaying in order of similarity,


import pprint

d_sorted = sorted(d_nobunaga.items(), key=lambda x:x[1])

from collections import Counter
