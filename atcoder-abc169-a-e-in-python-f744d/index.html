#D: Div Game How many kinds of $z$ can be prepared? In other words Since $z$ can be expressed by the power of the prime factor of $N$, the number of the same prime factors is counted for each prime factor, and 2 or less ⇒ 1, 3 or more and less than 6 ⇒ 2 ($p^1$, $ p^2$), 6 or more and less than 10 ⇒ 3 ($p^1$, $p^2$, $p^3$)… such as $z$ that can be expressed by each prime factor All you have to do is count. Since it is $2^{43}>10^{12}$, it is OK if you consider at most 42 the number of each prime factor. As an exception, only 1 is 0. For the function part of factorization, I referred to here.

def factorization(n):
    arr = []
    temp = n
    for i in range(2, int(-(-n**0.5//1))+1):
        if temp%i==0:
            cnt=0
            while temp %i==0:
                cnt+=1
                temp //= i
            arr.append([i, cnt])

    if temp!=1:
        arr.append([temp, 1])

    if arr==[]:
        arr.append([n, 1])

    return arr
    
a=factorization(int(input()))

b=[1,3,6,10,15,21,28,36,43]

ans=0
if len(a)==1 and a[0][0]==1:
    print(0)
    exit()

for i in a:
    for j in range(len(b)):
        if i[1]<b[j]:
            ans+=j
            break
print(ans)

#E: Count Median A value that can be taken as the overall median between the median of the set of A and the median of the set of B. If N is an even number, the interval is 0.5, and if it is an odd number, the interval is 1.

def median(q):
    l=len(q)
    q=sorted(q,reverse=True)
    if l%2==0:
        return (q[int(l/2)]+q[int(l/2)-1])/2
    else:
        return q[int((l+1)/2-1)]

n=int(input())
