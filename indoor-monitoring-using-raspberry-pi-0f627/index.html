Indoor monitoring using Raspberry Pi

Jan 16, 2020 Python RaspberryPi 

#Introduction

Qiita’s first post, thank you. #Proposal I wanted to make an inexpensive monitoring system with Raspberry Pi and a sensor. Specifically, it is a monitoring system that installs RaspBerryPi Zero, RaspBerryPi camera, human sensor (HC-SR501) in the laboratory, takes a picture with a camera when it detects a person, and sends the image to Slack (not completed).

Equipment used

・Raspberry Pi zero ・RaspberryPi camera ・HC-SR501 Human infrared sensor ・Three jumper wires (female/female)

#Implementation The following is the program to shoot with RaspberryPi camera when the human sensor detects a person.

import time
import picamera
import RPi.GPIO as GPIO

INTERVAL = 5

SLEEPTIME = 1

GPIO_PIN = 18

GPIO.setmode(GPIO.BCM)
GPIO.setup(GPIO_PIN,GPIO.IN)

if __name__ =='__main__':
    try:
        print("CTRL+C" to cancel processing)
        while True:

            if(GPIO.input(GPIO_PIN) == GPIO.HIGH):
                with picamera.PiCamera() as camera
                    camera.resolution = (1024,768)
                    camera.brightness = 70
                    camera.capture('picture.jpg')
            else:
                time.sleep(INTERVAL)
    except KeyboardInterrupt:
        print("end all processing")
    finally:
        GPIO.cleanup()

##Execution result I built a local web server and I was able to confirm that the images could be obtained.
