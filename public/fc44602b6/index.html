[Python] [python] A note about debug

Apr 24, 2016 Python Python3 

#Note
  • It’s just a memo #__debug__ This is one built-in constant. If you add -O during python execution, it becomes False, otherwise it becomes True. (This is capital letter O, not zero)
def debug_print(s):
     if not __debug__:
         print(s)

If you make it like this,

$ python -c'(__import__)("debug_print").debug_print(__debug__)'
$ python -O -c'(__import__)("debug_print").debug_print(__debug__)'
False

Like, is printed only when the -O option is added.
