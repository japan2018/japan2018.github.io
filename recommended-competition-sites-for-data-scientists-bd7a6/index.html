[Python] Recommended competition sites for data scientists

Jan 11, 2020 Python Data Scientist Machine Learning Primer Competition 

We have gathered competition sites to help career development of those who aim to be data scientists.

#Kaggle

It is a famous site in the data scientist community. It is written as Kaggle and read as “Kaguru”. Data scientists from all over the world compete here every day. Google acquired it on March 8, 2017. I want to request an analysis Major companies are winning the prize money, and in the past, competitions were held by Mercari and Recruit.

Official page https://www.kaggle.com

#signate

Compared to Kaggle, a competition site operated by signate in Japan, the number of data sets is not so large, but Japanese companies such as the Ministry of Economy, Trade and Industry are participating, so the feature is that the data is familiar, The problem is a little difficult.

signate official page https://signate.jp

#probspace

There are unique datasets such as “Ukiyo-e Creator Prediction” and “Kushizushi”. Also, although the amount of data is small, the level of data scientists participating is high, so the topic etc. is a worthwhile companion site.

Official page https://prob.space

#atna

A competition run by atma, a venture company that conducts data analysis, algorithm solutions, and system development. Unlike others, instead of submitting online, gather at the venue for a real-time competition. If you want to participate, please apply early because the number of people is still under capacity.

Official page https://atma.connpass.com/

#nishika

“Brandear Auction!” A competition for developing recommended engines. It’s just getting up here, but since there are few participants, it may be the aim.

Official page https://www.nishika.com/
