In some cases it may be useful to use eval. eval will execute the string given as an argument as an expression. Therefore, it may be shortened by appropriately processing the input string and passing it to eval. For example,

10 20

In case of the problem that two numbers are given in one line and the product is output,

print(eval(input().replace('','*'))))
