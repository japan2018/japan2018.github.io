I tried to summarize the string operation of Python

Nov 11, 2019 Python for beginners 

#Introduction

I’m coding in Python, and I had a built-in method that I could use when working with strings, so I’ve put it together as a memorandum. As for the method of dividing the system, we have systematized it on an individual basis, so don’t worry.

#Orthopedics

format method

The value in the string can be described later using index It becomes easy to edit the character string when used together with the variable.

print("This test is {0} for {1}.".format("easy", "you"))
# This test is easy for you.

hoge = "easy"
fuga = "you"
print("This test is {0} for {1}.".format(hoge, fuga))
# This test is easy for you.

join method

Add the previous value of the method between the arguments Not added after the point starting with the argument and after the last character of the argument

upper_hoge = "ABC"
lower_hoge = "abc"
print(lower_hoge.join(upper_hoge)) # AabcBabcC

replace method

Replace characters in a string The first argument is the character before replacement Second argument is the replacement character If you give the third argument, you can specify the number of characters to replace

You can delete the target character by setting blank in the second argument.

hoge = "ababab"
print(hoge.replace('a','A')) # AbAbAb
print(hoge.replace('a','A', 1)) # Ababab
print(hoge.replace('a','A', 2)) # AbAbab
print(hoge.replace('a', ``)) # bbb

strip method

Remove the target character string from the beginning and end The strings included in the argument are processed in any order, if their sets match. If there is no argument, remove the string

hoge = "A testAtestAtest A "
# Leading and trailing blanks are removed
print(hoge.strip()) # A testAtestAtest A
# A is not removed because the beginning and end are blank
print(hoge.strip('A')) # A testAtestAtest A

fuga = "A testAtestAtest A"
# Leading and trailing A are removed
print(fuga.strip('A')) # testAtestAtest
#Looking from the beginning and the end, the part that does not correspond to any of'A','t','' of the argument is removed.
print(fuga.strip(("At "))) # estAtestAtes

lstrip method

Remove the target string from the end The strings included in the argument are processed in any order, if their sets match. Remove blanks if there are no arguments

hoge = "A testAtestAtest A "
# Leading blanks only removed
print(hoge.lstrip()) # A testAtestAtest A
#Because the beginning is blank, it does not reach the argument A, so nothing is removed
print(hoge.lstrip('A')) # A testAtestAtest A

fuga = "A testAtestAtest A"
# Only the leading A is removed
print(fuga.lstrip('A')) # testAtestAtest A
#Looking from the beginning, the part that does not correspond to any of the arguments'A','t',''is removed.
print(fuga.lstrip(("At "))) # estAtestAtest A

rstrip method

Remove the target string from the end The strings included in the argument are processed in any order, if their sets match. Remove blanks if there are no arguments

hoge = "A testAtestAtest A "
# Only trailing blanks are removed
print(hoge.rstrip()) # A testAtestAtest A
# Nothing is removed because it doesn't reach the argument A because the end is blank
print(hoge.rstrip('A')) # A testAtestAtest A

fuga = "A testAtestAtest A"
# Only the trailing A is removed
print(fuga.rstrip('A')) # A testAtestAtest
# Starting from the end, the part that does not correspond to any of'A','t' and'' in the argument is removed.
print(fuga.strip(("At "))) # A testAtestAtes

#Search system

find method

Returns where the corresponding character is, counting from the beginning If there is no corresponding character, output “-1”. Search start position and end position can be passed as arguments (Note that the index of the last position is the index up to the previous one!)

hoge = "test"
print(hoge.find('t')) # 0
print(hoge.find('a')) # −1
print(hoge.find('t', 1, 4)) # 3
print(hoge.find('t', 1, 2)) # -1

count method

Print the number of certain characters in the list

# Output the number of t this time
hoge = "test"
print(hoge.count('t')) # 2

Split system

split method

Decomposes a string with the characters of the argument and stores it in a list The second argument is the number of splits

hoge = "a b c d e f g"
print(hoge.split('')) # ['a','b','c','d','e','f','g']
print(hoge.split('', 0)) # ['a b c d e f g']
print(hoge.split('', 1)) # ['a','b c d e f g']
print(hoge.split('', 4)) # ['a','b','c','d','e f g']

rsplit method

Similar to the split method, it splits a string, but when splitting, it will be split from the right side. The second argument is the number of splits

hoge = "a b c d e f g"
print(hoge.rsplit('')) # ['a','b','c','d','e','f','g']
print(hoge.rsplit('', 0)) # ['a b c d e f g']
print(hoge.rsplit('', 1)) # ['a b c d e f','g']
print(hoge.rsplit('', 4)) # ['a b c','d','e','f','g']

#Discrimination system

startswith method

Method to determine the start of the string Returns true if it begins with an argument, false otherwise

hoge = "test"
print(hoge.startswith('te')) # True
print(hoge.startswith('tt')) # False

endswith method

Method to determine the end of the string Returns true if it ends with an argument, false otherwise

hoge = "test"
print(hoge.endswith('st')) # True
print(hoge.endswith('tt')) # False

Uppercase/lowercase conversion system

upper method

Convert string to upper case

lower_hoge = "abc"
print(lower_hoge.upper()) # ABC

lower method

Convert string to lower case

upper_hoge = "ABC"
print(upper_hoge.lower()) # abc

swapcase method

Swap upper and lower case

hoge = "AbcDefG"
print(hoge.swapcase()) # aBCdEFg

capitalize method

Capitalize the first letter

hoge = "test"
print(hoge.capitalize()) # Test
