It’s a two-dimensional array, but using a library of matrices can be easier to do.

Ruby

require'matrix'

class Matrix
  #Not assignable in v2.3
  def []=(i, j, x)
    @rows[i][j]=x
  end
end

n, m = gets.split.map(&:to_i)
rows = Array.new(n + m){gets.chomp.chars.map{|c| (c =='#') ?1 :0}}
a = Matrix.rows(rows.shift(n))
b = Matrix.rows(rows)
(n-m + 1).times do |i|
  (n-m + 1).times do |j|
    if a.minor(i, m, j, m) == b
      puts "Yes"
      exit
    end
  end
end
puts "No"

Sub-matrix is acquired by minor and matrix comparison is performed.

  (n-m).times do |j|

When ~~1 x 1 matrix is used, the matrix comparison does not work, so the corresponding processing is included. ~~(n-m) was wrong and (n-m + 1) was the correct answer.

Python

import numpy

n, m = map(int, input().split())
a = numpy.zeros([n, n], dtype=int)
b = numpy.zeros([m, m], dtype=int)
for i in range(n):
    s = input()
    for j in range(n):
        if s[j] =='#':
            a[i, j] = 1
for i in range(m):
    s = input()
    for j in range(m):
        if s[j] =='#':
            b[i, j] = 1
for i in range(n-m + 1):
    for j in range(n-m + 1):
        if numpy.all(a[i: i + m, j: j + m] == b):
            print("Yes")
            exit()
print("No")
