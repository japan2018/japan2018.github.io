#Introduction

In recent years, I often hear the word ~~ as a Service such as IaaS (Infrastructure as a Service) and PaaS (Platform as a Service) in IT area. Here, I would like to collectively call them [A-Z]aaS. I think that there are increasing opportunities to listen to various [A-Z]aaS such as BaaS (Backend as a Service) such as Firebase and FaaS (Function as a Service) such as AWS Lambda. This time I searched for various [A-Z]aaS. As a result, we could find 910 [A-Z]aaS. (All the searched results are displayed in the second half.)

How to find #[A-Z]aaS Let’s fix the first two letters and collect what you see in the Google search suggestion. Since there are 26 letters in the alphabet, there are 26*26=676 patterns. (Caution) Just because it appeared in the Google search suggestion, it does not mean that such a service is actually available.

There is Google Suggest API, and you can get the suggestion of Google search when you hit a certain word. Therefore, I couldn’t understand what URL page to send the request with Google Suggest API. . . If this were possible, I think the amount of work for collection would have been reduced to 1/10.

(Study 2) Use python OCR

I wanted to avoid hitting the suggested result by hand, so I will use OCR to convert the image information into a character string. Just hit Aa as a Service to Zz as a Service in the search bar of chrome and take a screen shot of the suggested screen. (676 screenshots taken)

Use the following program to extract character information from an image using OCR. After that, extract the ones that fit the [A-Z]aaS format. With the following code, I was able to extract [A-Z]aaS from the image.

from PIL import Image
import glob

import pyocr
import pyocr.builders

tools = pyocr.get_available_tools()
tool = tools[0]

langs = tool.get_available_languages()
lang = langs[0]

Turn with # a ~ z
for i in range(97, 97+26):
    image_files = glob.glob("images/%c/Screen*"%chr(i))
    words = []
    for image_file in image_files:
        im = Image.open(image_file)
        croped_image = im.crop((300, 250, 1000, 730)) # criteria

        # OCR reads characters line by line from image
        line_and_word_boxes = tool.image_to_string(
            croped_image,
            lang=lang,
            builder=pyocr.builders.LineBoxBuilder()
        )

        #Read each string from the obtained string
        for line_words in line_and_word_boxes:
            if line_words.content.endswith('as a service') and line_words.content.startswith(chr(i)):
                word = line_words.content.capitalize()
                word = word.replace('service','Service')
                print(word)
                words.append(word)

    #Concatenate with line feed code and write to file
    words ='\n'.join(list(set(words)))
    with open("result/"+chr(i)+".txt",'w') as f:
        print(words, file=f)
