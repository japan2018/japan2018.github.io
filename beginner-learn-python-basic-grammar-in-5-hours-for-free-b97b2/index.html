List comprehension

**List Comprehensions create a new list from an existing list or generator. ** For example, the following is a sample program that writes a list of numbers in which the numbers 1 to 10 are each squared, written in list comprehension notation.

result = [x**2 for x in range(1,11)]
print(result) # [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

# In a normal loop:
result = []
for i in range(1,11):
    result.append(i**2)

print(result) # [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

**List comprehension can be used not only for processing the elements extracted from the existing list, but also for adding to the new list only when the conditions are met. ** The advantage of comprehensions is that you can write the process concisely and reduce the cost of calling additional methods such as new lists.

Other inclusion notation

In addition to list comprehensions, there are comprehensions that generate sets and dictionaries. **In the case of set and dictionary comprehension notation, the order of the output value may not be retained depending on the generated value, so please be careful when using. **

The way to write the dictionary comprehension is to separate the key and value with a colon “:”, Use {} as in set comprehension.

d = {x*2:x**2 for x in range(1,11)}
print(d) # {2: 1, 4: 4, 6: 9, 8: 16, 10: 25, 12: 36, 14: 49, 16: 64, 18: 81, 20: 100}
