What is pytube

Python library. You can easily download videos on YouTube.

##environment

  • Windows 10 Pro
  • Python 3.7.4

Let’s install

pip install pytube

If you type the above into bash or command prompt, it will be installed.

#Note Please see the video author’s channel name before downloading. If there is no check mark next to the channel name, If you do, you will violate the YouTube rules.

Let’s download the video

Let’s download it now.

from pytube import YouTube

YouTube(input("URL of the video you want to download:")).streams.first().download()

Just copy and paste the above code and run it. When asked the URL, paste the video URL and press Enter. If it works, you should have a video file in the same folder.

Specify the folder to save

from pytube import YouTube

YouTube(input("URL of the video you want to download:")).streams.first().download(r "folder path")

If you want to specify the folder to save, let’s pass the path of the save destination folder in the argument of the download method. Since you specify the folder path, don’t forget to add r to disable the escape sequence. If you want to change the name of the video file when downloading, set the name you want to give to the second argument of the download method.

Try to specify the resolution and format

from pytube import YouTube

url = input("URL of the video you want to download:")
print(*YouTube(url).streams.all(), sep="\n")
itag = int(input("Video tag you want to download:"))
YouTube(url).streams.get_by_itag(itag)).download()

Now try running this code. You will be asked for the URL immediately after executing it, so enter it and press Enter.
