Create a Polls application

Create it with the following command. Also, it seems that application creation can be done anywhere in the file.

$ python manage.py startapp polls

The contents of the created application are as follows.

polls/
    __init__.py
    admin.py
    apps.py
    migrations/
        __init__.py
    models.py
    tests.py
    views.py

Creating your first view

Edit polls/views.py.

from django.http import HttpResponse

def index(request):
    return HttpResponse("Hello, world. You're at the polls index.")

It is necessary to associate the url. Create urls.py.

from django.urls import path

from .import views

urlpatterns = [
    path('', views.index, name='index'),
]

Then reflect it in mysite/url.py.

from django.contrib import admin
from django.urls import include, path

urlpatterns = [
    path('polls/', include('polls.urls')), #add
    path('admin/', admin.site.urls),
]

This sequence of operations is a little different from the routes such as Rails.

If you access http://localhost:8000/polls/ with the settings so far ![Screenshot 2020-01-02 1.42.43.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/417600/eaff6fdf-483b-680a-2bb3-(fa513d060e95.png)

If this is the case, you are successful!

(Extra) path

path takes arguments as follows

path(route, view, kwargs=None, name=None)

route path pattern such as “polls/”

view called when the view pattern matches

kwargs Pass any keyword argument as a dictionary to the target view.

Give the name URL a name so that it can be called from anywhere in your project.
