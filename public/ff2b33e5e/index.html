Meaning of arguments of setup() function

  • name
    • Project name shown on PyPI
  • version
    • version
  • description
    • A brief overview of the project. Shown in PyPI
  • url
    • Project home page
  • author and author_email
    • About the creator
  • license
    • License type
  • classifiers
    • List of project categorizations
  • keywords
    • Project keywords
  • pakages
    • List of packages required for this project. A package is a directory
    • If you create something under the directory called simpleapi,
    • becomes packages=['simpleapi']
    • Also, as from setuptools import setup, find_packages
    • packages=find_packages(excluede['contrib','docs', tests*]) Can be *
      • Exclude excludes the directories below
  • install_requires
    • Dependent packages (must be piped in advance)
  • package_data
    • Additional packages required for the project

  • data_files
    • Additional files required
  • script
    • Script to be executed at installation
