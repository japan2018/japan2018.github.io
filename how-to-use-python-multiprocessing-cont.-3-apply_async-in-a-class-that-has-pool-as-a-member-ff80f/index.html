Specifications and implementation policy

As a specification

  • Call the process input with Push method that takes a parameter as an argument. This method does not block.
  • Internally, we have records of in-process and end processing
  • If you need to signal the end, you can implement that too

So,

  • Implement one class. Only processing is a separate function. (Image in the outside file)
  • Use multiprocessing.Pool. Besides this, manage the result of apply_async.
  • The callback function that is set when apply_async is also made a method of the class so that the queue that is a member of the class can be accessed.
  • (I don’t know if this is good) Internally, use job_id to manage. Implement so that the number of the input processing is unique.

Implementation

I wrote it like this.

from multiprocessing import Pool
from time import sleep
from os import getpid, getppid, cpu_count
from datetime import datetime, time, date, timedelta
import sys, json

def _data_handler(o):
    """ Use as default=_data_handler in json.dump """
    if isinstance(o, (datetime, date) ):
        return o.strftime('%Y/%m/%d %H:%M:%S')
    elif hasattr(o, "isoformat"):
        return o.isoformat()
    else:
        return str(o)


class JobError(Exception):
    def __init__(self, job_id:int, pid:int, msg:str, error):
        self.job_id = job_id
        self.pid = pid
        self.msg = msg
        self.error = error


def f(*args, **kwargs):
    """Sleep for specified time """
    try:
        print("[{}---{}] f(args {} kwargs={})".format(getpid(), getppid(), args, kwargs))
        t = kwargs["params"]["sleep_time"]
        if t == 0:
            raise Exception("Exception!! sleep time = 0")
        sleep(t)
        return {"f_answer": 0.0, "pid": getpid(), "job_id": kwargs["job_id"]}
    except Exception as e:
        raise JobError(kwargs["job_id"], getpid(), "Exception in except in f", e)


class JobController(object):

    def __init__(self, num_cpu:int=0):
        """
        Specify the number of cores to use. If 0, the core managed by the OS
        """
        print("main pid={} cpu_count={}".format(getpid(), cpu_count()))
        num_cpu = cpu_count()
        self._pool = Pool(num_cpu)
        self._working_jobs = {} #job_id is the key. In progress
        self._closed_jobs = {} # job_id is the key. End
        self._new_job_id = 0 # job_id to use for the next submitted job

    def __del__(self):
        pass # Originally something should be done to keep zombies out (not yet known).


    def my_cb(self, *args):
        """Move the Job result that completed normally to the buffer."""
        print("callback args={} jobid={}".format(args, args[0]["job_id"]) ))
        try:
            jobid = args[0]["job_id"]
            self._closed_jobs[jobid] = self._working_jobs.pop(jobid, {})
            self._closed_jobs[jobid]["end_time"] = datetime.now()
            self._closed_jobs[jobid]["successful"] = True
            self._closed_jobs[jobid]["return"] = args
        except:
            pass
        if len(self._closed_jobs) == 0:
            self._pool.join()


    def my_err_cb(self, args):
        """ Move the result of the abnormally terminated Job to a buffer. args is JobError """
        print("error callback args={} job_id={}".format(args, args.job_id))
        try:
            jobid = args.job_id
            self._closed_jobs[jobid] = self._working_jobs.pop(jobid, {})
            self._closed_jobs[jobid]["end_time"] = datetime.now()
            self._closed_jobs[jobid]["successful"] = False
            self._closed_jobs[jobid]["return"] = args
        except:
            pass
        if len(self._closed_jobs) == 0:
            self._pool.join()


    def PushJob(self, params:dict):
        """ Drops a Job. The argument is given here as dictionary data. """
        print("PushJob ", getpid(), getppid())
        res = self._pool.apply_async(f, args=(1,), kwds={"params":params, "job_id":self._new_job_id},
            callback=self.my_cb, error_callback=self.my_err_cb)
        self._working_jobs[self._new_job_id] = {"start_time": datetime.now(), "async_res": res}
        self._new_job_id += 1


    def GetCurrentWorkingJobCount(self):
        """ Number of jobs in process (submitted but not completed) """
        return len(self._working_jobs)


    def GetCurrentClosedJobCount(self):
        """ Number of completed Jobs """
        return len(self._closed_jobs)


if __name__ == "__main__":
    try:

            print("working_jobs {}:", job_controller.GetCurrentWorkingJobCount())
            print(json.dumps(job_controller._working_jobs, indent=2, default=_data_handler))
            print("closed_jobs {}:", job_controller.GetCurrentClosedJobCount())print(json.dumps(job_controller._closed_jobs, indent=2, default=_data_handler))
            if job_controller.GetCurrentWorkingJobCount() == 0:
