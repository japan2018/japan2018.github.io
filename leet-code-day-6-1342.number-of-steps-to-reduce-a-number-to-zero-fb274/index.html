Problem

1342. Number of Steps to Reduce a Number to Zero

If the given natural number num is odd, subtract 1; if it is even, divide by 2. The problem is to create a function that counts the number of times it is repeated until it reaches 0 and returns that number.

For the time being, I will write something that comes to my mind.

class Solution:
    def numberOfSteps (self, num: int) -> int:
        steps = 0
        while num> 0:
            if num %2 == 0:
                num //= 2
            else:
                num -=1
            steps += 1
        return steps
