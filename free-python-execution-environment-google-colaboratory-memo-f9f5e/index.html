# What is Google Colaboratory

Jupyter notebook environment running on Google’s cloud. https://colab.research.google.com/notebooks/welcome.ipynb

It’s free to use and no environment is required, so it’s convenient for learning Python.

In addition, GPU and TPU can be used to reduce machine learning execution time.

usage limit

It is available free of charge, but there is a 12 hour continuous usage limit.

Be careful when using machine learning for a long time because the virtual machine will be disconnected after 12 hours.

The virtual machine will also be disconnected if left idle for the idle timeout period.

*The idle timeout time is not listed in the FAQ, but it currently seems to be 90 minutes. https://research.google.com/colaboratory/faq.html

#Note The following are usage notes.

Using GPU/TPU

Select from Runtime> Change Runtime Type.

Mount Googole Drive

Select “Mount Drive” from the side menu and the code for mounting will be generated automatically.

TensorFlow version change

Currently the default is 1.x series. You can change to 2.x system by executing the following.

%tensorflow_version 2.x
