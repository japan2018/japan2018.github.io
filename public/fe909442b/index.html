[Python] Create a graph with borders removed with matplotlib

Jan 30, 2017 Python matplotlib 

How to make a graph like this with matplotlib.

import matplotlib

First, import matplotlib with the name plt. I used Python 3.5 and matplotlib 1.5.3. It should work with matplotlib 2.0.

import matplotlib.pyplot as plt

Plot normally

plt.plot([0,10,20,30,40,50], [10,20,30,5,25,30])
plt.show()

A graph with frame lines on the top, bottom, left and right, which is the default design of matplotlib, is drawn.

Erase right and top borders

Borders can be manipulated by calling spines from plt.gca(). You can manipulate the corresponding borders with 'right', 'left', 'top', and 'bottom'. You can specify the frame you want to operate and erase it with set_visible(False). However, the frame line disappeared, but the scale did not disappear.

plt.plot([0,10,20,30,40,50], [10,20,30,5,25,30])
plt.gca().spines['right'].set_visible(False)
plt.gca().spines['top'].set_visible(False)
plt.show()

#Turn off the right and upper scales Similarly, only the scale specified by yaxis.set_ticks_position() from plt.gca() is displayed. This time, I want to keep the left and bottom scales, so specify 'left' and 'bottom'.

plt.plot([0,10,20,30,40,50], [10,20,30,5,25,30])
plt.gca().spines['right'].set_visible(False)
plt.gca().spines['top'].set_visible(False)
plt.gca().yaxis.set_ticks_position('left')
plt.gca().xaxis.set_ticks_position('bottom')
plt.show()

Make the borders disappear by default

You can set a graph without borders as the default graph by playing with matplotlibrc. In case of mac, open the setting file of ~/.matplotlib/matplotlibrc. If you installed Python with Homebrew, it is in /usr/local/lib/python3.6/site-packages/matplotlib/mpl-data/matplotlibrc.

Since there is such a place in matplotlibrc, delete # and rewrite True to False.

# axes.spines.left: True # display axis spines
# axes.spines.bottom: True
# axes.spines.top: True
# axes.spines.right: True

Since I want to erase the top and right borders this time, delete the # in .top and .right and rewrite True as False.

# axes.spines.left: True # display axis spines
# axes.spines.bottom: True
axes.spines.top: False
axes.spines.right: False

This will output a graph with the right and top borders disappearing by default.
