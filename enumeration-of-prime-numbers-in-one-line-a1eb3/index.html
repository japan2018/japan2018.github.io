Enumeration of prime numbers in one line

Mar 26, 2020 Ruby Python Haskell prime numbers 

#Introduction

I suddenly wrote a program that finds prime numbers in one line. Emphasis on code volume, not efficiency.

Ruby

If you think it is a filter, it is select in case of Range. (Ruby 2.5)

(2..99).select {|x| (2...x).select {|i| x %i == 0} == []}

Python

You can use filter, but the list comprehension is cleaner.

[x for x in range(2, 100) if [i for i in range(2, x) if x %i == 0] == []]

Haskell

Haskell can’t get out of a beginner who is studying for years.

[x | x <- [2..100], [i | i <- [2..(x-1)], mod x i == 0] == []]

#Summary I was surprised to find that Ruby, which does not allow list comprehension, is the shortest of all I wrote.
