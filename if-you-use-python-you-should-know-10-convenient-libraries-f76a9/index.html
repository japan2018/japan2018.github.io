If you use #Python you should know! 10 convenient libraries

Python has many standard and external libraries, but due to their abundance, it’s difficult to know which one to use. This time, I will introduce a Python library that is useful to know from among them.

Standard library

###datetime

This module handles dates and times. There are convenient and easy-to-use items such as date/time acquisition, character string ⇔ date conversion, date acquisition after n days, and date acquisition before n days. For example, you can easily get the current date and time by using a datetime object that can handle dates and times together.

from datetime import datetime
now = datetime.now()
print(now)
# 2016-04-13 02:29:50.464488

###shutil

A module that provides high-level file operations. It’s very easy to write files and operations on directories.

import shutil
shutil.copy("/src/src.txt", "/hoge/hoge.txt")

Copy the src.txt file to the hoge.txt file.

###collections

There are deque (performs append and pop at high speed), OrderedDict (ordered dictionary), defaultdict (dictionary with default value), Counter (dictionary with counter), etc. We will be grateful for the efficient implementation of algorithms and programming contests.

from collections import Counter
count = Counter('hogehoge')
print count
# Counter({'g': 2,'e': 2,'h': 2,'o': 2})

###pdb

This is a debugger that provides functions such as breakpoint setting and single step execution at the source line level. You can also run it in an interactive shell or as a script file such as .py.

###timeit

This module measures the execution time of a Python program. Since you can measure the time for each code, you can measure detailed performance.

import timeit
timeit.timeit('"-".join(str(n) for n in range(100))', number=10000)

External library

###tqdm

When you want to check the progress of the loop, you can easily create the following progress bar.

41%||█████████ | 41/100 [00:04<00:05, 10.00it/s]

###py2exe

This is a library that converts Python scripts into .exe for Windows. Mac version “py2app” and general-purpose “PyInstaller” are also famous.

###simplejson

JSON encoding/decoding library. There is “json” in the standard library, but it can be used in the same way as that, and it features faster operation.

import simplejson

###requests

An easy-to-use HTTP library. There is “urllib” in the standard library, but it was a little inconvenient to use. requests are very easy to use and are often recommended in recent books.

import requests
r = requests.get('URL')
print r.text

###pep8

Source code check tool. Python has a style guide called PEP8, which is a common coding convention. With pep8, your code will tell you where the violation is.
