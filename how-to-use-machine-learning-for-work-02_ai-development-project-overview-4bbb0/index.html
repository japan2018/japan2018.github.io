Overview of AI development

Although AI development is organized in the figure below, we will proceed in the process of “planning-PoC-development-operation”.

  • Planning: Clarify what you want to do with AI
  • PoC: Is the planned AI technically developable? Can the project be profitable? To check
  • Development: Build the “model” that is the heart of AI and implement it in the system
  • Operation: Maintain and operate the developed system so that it operates stably.

Where to use machine learning?

Now that we can see the whole process of AI development, let’s consider where to use machine learning. The answer is to use machine learning in “PoC” and “development”.

The “model” shown in the figure is AI itself, and machine learning skills are required to build a good model. For those who have forgotten what a model is, please refer to “Part 1: Understanding the Purpose of Machine Learning”.

Steps to build the model

In order to master machine learning, there are three major skills required. Specific programming will be organized in the third article. First of all, let’s understand the whole picture.

  • Data preprocessing
  • Model training
  • Model validation

Data preprocessing

The purpose of data preprocessing is to prepare the data so that the model can be trained.

Specifically, we will carry out three things, but essentially the analysis that also serves as “preparation” ** of machine learning is the main. Articles written by people with practical experience are often written as a field of data science instead of machine learning.

  • EDA (Exploratory Data Analysis): Capturing statistical overall trends in data
  • Processing of missing values and outliers: removing obstructive data that becomes noise
  • Feature extraction: Analyze where the law of data appears

model learning

From here, the technology of machine learning will appear in earnest. As mentioned in “Part 1: Understanding the Purpose of Machine Learning” To find out the law from a large amount of prepared data Is the biggest purpose.

The machine learning method prepared for that is the algorithm. If you liken it to cooking, you can think of it as “data = ingredients” ** and ** “algorithm = recipe” **.

You can make delicious dishes by cooking with the appropriate recipe according to the ingredients. Machine learning is the same as that, and if you use an appropriate algorithm for the data, you can make a model with good prediction accuracy (= good AI).
