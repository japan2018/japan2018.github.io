[Python] Expand escape sequences in here-docs

Mar 4, 2020 Python Windows JSON Python3 heredoc 

I'm addicted to it

Even if the escape sequence (\) is written in the here-document as it is, it is not expanded as an escape sequence.

It can be expanded by writing two consecutive characters like \\ or using a raw string.

>>> import json
>>>
>>> json_dict1 = json.loads('''
... {
... "id": 1,
... "name": "hoge",
... "discription": "Houji \" Hojo \ "Hoge"
...}
...''')
