[Python] Basic story of inheritance in Python (for beginners)

Jul 30, 2013 Python 

My acquaintance stumbled, so it's a basic story.

Introduction

For example, suppose you define the following class method in Python.

class Parent(object):
    name = "Parent"

    @classmethod
    def get_name(cls):
        return cls.name


class Child(Parent):
    name = "Child"

Now, what is output when called from Child at this time?

In: Child.get_name()
OUT: "Child"

A little commentary

Basically, the called instance or the class itself is bound to the first argument defined in class. Therefore, even if it is inherited, the class in the inheritance destination is included. So, for example, it is OK if you do not make such a strange definition.

class Child(Parent):
    name = "Child"
    get_name = Parent.get_name

Regarding this, since the method basically has the context “where it belongs to”, in this case, whatever the class is, the method is executed according to the context of the parent that is the reference source.
