#Crawling This time, crawling was done using “[Search for commuting/commuting time]” (https://sumaity.com/chintai/jikan/) of Smyty, and the search result was within 60 minutes to Kanamachi station.

  1. Save the total number of properties on the site in a text file
  2. Specify the URL of the first page and save as an HTML file
  3. Get the URL of the next page from pagination and transition
  4. Save the transition destination page as an HTML file
  5. Repeat steps 2 and 3 until there are no more pages

The crawling code is as follows.

import requests
from bs4 import BeautifulSoup
import time
import os
import datetime

def crawling():
    # Path of directory for saving html file
    dirname ='./html_files'
    if not os.path.exists(dirname):
        # Create directory if it does not exist
        os.mkdir(dirname)

    # Make the first page html
    url = "https://sumaity.com/chintai/commute_list/list.php?search_type=c&text_from_stname%5B%5D=%E9%87%91%E7%94%BA&cost_time%5B%5D=60&price_low=&price_high="
    response = requests.get(url)
    time.sleep(1)
    # Save to file
    page_count = 1 # page count
    with open('./html_files/page{}.html'.format(page_count),'w', encoding='utf-8') as file:
        file.write(response.text)

    # Acquisition of total number of properties (theoretical value)
    soup = BeautifulSoup(response.content, "lxml")
    num_bukken = int(soup.find(class_='searchResultHit').contents[1].text.replace(',', ``))
    print("Total number of properties within 60 minutes to school:", num_bukken)
    # Save the total number of properties in a text file as it will be used to check the inspection conditions when scraping
    path ='./data.txt'
    with open(path, mode='w') as f:
        f.write("{}\n".format(num_bukken))

    # Crawling after the second page, continue until there is no next page
    while True:
        page_count += 1

        # Find next url
        next_url = soup.find("li", class_="next")

        # Break when the next page is gone
        if next_url == None:
            print("Total number of pages:", page_count-1)
            with open(path, mode='a') as f:
                f.write("{}\n".format(page_count-1))
            break

        #Get url of next page and save as html file
        url = next_url.a.get('href')
        response = requests.get(url)
        time.sleep(1)
        with open('./html_files/page{}.html'.format(page_count),'w', encoding='utf-8') as file:
            file.write(response.text)

        #Prepare parsing to get url of next page
        soup = BeautifulSoup(response.content, "lxml")

        # Crawling progress output
        if page_count %10 == 0:
            print(page_count,'get page')

#Main function
if __name__ == "__main__":
    date_now = datetime.datetime.now()
    print("Start crawling:", date_now)
    crawling()
    date_now = datetime.datetime.now()
    print("End crawling:", date_now)

from bs4 import BeautifulSoup
import datetime
import csv
import re

#Regular expression to separate the address by prefecture and city
pat ='(...?? [prefecture]) ((?: Asahikawa | Date | Ishikari | Morioka | Oshu | Tamura | Minamisoma | Nasu-Shiobara | Higashimurayama | Musashimurayama | Hamura | Tokamachi | Joetsu | Toyama | Nonoichi | Omachi | Gamagori | Yokkaichi | Himeji | Yamatokoriyama | Hatsukaichi | Shimo> Matsu | Iwakuni | Tagawa | Omura | Miyako | Furano | Beppu | Saiki | Kurobe | Komoro | Shiojiri | Tamano | Shunan] | City|[^ City]{2,3}?) County (?: Tamamura|Omachi|.{1,5}?) [Machimura]|(?:.{1,4} City)?[^ Town]{ 1,4}? Ward |.{1,7}?[Municipalities])(.+)'

def scraping(total_page, room_num):
    # Initialize the number of properties
    room_count = 0

    # Prepare csv file (add header)
    with open('room_data.csv','w', newline='', encoding='CP932') as file:
        header = ['No','building_name','category','prefecture','city','station_num','station','method','time','age','total_stairs','stairs' ,'layout','room_num','space','south','corner','rent','unit_price','url']
        writer = csv.DictWriter(file, fieldnames=header)
        writer.writeheader()


    for page_num in range(total_page):
        #Scraping progress output
        if page_num %10 == 0:
            print(page_num ,'/', total_page)

        #Open html file for scraping with Beautiful Soup
        with open('./html_files/page{}.html'.format(page_num + 1),'r', encoding='utf-8') as file:
            page = file.read()
        soup = BeautifulSoup(page, "lxml")

        # Get information for each building
        building_list = soup.find_all("div", class_="building")
        for building in building_list:
            #Building Category: Mansion or Apartment or House
            buildingCategory = building.find(class_="buildingCategory").getText()

            # Building name
            buildingName = building.find(class_="buildingName").h3.getText().replace("{}".format(buildingCategory), "").replace("New", "")

            # Candidate extraction of the nearest station and the distance from the station
            traffic = building.find("ul", class_="traffic").find_all("li")
            # Number of nearest stations
            station_num = len(traffic)
            # Extract things with short walking time
            min_time = 1000000 # Initialize the minimum required time
            for j in range(station_num):traffic[j] = traffic[j].text
                figures = re.findall(r'\d+', traffic[j])
                time = 0
                for figure in figures:
                    #Calculation of time required
                    time += int(figure)
                # Save minimum time and index if minimum
                if time <min_time:
                    min_time = time
                    index = j

            #If you have information about the station or route
            if len(traffic[index].split(''))> 1:
                # Route decision
                line = traffic[index].split('')[0]
                # Determine the nearest station
                station = traffic[index].split('')[1].split('Station')[0]
                # Acquisition of transportation (bus, car, walking) to the station
                if len(traffic[index].split(''))> 2:
                    if "bus" in traffic[index].split('')[1]:
                        method = "bus"
                    elif "car" in traffic[index].split('')[2]:
                        method = "car"
                    else:
                        method = "walk"
                # No information about transportation to the station
                else:
                    method = None
            #If there is no information on the station or route
            else:
                station = None
                line = None
                method = None
                time = None

            # Street address
            address = building.find(class_="address").getText().replace('\n','')
            address = re.split(pat, address)
            if len(address) <3:
                prefecture = "Tokyo"
                city = "Adachi-ku"
            else:
                prefecture = address[1]
                city = address[2]

            #Building details (age, structure, total number of floors)
            building_detail = building.find(class_="detailData").find_all("td")
            for j in range(len(building_detail)):
                building_detail[j] = building_detail[j].text

            # ---- Get only the age value ----
            #Unknown
            if'unknown' == building_detail[0]:
                building_detail[0] = None
            # 0 years old
            elif'less than' in building_detail[0]:
                building_detail[0] = 0
            # Normal value
            else:
                building_detail[0] = int(re.findall(r'\d+', building_detail[0])[0])

            # Get only the number of total floors
            building_detail[2] = int(re.findall(r'\d+', building_detail[2])[0])


            # ---- Get room details ----
            rooms = building.find(class_="detail").find_all("tr",
                                                            ('class': ['estate applicable','estate applicable gray']})
            for j in range(len(rooms)):
                #Counting the number of properties
                room_count += 1

                # ---- Number of floors ----
                stairs = rooms[j].find("td", class_="roomNumber").text
                #Acquire only numerical values ("floor" deleted, missing value processing)
                if "-" == stairs:
                    stairs = None
                else:
                    stairs = int(re.findall(r'\d+', stairs)[0])

                # Make rent an integer
                price = rooms[j].find(class_="roomPrice").find_all("p")[0].text
                price = round(10000 * float(price.split('10,000')[0]))

                #Administration fee
                kanri_price = rooms[j].find(class_="roomPrice").find_all("p")[1].text
                Unification of # notation (deleting ten thousand yen notation, processing missing values for "-" and "0 yen")
                if "-" in kanri_price or "0 yen" == kanri_price:
                    kanri_price = 0
                else:
                    kanri_price = int(kanri_price.split('yen')[0].replace(',',''))

                #Room type (floor)
                room_type = rooms[j].find(class_="type").find_all("p")[0].text
                if room_type == "One Room":
                    room_type = "1R"
                # number of rooms
                num_of_rooms = int(re.findall(r'\d+', room_type)[0])


                #Room area, unit "m2" deleted
                room_area = rooms[j].find(class_="type").find_all("p")[1].text
                room_area = float(room_area.split('m')[0])

                #South facing corner room
                special = rooms[j].find_all("span", class_="specialLabel")
                south = 0
                corner = 0
                for label in range(len(special)):
                    if "south facing" in special[label].text:
                        south = 1
                    if "Corner room" in special[label].text:
                        corner = 1

                # Get detailed url
                room_url = rooms[j].find("td", class_="btn").a.get('href')

                # Rent = rent + management fee required
                rent = price + kanri_price

                # Calculate the rent (unit price) for each 1m^2
                unit_price = rent / room_area

                # Output to csv file: encoding default is "utf-8", "cp932" for Japanese in windows
                with open('room_data.csv','a', newline='', encoding='CP932') as file:
                    writer = csv.DictWriter(file, fieldnames=header)
                    writer.writerow(
                        {'No':room_count,'building_name':buildingName,'category':buildingCategory,'prefecture':prefecture,'city':city,'station_num':station_num,'station':station,'method':method,'time':min_time,'age':building_detail[0],'total_stairs':building_detail[2],'stairs':stairs,
                              'layout':room_type,'room_num':num_of_rooms,'space':room_area,'south':south,'corner':corner,'rent':rent,'unit_price':unit_price,'url':room_url})

    print("Acquired property data for {}.".format(room_count))
    #Check acceptance conditions
    if room_count == room_num:
        print("Clear acceptance conditions")
    else:
        print("There is a difference in {} items. The acceptance conditions have not been cleared.".format(abs(room_count-room_num)))

if __name__ == "__main__":
    date_now = datetime.datetime.now()
    print("Start scraping:", date_now)
    # Pass the total number of pages and the number of properties to the scraping function (acceptance condition)
    path ='./data.txt'
    with open(path) as f:
        data = f.readlines()
    scraping(int(data[1].replace("\n","")), int(data[0].replace("\n","")))
    date_now = datetime.datetime.now()
    print("End scraping:", date_now)

#Data visualization First, we checked the histogram to see how the rent is distributed, and considered that properties whose rent was too high would not be suitable for living alone.

Now let’s see how each variable affects rent.

Floor plan

Let’s look at the number of properties and the rent distribution for each floor plan.

A bar graph showing the number of properties for each floor plan shows that the floor plans from 1R to 3LDK account for 98% of the total. If you look at the distribution of rent for each floor plan on a violin plot, you can see that the distribution of rent differs for each floor plan. Therefore, the floor plan is likely to be a variable that affects rent.

place

Let’s see where there are many properties.

By prefecture, Tokyo was mostly Chiba and Saitama was around 3%. If you look at the details by municipality, Adachi-ku Katsushika-ku Matsudo-shi Kashiwa-shi Arakawa-ku has more than 1000 properties, so it looks good to search for properties. Let’s look at the distribution of rent by area.

Looking at the rent histogram by prefecture, we can see that there are many properties in Tokyo, but many have high rents, and many properties are cheaper in Chiba. If you take a closer look at the box plot of the rent for each city, you can see that the green boxes in the Chiba district are located at the bottom. You can find cheap properties in Matsudo, Kashiwa, Nagareyama, Ichikawa, Abiko, Yoshikawa, Soka. If you look at the boxplot, you can see that the distribution of rent differs depending on the area, so where the property is located is likely to affect the rent.

Time required from the station and its means

There is a weak negative correlation between travel time and rent, and it seems that rent decreases as travel time increases. Also, the difference in rent depending on the mode of transportation such as the bus and foot used at that time is illustrated. It can be seen that the blue rent is generally higher than the bus rent. Therefore, both transportation and travel time will affect the rent.

Age

Grouped the building ages every 5 years and put out a box plot of rent.

You can see that the rent is gradually decreasing from properties built 15 years onwards. Therefore, the building age is likely to be a variable that affects rent.

Total number of floors and type of building

Let’s look at the rent distribution by total floor number of buildings and a histogram of total floor numbers.

Looking at the total number of floors and the distribution of rent, it seems that the rent of the property up to the second floor is cheap. Looking at the histogram of the total number of floors, most of the two-story properties were apartments. Also, 95% of the properties are within 10 floors, so I think that you may have a longing for a higher-floor property for the first time living alone, but it seems difficult when it comes to living alone. From the above results, it was found that building information also affects rent.

facing south

I think that the characteristic of the property, facing south, will affect the rent. Here is a histogram of properties facing south and those not.

Looking at the histogram, the distribution seems to be the same, so I tested whether the difference in rent was significant. There was a significant difference in the average rent for properties facing south and those not. At this time, the F-test performed the equal-variance test, and the equal-variance was not rejected. Therefore, the t-test assuming the equal-variance was performed. As a result, the south-facing property is about 1500 yen cheaper. From the above, it turned out that whether or not it faces south has an effect on rent.

Corner room

In the same way, we will look at the effects on corner rooms as well.

Since the F-test performed a test of equal variances and rejected the equal variances, we performed a t-test assuming that the variances are not equal. As a result, it was found to be a significant difference, and the corner room was about 2000 yen higher. From the above, it was found that the corner room also affects the rent.
