Make a calculation drill print

May 1, 2020 Python PDF reportlab 

#Introduction

I made a calculation drill generation program to make a self-study print for elementary school lower grades. Randomly create mathematical formulas with python and convert to pdf with A4 paper size. Print the created file at home or at a convenience store, and if the child learns for himself, the purpose is achieved.

environment

  • Ubuntu 18.04 [Using WSL (Windows Subsystem for Linux) on Windows 10]
  • python3.6

Install ReportLab

In order to print the generated formula, use the library for Report PDF, ReportLab.

$ sudo pip3 install ReportLab

The detailed documentation for ReportLab is below. https://www.reportlab.com/docs/reportlab-userguide.pdf

Output image

This time, print out the 2-digit addition, subtraction, and multiplication.

How to use drawString and line functions for pdf output

The print creation this time is mainly realized using two functions, drawString() and line(). Before entering the program body, let’s summarize only the basic usage.


from reportlab.pdfgen import canvas
from reportlab.pdfbase import pdfmetrics
from reportlab.pdfbase.ttfonts import TTFont
from reportlab.lib.pagesizes import A4

Create # Canvas. Specify file name and size
c = canvas.Canvas("calc_train.pdf",pagesize=A4)
# Set output coordinates
x=0
y=0
#Write the string "hoge" to coordinates (0,0)
c.drawString(x,y,"hoge")
# Draw a straight line
c.line(x,y,x+10,y)
# pdf generate 1 page
c.showPage()
# Save file
c.save()

The origin of ReportLab coordinates is “lower left” of the page.

Calculation drill automatic generation program

The following is an automatic calculation drill generation program using ReportLab which is the purpose of this time.

import os, sys
import random

from reportlab.pdfgen import canvas
from reportlab.pdfbase import pdfmetrics
from reportlab.pdfbase.ttfonts import TTFont
from reportlab.lib.pagesizes import A4

#5x5 basic formula coordinates
x_list = [20, 130, 240, 350, 460]
y_list = [730, 580, 430, 280, 130]
# You can also specify a font
fontname = "Helvetica"
# If you want to output Japanese, set as follows.
#Please refer to your environment for the path.
#pdfmetrics.registerFont(TTFont(fontname,"/mnt/c/Windows/Fonts/msgothic.ttc"))

# Function that outputs one formula to the specified coordinates
#Estimated calculation has 2 digits or less for each element.
def Set_calc(c,x,y,ope,d1,d2):
    # c: canvas
    # x, y: coordinates
    #ope: string of mathematical symbols
    offset = 30
    _d1 = str(d1)
    _d2 = str(d2)
    # Adjust character width
    if(d1<10):
        _d1 = "" + _d1
    if(d2<10):
        _d2 = "" + _d2
    c.drawString(x+offset, y, _d1)
    c.drawString(x, y-offset, ope)
    c.drawString(x+offset, y-offset, _d2)
    c.line(x,y-offset-5,x+offset*2.5,y-offset-5)

# A function that generates a formula for one page
def Make_page(c,ope,d1_max,d2_max):
    # c: canvas
    #ope: string of mathematical symbols
    # d1_max, d2_max: Maximum value of calculated value
    c.setFont(fontname,20)
    # Output the character string of the header part
    c.drawString(5,800, "Name: Day: / ")
    c.setFont(fontname,30)

    # Set Value
    for row in y_list:
      for col in x_list:
        # Randomly generate elements within the set maximum value
        d1 = random.randint(1, d1_max)
        d2 = random.randint(1, d2_max)
        In case of # addition
        if(ope=="+"):
            Set_calc(c,col,row,"+ ",d1,d2)
        # For subtraction
        elif(ope=="-"):
            # Do not handle negative values
            if(d1<d2):
                tmp=d1
                d1=d2
                d2=tmp
            Set_calc(c,col,row,"- ",d1,d2)
        For # multiplication
        elif(ope=="x"):
            Set_calc(c,col,row," Ã— ",d1,d2)
        else:
            print("[Error] Not support operation:",ope)
    c.showPage()

Main function below #
print(" >>> Start")
c = canvas.Canvas("calc_train.pdf",pagesize=A4)

# Addition (maximum value of each element: 99)
Make_page(c,"+",99,99)
# Subtraction (max elements: 99 and 9)
Make_page(c,"-",99,9)
# Subtraction 2 (Maximum value of each element: 99)
Make_page(c,"-",99,99)
# Multiplication (maximum value of elements: 99 and 9)
Make_page(c,"x",99,9)
c.save()

print(" >>> Finish")

When the above sample is executed, 5x5 calculation formulas are output on A4 size paper. Please correct the calculation range according to your child’s grade and adjust the number of formulas per sheet before using.
