Purpose

A rich open source development environment has made it really easy to use machine learning. However, it may be caught in an unexpected pitfall because it can be easily used without studying. MALSS was developed especially for the purpose of automating the parts that beginners tend to get stuck and learning how to use machine learning. In addition, I think that even skilled users can contribute to efficiency by automating the introduction of analysis.

Target

The following two can be applied to the application of machine learning in business.

  • Data analysis case
  • ** System development project **

In data analysis projects, it is required to discover some knowledge from the data (data mining). On the other hand, in system development projects, a model that performs some kind of prediction using unknown data is used by incorporating it into the system. This tool is supposed to support the latter machine learning for system development projects. This is because I think that machine learning for system development projects has a larger part that can be patterned (automated) at the initial examination stage.

Also, as a classification of machine learning,

  • ** Supervised learning **
  • Unsupervised learning

there is. In supervised learning, the data consists of input and output pairs, the output is a numerical value, the regression that predicts it from the input, and the output is a categorical value, and which category is classified from the input There is a classification/classification to predict. In unsupervised learning, data consists of inputs only. This applies to clustering, which classifies inputs according to some criteria, and outlier detection. This tool is intended for the former supervised learning. This is also because I think that the part that can be patterned is larger in supervised learning.

Introduction

Dependency

MALSS depends on the following packages. We do not automatically download dependent packages during installation, so please take care of this manually. Anaconda is convenient because it contains all.

  • python (>= 2.7 or >= 3.4)
  • numpy (>= 1.10.2)
  • scipy (>= 0.16.1)
  • scikit-learn (>= 0.17)
  • matplotlib (>= 1.5.1)
  • pandas (>= 0.14.1)
  • jinja2 (>= 2.8)

Install

From the command line pip install -U malss You can install it with.
