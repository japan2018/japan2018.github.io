Problem

169. Majority Element

The difficulty level is easy. Excerpt from the Top 100 Liked Questions.

Find the majority of the elements in a given size array of n. Note that the array is not empty, and always has a majority of elements.

Example 1: Input: [3,2,3] Output: 3

Example 2: Input: [2,2,1,1,1,2,2] Output: 2

Both are returning the majority.

Solution

It is good to use the built-in function count, but I think it is easier to use that because there is a Counter function in Python.

The difference between the two is

# count: Can be used without importing. Count the number of each element.
nums = [2,2,1,1,1,2,2]
print(nums.count(0))
#0

print(nums.count(1))
#3

print(nums.count(2))
# Four

#counter: You can't use it without importing. Elements can be acquired in descending order of appearance frequency.
from collections import Counter
nums = [2,2,1,1,1,2,2]

num = collections.Counter(nums)

print(num)
# Counter({2: 4, 1: 3})

There are differences such as. This time, it is sufficient to return the one that occupies the majority, so consider using counter.

from collections import Counter
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        ans = collections.Counter(nums)
        return ans.most_common(1)[0][0]
