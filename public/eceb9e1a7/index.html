[Python] Create QR code in Python. I do not want to line up with the same statement every time I issue a commuter pass. ..

Jan 8, 2020 Python QRcode 

I went to issue (reissue) a commuter pass for commuting at the beginning of the year.

Great queue. .. ..

What is this? Disneyland-like.

I write the same thing every time. Line up. Troublesome.

This means that if you issue a QR code once and hold each with your smartphone, you can improve efficiency considerably. ?

First, install the module for creating QR code with pip.

$ pip install qrcode

After that, I asked for attribute information at the first time and made a QR code.

qrmake.py


import qrcode

name=input ('What's your name?')
address=input('What's your address?')
age=input('What's your age?')
tel=input('What's your phone number?')
start=input('What's your departure?')
goal=input ('Arrival?')
via=input('What is the waypoint?')

img=qrcode.make('Name:'+str(name) +'address:' +'Age:' +str(age) +'Telephone number' + str(tel) +'Starting point:' +str(start ) +'Arrival:'+str(goal) +'Via:' +str(via))
img.save('attribute information')
