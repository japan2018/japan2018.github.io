Jun 1, 2020 Python automation Outlook 

In a previous article, I showed you how to load folder and email information from Outlook.

Read Outlook emails with Python https://qiita.com/konitech913/items/8a285522b0c118d5f905

Even without doing so, I think there are cases where you want to read individual mail data (msg file) and read the information in the mail.

For example, you may want to save an “inquiry email” as a msg file in a folder on your PC and extract the information from that file and copy it to Excel.

Read msg file with Python

You need to import win32com.client to operate outlook. I’m using Anaconda and I was able to import it without any additional installation.

import win32com.client

Then create the Outlook object.

outlook = win32com.client.Dispatch("Outlook.Application").GetNamespace("MAPI")

Then, use the method OpenSharedItem("xxx.msg") to read the target msg file. Here, the file “This is a test.msg” is read.


mail = outlook.OpenSharedItem("This is a test.msg")

This mail is an instance that represents a mail. The meanings of the attributes are as shown in the table below.

Attribute Meaning
mail.subject Subject
mail.sendername From
mail.senderEmailAddress From email address
mail.receivedtime Received date and time
mail.body Body
mail.Unread Unread flag 

print(" Subject: ",mail.subject)
print("From: %s [%s]" %(mail.sendername, mail.senderEmailAddress))
print("Received date: ", mail.receivedtime)
print("Unread: ", mail.Unread)
print("body: ", mail.body)

Subject: This is a test

Unread: False

You can read it properly.
