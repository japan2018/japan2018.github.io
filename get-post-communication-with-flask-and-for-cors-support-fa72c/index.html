Communication in Json format

from flask import Flask, request, jsonfy
import json
app = Flask(__name__)

@app.route("/", methods=["POST"])
def test():
    data = json.loads(request.data.decode('utf-8')) # decode request.data to utf-8 and make it a dictionary type in json library
    return jsonfy(data) # Return as is for the sample

It seems that you can do the following, but I have not tried it yet.

from flask import Flask, request, jsonfy
import json

@app.route("/", methods=["POST"])

    return jsonfy(data) # Return as is for the sample

Form communication

from flask import Flask, request, jsonfy
app = Flask(__name__)

@app.route("/", methods=["POST"])
def test():
    data = request.form["hoge"] # Get the value of the ""hoge"" key present in the form
    return jsonfy(data) # Return as is for the sample

CORS (Cross-Origin Resource Sharing) support

This content is not particularly relevant, but next I will explain CORS compliance. I’m a web engineer, so I don’t know the details, but in a nutshell, it seems to be a rule to use another server resource (if I explain it a little more, whether the URL base is the same) I wonder if they are the same).

First of all, install “flask-cors” with pip.

pip install flask-cors

The rest is like below.

from flask import Flask
from flask_cors import CORS # and this
app = Flask(__name__)
CORS(app) # this
