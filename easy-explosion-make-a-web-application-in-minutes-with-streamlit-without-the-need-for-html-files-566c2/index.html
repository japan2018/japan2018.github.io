[Easy explosion] Make a web application in minutes with Streamlit without the need for HTML files

May 21, 2020 Python pip WebApps Streamlit 

## What is Streamlit?

Streamlit is an open source framework for Python.

Goal this time

For the time being, I would like to easily make something like a web application. The most difficult part in creating a web application is the processing of the back end and the creation of HTML files, but Streamlit seems to have no need to do those tasks at all. You can make a web page with just one Python file.

Install

It can be installed using pip.

$ pip install streamlit

Launch demo page

$ streamlit hello

The above command will immediately launch the demo screen from http://localhost:8501. It’s easy. ![Screenshot 2020-05-18 19.34.59.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/571279/5a319f31-a7ec-6dce-26a3-(69514231a105.png)

Web page creation

Create a Python file in an appropriate directory. You can name the file anything you like, but this time it’s officially named first_app.py.

$ touch first_app.py
$ cd first_app.py

Then write to this file. Since it’s the very first, let’s display Hello world.

import streamlit as st
st.title('MyApp')
st.write("HelloWorld")

Execute the following command on the directory containing first_app.py

$ streamlit run first_app.py

Probably the web page says Hello world. I’m a little impressed that you can create a web page with such a small amount of code without writing HTML or CSS files. You can also easily display headers and data frames.

import streamlit as st
import numpy as np
import pandas as pd

st.title("HelloWorld")
st.subheader("This is subheader")
st.write("This is testdata")
st.write(pd.DataFrame({
    'first column': [1, 2, 3, 4],
    'second column': [10, 20, 30, 40]
}))

As you can see, Streamlit has various methods like st.write() and st.title(). Please refer to Official Document for other methods.

magic commands

This is enough convenient because you can process the front without HTML file, In fact, streamlit has more innovative commands.

import streamlit as st
import numpy as np
import pandas as pd

"""
# My first app
### This is subheader

This is testdata
"""

st.write(pd.DataFrame({
    'first column': [1, 2, 3, 4],
    'second column': [10, 20, 30, 40]
}))

Make your app interactive with widgets

You can easily add check boxes and select boxes. As a test, select either Tokyo or Osaka in the select box so that you can hit the points scattered on the map. Then, if you check the check box, you can see the list of coordinate data.

locate = {"Tokyo":[35.68, 139.76] ,"Osaka":[34.70 ,135.49]}

# Select box
l = st.selectbox(
    "Which places do you like best?",
     ("Tokyo", "Osaka"))
'You selected:', l

# Plot the map
map_data = pd.DataFrame(
    np.random.randn(1000, 2) / [50, 50] + locate[l],
    columns=['lat','lon'])
st.map(map_data)

# Checkbox
if st.checkbox('Show dataframe'):
    st.write(map_data)

In addition, most commands can add each function to the left sidebar by adding st.sideber. st.selectbox → st.sidebar.selectbox st.checkbox → st.sidebar.chechbox And

Now it’s like a simple web app.

At the end

I was able to create a web application with a single Python file and a few lines of code. I felt that Django and Dash were fast enough in the same Python, but Streamlit was explosive. Besides, I embedded graphs, images, and videos with a single line of simple code. If you are not so particular about design and want to publish something that you developed quickly, I think that it is worth using it.
