High-level scripting language

By the way, the script language is an interpreter language (translating a program into machine language one line at a time and executing it). “High-level” means here object-oriented. In space astronomy, I think that python or ruby is the mainstream, but here python is treated as a king.

I will explain how to use python later, so I will only introduce general theory here. Since the interpreter language interprets sequentially, the execution speed is orders of magnitude slower than the compiler language introduced next. However, when performing simple processing such as vector and matrix operations like space astronomical astronomy, a program written in C language or fortran can be read only for that part. Many of python’s numerical calculation libraries called numpy use the legacy high-speed module of the past, but python is successful, but it has succeeded in realizing the fast processing. It’s a bit technical, but this is due to the technology called binding, which makes python talk to the C/C++ language. You can do it.

The idea of python is that no matter who writes it, it will be the same. This was a free world where the same processing could be written completely differently in the past such as perl and C language, but on the other hand, it was difficult to read and team development was difficult. Although the rules of python may be strict, the differences between individuals are suppressed to that extent, making the language readable by anyone.

For example, in C language, the code to calculate the difference sin(i+1)-sin(i) of sin(x) is as follows.

#include <stdio.h>
#include <math.h>
#define f(x) sin(x)
int main() {
    double x1,x2,dy;for (int i=0; i<100; i++) {
    x1 = i;
    x2 = i+1;
    dy = f(x2)-f(x1);
    printf("x, f'(x) = %f %f \n", x1,dy);
  }
}

If you write this in python,

#!/usr/bin/env python
import numpy as np
x = np.arange(101)
y = np.sin(x)
dy = y[1:]-y[:-1]
print("x1, dy =", x[:-1], dy)

You can write without a for loop like this. For python, it’s calculated using angle brackets. This is called a “slice”, and it is a mechanism that allows you to easily access array elements. For details, see [Python] Slice operation summary. Please refer.

In space astronomy, there are many vector and matrix operations, but in python, the operations can be written in an overwhelmingly simpler way than in the C/C++ language. The C/C++ language uses a lot of for loops, but in python, it is easier to operate and access an array without looping, as it is said that “don’t use a for loop” is more like python. Recently, many of the tools for machine learning are based on python. The young people who start from now on will start from python, so there will be no big mistake.

Tips to protect your laptop

  • Press the track pad deeply to release the click

Mechanical wear occurs when clicking with a deep press on the trackpad. For mac users, click System Preferences ==> Trackpad ==> Tap to check Click with one finger. Now you can click with a soft touch. (If you care more, you can use an external keyboard, mouse, or trackpad, but isn’t it a laptop?)

  • Do not leave the power on for a long time.

From the viewpoint of the life of the power supply, there are various opinions about whether to frequently turn the power supply on/off or reduce it. However, if it is obviously slow, there are many cases where dead apps are wasting energy just because they have not released memory, so I personally turn off the power about once a week.
