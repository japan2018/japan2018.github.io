#matplotlib style There are two styles of matplotlib.

It is a Pyplot interface that can be done with all plt. somehow, and an object-oriented interface written with ax.plot after defining fig and ax.

Let’s see a concrete example.

##Pyplot interface

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 1, 100)
y = x ** 2
plt.plot(x, y)
plt.show()

Object-oriented interface

x = np.linspace(0, 1, 100)
y = x ** 2
fig = plt.figure()
ax = fig.add_subplot(111)
ax.plot(x, y)
plt.show()

When using the object-oriented interface, you need to understand the following hierarchical structure. In this way, you can see that there is a hierarchical structure in which the Axes object exists within the figure object. In other words, the above code creates a figure object with fig = plt.figure() and an axes object with ax = fig.add_subplot(111).

As an image, remember that a figure object is like a browser window and an axes object is like a browser tab.

Draw multiple graphs

Do the following to draw multiple graphs.

x_1 = np.linspace(0, 1, 100)
y_1 = x_1
x_2 = np.linspace(0, 1, 100)
y_2 = x_2 ** 2
x_3 = np.linspace(0, 1, 100)
y_3 = x_3 ** 3
x_4 = np.linspace(0, 1, 100)
y_4 = x_4 ** 4
fig = plt.figure()

ax_1 = fig.add_subplot(221)
ax_2 = fig.add_subplot(222)
ax_3 = fig.add_subplot(223)
ax_4 = fig.add_subplot(224)

ax_1.plot(x_1, y_1)
ax_2.plot(x_2, y_2)
ax_3.plot(x_3, y_3)
ax_4.plot(x_4, y_4)

plt.show()

4 axis objects are created in one figure object.

The argument of add_subplot() is (row, column, what number), and in the above example, the data of 2 rows and 2 columns is generated, and the axes object is generated in order.

#Adjust graph size You can adjust the size of the graph by passing figsize as an argument when creating the figure object.

x = np.linspace(0, 1, 100)
y = x ** 2
fig = plt.figure(figsize=(12, 8))
ax = fig.add_subplot(111)
ax.plot(x, y)
plt.show()

Generate #figure and ax objects at the same time Until now, a figure object was created and then an ax object belonging to it was created, but you can create a figure object and an ax object at the same time as follows.

fig, axes = plt.subplots(2, 2, figsize=(20, 12))
plt.show()

Since we have not plotted anything yet, the empty box above is generated. Let’s look at the contents of the data.

print(axes)
print(axes.shape)

Thus, the axes object stores 2x2 data. Each one corresponds to the above four graphs.

You can plot it with:

x_1 = np.linspace(0, 1, 100)
y_1 = x_1
x_2 = np.linspace(0, 1, 100)
y_2 = x_2 ** 2
x_3 = np.linspace(0, 1, 100)
y_3 = x_3 ** 3
x_4 = np.linspace(0, 1, 100)
y_4 = x_4 ** 4
fig, axes = plt.subplots(2, 2, figsize=(20, 12))
axes[0][0].plot(x_1, y_1)
axes[0][1].plot(x_2, y_2)
axes[1][0].plot(x_3, y_3)
axes[1][1].plot(x_4, y_4)

plt.show()

convert axes to one-dimensional array

Since the axes are two-dimensional data, it’s difficult to rotate with a for statement.

By converting it into a one-dimensional array using ravel(), it will be easier to turn it in a for statement. See the example below.

fig, axes = plt.subplots(2, 2, figsize=(20, 12))
one_dimension_axes = axes.ravel()
x = np.linspace(0, 1, 100)
for i, ax in enumerate(one_dimension_axes):
    ax.plot(x, x ** (i+1))

plt.show()

Draw multiple lines

You can draw multiple lines by plotting with the same ax object.

fig = plt.figure(figsize=(12, 8))
ax = fig.add_subplot(111)
x_1 = np.linspace(0, 1, 100)
y_1 = x_1
x_2 = np.linspace(0, 1, 100)
y_2 = x_2 ** 2
ax.plot(x_1, y_1)
ax.plot(x_2, y_2)
plt.show()

Set the title, axes, and legend.

fig = plt.figure()
ax = fig.add_subplot(111)
x_1 = np.linspace(0, 1, 100)
y_1 = x_1
x_2 = np.linspace(0, 1, 100)
y_2 = x_2 ** 2
ax.plot(x_1, y_1, label='y=x')
ax.plot(x_2, y_2, label='$y=x^2$')
ax.set_xlabel('x_value')
ax.set_ylabel('y_value')
plt.legend(loc='best')
plt.show()

You can create a label by using plt.xlabel etc., but this time, I set the label using set_xlabel to show that the label belongs to the axes object.

The label is stored in the axis object after the axes object.

Let’s create an axis object and check the label variable in it.

xax = ax.xaxis
yax = ax.yaxis
print(xax.label)
print(yax.label)

Text(0.5, 0,‘x_value’) Text(0, 0.5,‘y_value’)

Change line color, thickness and style

Please refer to the article here for color.

fig = plt.figure()
ax = fig.add_subplot(111)
x_1 = np.linspace(0, 1, 100)
y_1 = x_1
x_2 = np.linspace(0, 1, 100)
y_2 = x_2 ** 2
ax.plot(x_1, y_1, label='y=x', color='r', linewidth=2, linestyle='solid')
ax.plot(x_2, y_2, label='$y=x^2$', color='g', linewidth=2, linestyle='dashed')
ax.set_xlabel('x_value')
ax.set_ylabel('y_value')
plt.legend(loc='best')
plt.show()

You can change the color and style by specifying the argument of ax.plot as above.

#How to write a graph with two axes on the left and right The first plot can be written in the usual way, the second plot can be drawn using ax generated using twinx() to plot a graph with a common x axis and different y axes.

fig = plt.figure()
ax_1 = fig.add_subplot(111)
ax_2 = ax_1.twinx()
x = np.linspace(0, 5, 100)
y_1 = x
y_2 = x ** 2
ax_1.plot(x, y_1, label='y=x', color='r', linewidth=2, linestyle='solid')
ax_2.plot(x, y_2, label='$y=x^2$', color='g', linewidth=2, linestyle='dashed')
ax_1.set_xlabel('x_value')
ax_1.set_ylabel('y=x')
ax_2.set_ylabel('$y=x^2$')
fig.legend(loc='lower right')
plt.show()

#Scatter plot You can generate a scatter plot by using scatter instead of plot.

df = pd.DataFrame(data=np.random.randn(200, 2),
                  columns=['A','B'])
fig = plt.figure()
ax = fig.add_subplot(111)
ax.scatter(df['A'], df['B'])
plt.show()

Change of dot size, style, color

Here is the code:

df = pd.DataFrame(data=np.random.randn(200, 2),
                  columns=['A','B'])
fig = plt.figure()
ax = fig.add_subplot(111)
ax.scatter(df['A'], df['B'], s=100, c='r', alpha=0.5)
ax.set_xlabel('$x_0$', fontsize=10)
ax.set_ylabel('$x_1$', fontsize=10)
ax.set_title('title')
plt.show()

The size of dots is set by s in scatter, the color is set by c, and the transparency is set by alpha.

Change the size and color of dots according to the data

Here is the code:

df = pd.DataFrame(data=np.random.randn(200, 2),
                  columns=['A','B'])
df['data_size'] = np.random.randint(1, 100, 200)
fig = plt.figure()
ax = fig.add_subplot(111)
ax_cmap = ax.scatter(df['A'], df['B'], s=df['data_size']*10, c=df['data_size'], cmap='viridis', alpha=0.5)
fig.colorbar(ax_cmap)
ax.set_xlabel('$x_0$', fontsize=10)
ax.set_ylabel('$x_1$', fontsize=10)
ax.set_title('title')
plt.show()

data_size is specified in c of ax.scatter and viridis is specified in cmap. viridis is a colormap commonly used in matplotlib.

You can generate the right colorbar by passing the mappable object that created the colormap to fig.colorbar.

#bar graph Create a bar chart. Here is the code:

df = pd.DataFrame({'C1': ['A','A','A','B','B','C'],
                   'C2': [1, 2, 4, 5, 7, 7],
                   'C3': [1, 12, 7, 4, 8, 9]})
fig = plt.figure()
ax = fig.add_subplot(111)
ax.bar(df['C1'].value_counts().index, df.groupby('C1').sum()['C2'])
plt.show()

By df['C1'].value_counts().index, A,B,C which is the index of C1 element is received.

By df.groupby('C1'), it is grouped by C1 element, sum() is calculated, and C2 element in it is received.

Create multiple bar charts

Generate multiple graphs as follows.

df = pd.DataFrame(data=100 * np.random.rand(5, 5), index=['A','B','C','D','E'],
                  columns=['C1','C2','C3','C4','C5'])

fig = plt.figure(figsize=(12, 8))
ax = fig.add_subplot(111)
x = np.arange(len(df))
bar_width = 0.4
ax.bar(x, df['C1'], color='b', width=bar_width, label='C1')
ax.bar(x + bar_width, df['C2'], color='r', width=bar_width, label='C2')
plt.xticks(x + bar_width/2, df.index, fontsize=20)
plt.legend()
plt.show()

As an argument of ax.bar, first give the value of x, then the value of y, color the bar color, width the width, label the label.

xticks takes (x coordinate, name, fontsize=character size) as an argument.

#histogram Here is the code:

df = pd.DataFrame(data=np.random.randn(200), columns=['A'])
fig = plt.figure(figsize=(12, 8))
ax = fig.add_subplot(111)
ax.hist(df['A'], bins=15, rwidth=0.8, color='b')
plt.show()
`rwidth` specifies the width of the bar, which is 1 by default. When it is 1, it overlaps with the horizontal bar, so 0.8 is specified this time to make it easier to see.

The number of bars in the histogram is adjusted by bins.

#Heat map Here is the code:

fig = plt.figure(figsize=(12, 8))
ax = fig.add_subplot(111)
c_ax = ax.pcolor(df, cmap='Blues')
fig.colorbar(c_ax)
plt.show()

You can create a heatmap using pcolor.

You can add a color bar by passing the return value after using pcolor to the argument of fig.colorbar.

#About graph style I referred to the article of here.

x = np.linspace(0, 1, 100)
y = x ** 3
plt.rcParams["font.family"] = "arial" # set overall font
plt.rcParams["xtick.direction"] = "in" #x axis tick lines inward
plt.rcParams["ytick.direction"] = "in" #y axis graduation lines inward

plt.rcParams["ytick.minor.visible"] = True # Add y-axis minor tick marks
plt.rcParams["xtick.major.width"] = 1.5 # Line width of x-axis major tick lines
plt.rcParams["ytick.major.width"] = 1.5 #y-axis major tick line width
plt.rcParams["xtick.minor.width"] = 1.0 # Width of the x-axis minor tick marks
plt.rcParams["ytick.minor.width"] = 1.0 #y-axis auxiliary scale line width
plt.rcParams["xtick.major.size"] = 10 # x-axis major tick line length
plt.rcParams["ytick.major.size"] = 10 #y axis major tick line length
plt.rcParams["xtick.minor.size"] = 5 # x-axis minor scale line length
plt.rcParams["ytick.minor.size"] = 5 # y-axis minor scale line length
plt.rcParams["font.size"] = 14 # font size
plt.rcParams["axes.linewidth"] = 1.5 # Box thickness
fig = plt.figure(figsize=(12, 8))
ax = fig.add_subplot(111)
ax.plot(x, y)
ax.set_xlabel('X')
ax.set_ylabel('Y')
fig.savefig('test_5.png', bbox_inches="tight", pad_inches=0.05) # Save image
plt.show()

As above, plt.rcParams allows you to format the entire graph. You can also save the image with a good feeling by fig.savefig('test_5.png', bbox_inches="tight", pad_inches=0.05).
