Automatic login to ServiceNow with Selenium Web Driver

Nov 15, 2019 Python Selenium selenium-webdriver ServiceNow 

# ◆Overview
  • Using Selenium WebDriver, which is used for automation of Web UI tests, I tried to log in ServiceNow automatically.

◆Operating environment

  • OS: Windows10
  • Python version: 3.8.0

Preparation

Creating an instance of ServiceNow

  • You can create a developer instance by registering an account from the ServiceNow Developers page.
  • Make sure to write down the instance URL and the initial information of the admin account when the creation is completed.

Python execution environment preparation

  • Download and install the installer from Python official site.
    • Make sure that pip is included.

Preparing Selenium

  • Install Selenium with pip.
    • > pip install selenium
  • Download the driver for the browser you want to run from the Selenium official website.
    • Download “Google Chrome Driver” to run on Google Chrome this time.
    • Install the downloaded “chromedriver.exe” in any location.

◆ Try operating ServiceNow

Start browser

  • Specify Google Chrome as the browser and start it.
    • Incidentally maximize the window
from selenium import webdriver

# Launch Google Chrome
driver = webdriver.Chrome(executable_path ='Chrome Driver path')
# Maximize window
driver.maximize_window()

Access the ServiceNow login page

  • Specify the URL for each instance to access.
# Access to ServiceNow login page
driver.get('https://<ServiceNow instance name>.service-now.com/')
driver.implicitly_wait(20)

Login form element acquisition

  • Since the login form part is inside the iframe, if you specify id as it is, it will not be found and an error will be emitted.
    • It is necessary to get the iframe element first and move the frame with switch_to_frame.
    • When the operation is completed, exit from the iframe with switch_to.default_content().
# Get the iframe element that contains the login form
login_iframe = driver.find_element_by_xpath("//*[@id='gsft_main']")
# Move to specified iframe
driver.switch_to_frame(login_iframe)

# Get user name form element & enter user name
user_name_form = driver.find_element_by_id('user_name')
user_name_form.send_keys("<username>")

# Get password form element & enter password
password_form = driver.find_element_by_id('user_password')
password_form.send_keys("<password>")

# Get login button element & click login button
login_button = driver.find_element_by_id('sysverb_login')
login_button.click()

driver.implicitly_wait(20)

# exit from iframe
driver.switch_to.default_content()
