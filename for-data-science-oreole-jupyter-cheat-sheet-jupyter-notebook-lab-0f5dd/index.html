Development

I want to reload the modified .py module

Modules that have already been imported will be cached, or will not be updated when imported normally. ** You can reload (reimport) with the following. **

import importlib
importlib.reload(hoge)
# hoge is an imported module

I didn’t know this much until recently, and I restarted every time, so it’s really a scale. There is also How to update module automatically with %autoreload.

I want to change the tab name of JupyterLab

If you have multiple servers running, you often don’t know which tab is which server’s notebook. You can change the tab name as follows.

Run the following in Jupyter

%%javascript
document.title='Jupyter-GPU'

Alternatively, it can be specified at build time.

jupyter lab build --name='Jupyter-GPU'

Reference: https://github.com/jupyterlab/jupyterlab/issues/4422#issuecomment-395962448

Log

I want to debug with pdb

There are two timings for starting the debugger.

Specify a breakpoint and start the debugger Insert the code below

from IPython.core.debugger import Pdb; Pdb().set_trace()

Start the debugger when a bug occurs Applies only to specific cells

# Insert at the beginning of the cell you want to debug
%%debug

Applies to entire notebook

# Insert it somewhere in the notebook
%pdb on
# If you want to turn off the bug detection mode,
%pdb off

I want to prevent the DataFrame from being omitted

Increase the maximum number of characters that can be displayed or displayed in one cell.

import pandas as pd
pd.set_option("display.max_colwidth", 500) #1 Enter 500 characters in cell
pd.set_option("display.max_rows", 100) # can display 100 rows

** * If set_option does not work in JupyterLab, it will work well by displaying the records under max_rows like df[:100] (from my personal experience)**

If you want to prevent omission of only a specific cell, do the following (thanks to @chik_taks!)

with pd.option_context('display.max_colwidth', 200):
    display(df)
