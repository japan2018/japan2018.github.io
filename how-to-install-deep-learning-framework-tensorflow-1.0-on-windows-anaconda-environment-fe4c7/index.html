Installation environment

  • Windows 10 (64bit)
  • Anaconda 4.3.0 (64bit)
  • Python 3.5 (creates a virtual environment)
  • Tensorflow 1.0 (will be installed later)

Procedure

How to install Anaconda

Although the version of Anaconda is different, the procedure is the same, so please refer to the installation of Anaconda at the following URL. http://qiita.com/tomochiii/items/c17505872781c201d7b1

Install Tensorflow

Preparing Python 3.5

Tensorflow currently only supports Python 3.5.x. If you install the latest version of Anaconda as described above, the Python version installed will be 3.6. Therefore, I created a Python 3.5 environment on Anaconda as a virtual environment. If you have selected Python 3.5 as the Anaconda environment and installed it, you can skip this step. In order to create a Python 3.5 virtual environment for Tensorflow, start the Anaconda prompt and execute the following command. Those who develop with Jupyter Notebook are recommended to install jupyter at the same time.

Create virtual environment for #tensorflow, name parameter can be anything
$ conda create --name=tensorenv python=3.5
# You can install at the same time by listing the package name after
$ conda create --name=tensorenv python=3.5 jupyter numpy <package name> ...
# Check if the virtual environment is created
$ conda info -e

If the name specified by conda create is in the list, it is successful. Use the following commands to move to and from the virtual environment. For the name, specify the one in the list.

# How to move to virtual environment
$ activate tensorenv
#How to exit from the virtual environment
$ deactivate tensorenv

If you accidentally create it, you can delete the virtual environment with the following command.

# Delete virtual environment
$ conda remove --name=tensorenv --all

Install Tensorflow

There are two commands to install Tensorflow. You can install with either:

  • CPU version Tensorflow installation command.
#tensorflow installation command 1
$ pip install --upgrade tensorflow
#tensorflow install command 2
$ pip install --ignore-installed --upgrade https://storage.googleapis.com/tensorflow/windows/cpu/tensorflow-1.0.0-cp35-cp35m-win_amd64.whl

Install command for GPU version Tensorflow. This is not confirmed to work.

#tensorflow installation command 1
$ pip install --upgrade tensorflow-gpu
#tensorflow install command 2
$ pip install --ignore-installed --upgrade https://storage.googleapis.com/tensorflow/windows/gpu/tensorflow_gpu-1.0.0-cp35-cp35m-win_amd64.whl

The latest version of tensorflow can be installed with command 1, and the specified version can be installed with command 2.

Installation confirmation

Finally, check if the installation was successful. Start Python from the command line and see if Tensorflow can be imported.

# Start Python
$python
>>> import tensorflow as tf
>>> hello = tf.constant('Hello, TensorFlow!')
>>> sess = tf.Session()
>>> print(sess.run(hello))

Execute the above script and if “Hello, TensorFlow!” is displayed without any error, installation is complete!
