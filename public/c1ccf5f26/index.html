Increase to 100 questions

Next, we will increase the number of problems. The format is 5x20, but I want to make the indents neat, so I’m going to use half-width spaces to align the indents. The code is as follows.

#Problem increase code

def question_out_txt(): #function that outputs the question
    for cnt in range(100):
        num1 = random.randint(1,9)
        num2 = random.randint(1,9)
        if (cnt + 1) / 10 <1: In case of 1 digit
            question = "(" + str(cnt + 1) + ")" + \
                       str(num1) + "+" + \
                       str(num2) + "="
        else:
            question = "(" + str(cnt + 1) + ")" + \
                       str(num1) + "+" + \
                       str(num2) + "="
        answers.append(num1 + num2)
        if (cnt + 1) %5 == 0:
            f.write(question + "\n")
        else:
            f.write(question + "")

Here, the indentation shifts depending on whether the problem number is one digit or two digits, so the indents are aligned by specifying the number of half-width spaces in the conditional branch. When printing the problem, I am breaking the problem at multiples of 5.

def answer_out_txt(): #function that outputs the answer
    f.write("\nA.\n")
    for i,answer in enumerate(answers,1):
        if (i / 10 <1) and (answer / 10 <1): # number 1 digit, answer 1 digit
            out_ans = "(" + str(i) + ")" + \
                      str(answer) + ""
        elif (i / 10 <1) and (answer / 10 >= 1): # number 1 digit, answer 2 digits
            out_ans = "(" + str(i) + ")" + \
                      str(answer) + ""
        elif (i / 10 >= 1) and (answer / 10 <1): # number 2 digits, answer 1 digit
            out_ans = "(" + str(i) + ")" + \
                      str(answer) + ""
        else: # 2 digit number, 2 digit answer
            out_ans = "(" + str(i) + ")" + \
                      str(answer) + ""

        if i %5 == 0: Line feed for each problem number that is a multiple of #5
            f.write(out_ans + "\n")
        else:
            f.write(out_ans + "")

The point is to process the answer output. The number of digits of each of the question number and the answer is one digit or two digits, so there are 4 ways of conditional branch in 2^2. Again, the problem numbers are broken into multiples of 5.
