Get Boolean with Flask request

Nov 19, 2019 Python Flask Python3 

#Environment
  • python 3.7.0
  • Flask 1.0.2

#First SOURCE API that only receives and returns bool type is_hoge with request

from flask import Flask, request
app = Flask(__name__)

@app.route("/", methods=['GET'])
def hello():
    is_hoge = request.args.get('is_hoge', type=bool)
    return f'{ is_hoge }'

if __name__ == "__main__":
    app.run()

What do you say? As a result, True is returned. I think that all of the new paper papers were considered as character strings and True was returned.

I want to be able to use it as a Boolean somehow

It seems that python has a convenient function called strtobool().

from distutils.util import strtobool

def convert_to_bool(target_object, default_val):
    if target_object is None:
        return default_val
    try:
        return strtobool(target_object)
    except:
        return default_val
