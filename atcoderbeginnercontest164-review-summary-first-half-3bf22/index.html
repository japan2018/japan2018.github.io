Problem statement Lottery is done $N$ times, and at the $i$ time, I got a prize whose type is represented by the character string $S_i$. How many kinds of prizes have you got?

If you use a dictionary type, you are done. I’m lucky because I’ve been using it for natural language processing. If len is used in python, not only list but also the number of keys in the dict is returned, which is really convenient. Usually, when counting the frequency of occurrence of words

    if word in word_dict
        word_dict[word] += 1
    else:
        word_dict[word] = 1

Addendum: This time I use dict, but as the official explanation says, you can also implement it with set or map.
