## 02. “Patrol Car” + “Taxi” = “Pattokka See”

Get the character string “Patto Kukashi” by alternately connecting the letters “Patrol Car” and “Taxi” from the beginning.

###Go

package main

import (
"fmt"
"strings"
)

func main() {
var p1 string = "police car";
var p2 string = "Taxi";
var result string;

// make the character array
p1s := strings.Split(p1,"");
p2s := strings.Split(p2,"");

// Array number loop (match with "patrol car" because it has the same number of characters...)
for i := range(p1s) {
// Concatenate each character one by one
result += p1s[i] + p2s[i];
}

// display the result
fmt.Println(result);
}
