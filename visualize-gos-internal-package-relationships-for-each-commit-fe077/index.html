Overview

  1. It is difficult to understand programs that have complicated Go internal package relationships.
  • I want to visualize
  • Since it’s a big deal, I visualize it for each commit and make it an animation

#Preparation

  • Go
  • Python3
  • Graphviz
  • ImageMagick

#Code

  • st34-satoshi/for-each-commit

What you did

  1. Get the commit log
  2. Execute command for each commit
  3. Output the internal package relations of Go to dot faril and create a gif
  4. Align the gif size for each commit and create an animation

1. Get the commit log

You can get it with git log --pretty=oneline. python anyway

res = subprocess.check_output(["git", "-C", path_dir, "log", "--pretty=oneline"])

I want to get only the hash value of the commit (hash value + comment). Separated by white space, only the first hash value was acquired and made into a list.

2. Command execution for each commit

I ran the command with reset for each commit (is there a good way other than reset??).

for commit in log_list:
    subprocess.call(["git", "-C", path_dir, "reset", "--hard", commit]) |
    command(path_dir)

Use the path to the directory containing the program when executing the command.

3. Output the internal package relations of Go to dot faril and create a gif

3.1. Dot file creation

Here is the most difficult. There is a very convenient package called parser, and it will fetch the packages that are imported in the file. You can organize this from files to packages.

trouble

  • Per file –> per package The parser also gets the package name of the file, but it can’t tell if the same package name is in another directory. I saved the directory but there seems to be an easier way
  • Distinguish between external packages I wanted to target only internal packages (made by myself), but how should I distinguish them? ? I said “external if there is a github”, but it seems not to be a good way.

3.2. Create gif file from dot file

Use graphviz. dot -T gif sample.dot -o sample.gif

4. Make an animation by aligning the gif size for each commit

4.1. Align the gif sizes

When animating, if the size of each file is different, the large size will be cut off. mogrify -resize (width)x(height)! *.gif Select a size that you like.

4.2. Animation creation

Use ImageMagick. convert -delay 5 -loop 0 *.gif animation.gif
