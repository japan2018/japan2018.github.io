[Python] What do you do for visualization of 4 variables or more?

Mar 20, 2020 Python 可視化 Visualization plotly sankey-diagram 

# Purpose of this article

What if you want to check the relationship of many variables at once in data analysis? I think pairplot is the typical one, but it’s more! I thought I could summarize it, but recently I learned something like Sankey Diagram, so I drew it.

Added: Please read the additional part at the end of the article first.

How to use # Plotly Sankey Diagram

It seems that you can use Plotly, so first copy the sample code of official site and check if it works. The

import plotly.graph_objects as go

fig = go.Figure(data=[go.Sankey(
    node = dict(
      pad = 15,
      thickness = 20,
      line = dict(color = "black", width = 0.5),
      label = ["A1", "A2", "B1", "B2", "C1", "C2"],
      color = "blue"
    ),
    link = dict(
      source = [0, 1, 0, 2, 3, 3], # indices correspond to labels, eg A1, A2, A2, B1, ...
      target = [2, 3, 3, 4, 4, 5],
      value = [8, 4, 2, 8, 4, 2]
  ))])

fig.update_layout(title_text="Basic Sankey Diagram", font_size=10)
fig.show()

It moved! It is good that you can display the details of the part when you mouse over it!

Compared to matplotlib and seaborn, the code is long and tough, but the important part is the following part.

label = ["A1", "A2", "B1", "B2", "C1", "C2"],

source = [0, 1, 0, 2, 3, 3],
target = [2, 3, 3, 4, 4, 5],
value = [8, 4, 2, 8, 4, 2]

For example, in the Sankey Diagram diagram above, source:A1, target:B2, 2.00 corresponds to the orange parts of the three lists in the link below. It means that only 2 flows from label[0] to label[3].

If you can create a list that specifies the start and end points of nodes and the amount of flow that flows through them, you can draw Sankey Diagram!

Draw Sankey Diagram from data frame

So, I will start creating Sankey Diagram from the data frame of the main subject.

To show the results first, this time, I created the following figure using the data of Titanic.

Commentary

Load the library.

import numpy as np
import pandas as pd
import plotly.graph_objects as go

I will download the data.

!wget https://web.stanford.edu/class/archive/cs/cs109/cs109.1166/stuff/titanic.csv

Read the data and display only categorical variables and variables with integer values this time, so narrow down the variable names.

filename = "/content/titanic.csv"
df = pd.read_csv(filename, encoding='utf-8')

cate_list = ["Survived", "Pclass", "Sex", "Siblings/Spouses Aboard", "Parents/Children Aboard"]

n = len(cate_list)

Next, create label_list.

label_list = []
source_list = []
target_list = []
value_list = []

for cate in cate_list:
    tmp_label_list=[]
    for v in df[cate].unique():
        lab = "{0}={1}".format(cate, v)
        tmp_label_list.append(lab)
        tmp_label_list.sort()
    label_list.extend(tmp_label_list)

Create three lists of link information.

for i in range(n-1):
    source_cate = cate_list[i]
    target_cate = cate_list[i+1]

    for sc in df[source_cate].unique():
        for tc in df[target_cate].unique():

            v = sum((df[source_cate]==sc) & (df[target_cate]==tc))
            source_lab = "{0}={1}".format(source_cate, sc)
            target_lab = "{0}={1}".format(target_cate, tc)

            source_list.append(source_lab)
            target_list.append(target_lab)
            value_list.append(v)

Finally, since source_list and target_list must be specified by index, Convert by referring to label_list.

source_list = [label_list.index(si) for si in source_list]
target_list = [label_list.index(ti) for ti in target_list]

Then just run the same code as the sample.

fig = go.Figure(data=[go.Sankey(
    node = dict(
      pad = 15,
      thickness = 20,
      line = dict(color = "black", width = 0.5),
      label = label_list,
      color = "blue"
    ),
    link = dict(
      source = source_list,
      target = target_list,
      value = value_list
  ))])

fig.update_layout(title_text="Basic Sankey Diagram", font_size=10)
fig.show()

that’s all!
