Add CSV export function to admin screen with django-import-export

Nov 27, 2019 Python Django CSV management screen 

It seems that there are various plugins when I check the admin site of Django.

Since I wanted to import/export the data of the management screen in a format such as CSV, I checked it and it was easy with django-import-export. A memorandum at that time.

Install

First install with pip

$ pip install django-import-export
Configuration

Add import_export to INSTALLED_APPS

# settings.py
INSTALLED_APPS = (
    ...
    'import_export',
)
Added to admin.py

Add settings for the target data.

The sample model looks like this.

# models.py
class Book(models.Model):
    name = models.CharField('Book name', max_length=100)
    author = models.CharField('Book name', max_length=100)

django-import-export settings.

Add a class that inherits ModelResource for the target model. It seems that settings related information will be written here.

# admin.py
from django.contrib import admin
from import_export import resources
from import_export.admin import ImportExportModelAdmin

from .models import Book

class BookResource(resources.ModelResource):
    Setting django-import-export for # Model
    class Meta:
        model = Book


@admin.register(Book)
class BookAdmin(ImportExportModelAdmin):
    # Use ImportExportModelAdmin
    ordering = ['id']
    list_display = ('id','title','author')

    # django-import-exports settings
    resource_class = BookResource

Finally, prepare an Admin class that inherits ImportExportModelAdmin, Set a class that inherits ModelResource in resource_class OK!

Then, the button is displayed like this. Easy (‘ω`)

Small story

Export only: Disable Import

I thought I didn’t need to import anything, so I disabled it. It seems that you should only use ExportMixin.

# ...
from import_export.admin import ExportMixin

@admin.register(Book)
class BookAdmin(ExportMixin, admin.ModelAdmin):
    OK if you add #ExportMixin to admin.ModelAdmin
    ordering = ['id']
    list_display = ('id','title','author')

    # django-import-exports settings
    resource_class = BookResource
Specify exportable format

By default, you can choose JSON or YML, but CSV is all you need. I tried to narrow down the parts that can be selected. OK if you specify formats

# ...
from import_export.formats import base_formats

@admin.register(Book)
class BookAdmin(ImportExportModelAdmin):
    ordering = ['id']
    list_display = ('id','title','author')

    # django-import-exports settings
    resource_class = BookResource
    formats = [base_formats.CSV] # formats can be specified

that’s all!!
