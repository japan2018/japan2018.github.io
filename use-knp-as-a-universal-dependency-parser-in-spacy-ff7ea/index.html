Use KNP as a Universal Dependency parser in spaCy

Apr 16, 2020 Python NLP KNP camphr 

The other day, we released spaCy plugin [Camphr version 0.5.20](https://github.com/PKSHATechnology-Research/camphr/releases/tag/0.5.20).

The feature of this time is [universal dependency] from the analysis result of KNPimplementedbyProfessorKoichiYasuokaofKyotoUniversity.Itisafunctiontooutputthelabelof(https://universaldependencies.org/). In this article, I will briefly introduce how to use it.

Install

  1. Install knponthesystemorKNPdockerimage
  2. $ pip install camphr[juman]

#Analysis

import camphr
import spacy
nlp = camphr.load("knp")
doc = nlp ("Taro went to Mt. Fuji while eating an apple and a mandarin orange.")
spacy.displacy.render(doc)

“Apple” and “Mikan” are connected by conj(conjunct), which is a wonderful analysis result. About this conj, there is a commentary on Professor Yasuoka’s blog, so please read it.

It was easy, but this is all about the features. In addition to UD, Camphr provides KNP-based functions such as named entity extraction. Please read the following documents for details. Camphr KNP document
