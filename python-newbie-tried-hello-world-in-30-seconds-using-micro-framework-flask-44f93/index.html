What I did

** ① Flask installation **

** ② Make hello.py in your favorite folder ** ③ Paste the following

from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello_world():
    return "Hello World!"

if __name__ =='__main__':
    app.run()

④ Execution

python hello.py
