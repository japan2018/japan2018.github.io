[Python] How to revive a cell in iPython notebook

Dec 18, 2016 Python IPython 

# Problem

When working with Python, I often use the iPython notebook.

The trouble has happened recently. The cell where I wrote the script disappeared as if I had accidentally hit something on the keyboard.

The cause was that I pressed “x” (lowercase letters). iPython notebook seems to have “Edit mode” (state with focus in text box) and “Command mode” (state without focus in text box; press Esc to enter this mode). It seems that there was “Delete cell” in the shortcut for “mode”. That was the “x”.

You can check the shortcut list from Help as below.

Below is a list.

Solutions

The solution is simple. You can undo the deletion of cells by pressing “z” in “Command mode” in the shortcut!

I haven’t used the shortcut of iPython notebook so far, so I’d like to use it. If you don’t use it, you must be aware of its existence.
