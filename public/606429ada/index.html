Overview

When developing privately from packages that are often used at work, I have introduced the packages that are always installed. I think that people in the data science field use Jupyter, so it is suitable for web development. There are special packages that are useful when using Web frameworks such as Django, but they are not mentioned in this article. There are famous package managers such as Pipenv and poetry, but they are not written in this article.

Operating environment

Python package to introduce

flake8

https://pypi.org/project/flake8/


$ pip install flake8

It is a library that checks whether the code in the project is styled according to the python coding standard “PEP8”. It is one of the so-called Linters. There are ordinary “pep8” and “pyflakes” in Linter, flake8 has the functions of these two Linters, and can perform more strict coding checks. So, anyone can write the same code in the sense that it is good, and it leads to writing code with good visibility.

pyformat

https://pypi.org/project/pyformat/


$ pip install pyformat

It is one of the formatters. It automatically formats the code according to the Python coding standard. You can spend less time adjusting your coding style and focus on development. It will be powerful when used with flake8. There are other tools called “autopep8”, “yapf”, and “black”. Please choose according to your taste and project. I like the style of pyformat and use it.

isort

https://github.com/timothycrosley/isort


$ pip install isort

An automatic formatting tool that specializes in sorting imports.

In python

from hogehoge import foo
import sample

Use the library by importing. This import order will be sorted automatically. There is no normal formatter that arranges the order of import, so use “isort” to make up for it.

mypy

https://github.com/python/mypy


$ pip install mypy

For large projects, code becomes hard to understand in dynamically typed languages like Python. So using mypy, you can specify the type like a static language and check it.

bpython

https://bpython-interpreter.org/


$ pip install bpython

Code completion with a python interactive shell. Highlights the letters There is a similar one, “ipython”, but I use bpython because it is easy to see.

$ bpython

ipdb

https://pypi.org/project/ipdb/


$ pip install ipdb

Python has a standard debugger tool called “pdb”, but it doesn’t do code completion or highlighting. Therefore, “ipdb” is a highlight and complement to ipython-like.

Breakpoints can be set in the code with set_trace(), and variables declared within the scope of the code will be completed properly.

Finally

Install pip all at once

$ pip install flake8 pyfortmat isort mypy bpython ipdb

There are many similar libraries in Python, so it is easy to get lost in selecting the technology. I hope it will be helpful for efficient and pleasant development.
