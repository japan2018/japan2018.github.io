#1

Japanese: Basics of statistical learning (15,000 yen)

It’s commonly called a “castella book.”

  • Translated in Japan in 2014, but the original was published in 2001 and about 20 years ago. Even from the content, it was probably a revolutionary book that could systematically learn theory. It is just an “algorithm/theory dictionary”. It feels chronologically old, but the basis is always important. If you are not good at numbers, you will never be able to read. .. ..

Original: The Elements of Statistical Learning

Abbreviation: ELS

The thickness of the book, the color of the cover, exactly Castella, I want to go to Nagasaki. The contents are so heavy that I feel a stomach ache.

#2

Japanese: Pattern recognition and machine learning (up/down) (16,000 yen)

Common name “PRML”

  • If you start machine learning, this book is recommended as much as “Haji Pata (First Pattern Recognition)”. I hear it called “bishop book” around me.

    -It is not a book that learns algorithms completely like the first book, but theoretically introduces pattern recognition and its surrounding algorithms.

  • Mr. ctgk explained the problem in notebook format,[Anglo-CommunicationsSupportMaterial](https://herumi.github.io/Youcanalsoseethefamousworkfromeveryone'sassistance,suchasmakingit(prml/).

#3

Japanese: Introduction to statistical learning with R (7,000 yen)

Common name “ISL”

  • It seems that the name recognition is not so high compared to the above two books, but it is a well-known book. -As a concept, if “Basics of Statistical Learning: ESL” is a systematic book written for people who understand mathematics, this book is written more for beginners. That. -Certainly, the content comes from the foundation of an important idea in data science.

But it’s difficult for someone who doesn’t know anything. I think that it is better to have knowledge about probability distributions and regression analysis in advance so that you will not have to worry about it.

The text is long. I think it’s done carefully to make it easier to understand, but sometimes the story doesn’t come in.

Original: An Introduction to Statistical Learning with Applications in R

Original pdf

#4

Japanese: Deep learning (5,000 yen)

The topic of deep learning these days. A book written by Dr. ian Dai, the developer of the particularly hot generation model “GAN”. If you understand and understand “deep learning from scratch”, I would recommend this book as the next book. A book that advances while learning the knowledge of analysis and linear algebra necessary for deep learning in order.

Even if you can not read it, the cover may be beautiful so you can make it an interior?

Original: Deep learning

Original pdf

#5

Japanese: Statistical Causal Reasoning -Model, Reasoning, Guessing- (10,000 Yen)

I haven’t read much yet, but the book of the author who won the Turing Prize by developing causal reasoning. What is causal reasoning? When I thought about buying a book, I found this book and Iwanami Data Science.

Is it really a variable that should be included in the model when creating a statistical model? Does it really affect the forecast target? A book you want to read when thinking about.

General notes about model selection and linear regression will be learned in the third volume, “Introduction to Statistical Learning by R,” and the relationships between variables will be understood in this book.

Original: casuality models reasoning and inference

The Japanese translation is the first edition, and the original image of this is the second edition.
