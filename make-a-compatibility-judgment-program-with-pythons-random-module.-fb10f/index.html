#Program creation Let’s make it.

The goal this time is to enter the names of the two people you want to judge and to express the compatibility of those two as a percentage.

The random module is used for the% display part.

This module has a random function that displays a random number between 0 and less than 1. This time we will use this.

The following is a brief summary of program operations. ① Ask them to enter their names and receive them. ② Convert to% using random numbers ③ Output the result

When these are converted into a program, it becomes as follows.

import random#import random
male = input('Please enter the male name:')# ①
female = input('Please enter the female name:')# ②

p = random.random()# ②
p = p*100# ②
p = int(p)#% Round down the decimal point
print(male,'and',female,' are compatible with',p,'%')#③
