Poincaré series (loosely defined)

$$ V = \bigoplus_{k = 0}^\infty V_k $$ Let be the vector space with degree ($\deg V_k = k$). Then, **Poincare series **$P(V; t)$ of $V$ is transformed into the following formal power series $$ P(V; t) := \sum_{k = 0}^\infty (\dim V_k)\ t^k $$ It is defined by. That is, a formal power series in which the coefficient of each power $k$ of $t$ is equal to the dimension of the homogeneous component $V_k$ of $V$.

Example 1. (a fairly simple example)

Let $V = \mathbb{C}[x]$ (complex coefficient one-variable polynomial ring), and let $V_k, \ (k = 0, 1, 2…)$ be the total of $k$ homogeneous polynomials. I will try as a vector space. Then $\mathbb{C}[x]$ can be regarded as a vector space with degree, but since there is only one variable, the $k$ homogeneous polynomial is $a_k x^k, (a_k \in \mathbb{C} ) It only has the form $. That is, $$ \forall k, \ \ \dim V_k = 1 $$ Will be. Therefore, the Poincare series $P(\mathbb{C}[x]; t)$ of $\mathbb{C}[x]$ is $$ P(\mathbb{C}[x]; t) = \sum_{k = 0}^\infty t^k = 1 + t + t^2 + t^3 + \dots $$ I want it. Recalling the formula of the geometric series, this is $$ P(\mathbb{C}[x]; t) = \frac{1}{1-t} $$ You can write in a closed form. (Because it is a formal power series, it is written like this without considering convergence.)

Example 2. (2-variable polynomial ring)

Now consider the bivariate polynomial ring $V = \mathbb{C}[x, y]$. The $k$ homogeneous component $V_k$ of $V$ is the vector space formed by the entire $k$ homogeneous polynomial as above. In this case, $\dim V_k$ changes depending on the value of $k$. Looking from the small one,

\begin{align}
V_0 &= \text{Span}\langle 1 \rangle, \\
V_1 &= \text{Span}\langle x, y\rangle, \\
V_2 &= \text{Span}\langle x^2, xy, y^2\rangle, \\
V_3 &= \text{Span}\langle x^3, x^2y, xy^2, y^3\rangle, \\
&...
\end{align}

And you can see that $\dim V_k = k + 1$. Therefore $$ P(\mathbb{C}[x, y]; t) = \sum_{k = 0}^\infty (k + 1)t^k $$ is. This can be transformed nicely by using the term-wise differentiation,

\begin{align}
P(\mathbb{C}[x, y]; t) &= \sum_{k = 0}^\infty \frac{d}{dt}t^{k + 1} \\
&= \frac{d}{dt} \left( t \sum_{k = 0} t^{k} \right) = \frac{d}{dt} \frac{t}{1-t} = \ frac{1}{(1-t)^2}
\end{align}

You can write in a closed form.

Example 3. (3-variable polynomial ring)

Similarly, what if $V = \mathbb{C}[x, y, z]$, $V_k$ is a subspace spanning all $k$ homogeneous polynomials? $V_k$ is based on a homogeneous expression of the form $x^p y^q z^r$, $p + q + r = k$.

\frac{(k + 2)!}{k!2!} = \dbinom{k + 2}{2}

Since you can see that there are types, $$ P(\mathbb{C}[x, y, z]; t) = \sum_{k = 0}^\infty \dbinom{k + 2}{2} t^k $$ Will be. Well, can this be written in a closed form? It may be understood by considering the term-wise differentiation again, but this time we will consider a slightly different method. $$ \frac{1}{(1-x)(1-y)(1-z)} = \sum_{p = 0}^\infty x^p \sum_{q = 0}^\infty y^q \ sum_{r = 0}^\infty z^r = \sum_{p, q, r = 0}^\infty x^p y^q z^r $$ Considering the series, the basis $x^p y^q z^r$ of $\mathbb{C}[x, y, z]$ appears exactly once on the right side. $k$ The next homogeneous part is $x^{p_1} y^{q_1} z^{r_1} + x^{p_2} y^{q_2} z^{r_2} + … + x^{p_N} y^{q_N} z^{r_N}$ becomes $p_i + q_i + r_i = k, \ (i = 1, 2, …, N, \ \ N = \dim(V_k))$ I am. If we substitute $x = y = z = t$ here, it will be exactly like $(1 + 1 + … + 1)t^k = \dim(V_k)\ t^k$. $$ P(\mathbb{C}[x, y, z]; t) = \frac{1}{(1-t)(1-t)(1-t)} = \frac{1}{(1- t)^3} $$ I found out. That is $$ \frac{1}{(1-t)^3} = \sum_{k = 0}^\infty \dbinom{k + 2}{2} t^k \ \ \ \ \cdots \ (★) $$ is. (As can be inferred so far, $P(\mathbb{C}[x_1, …, x_n]; t) = \dfrac{1}{(1-t)^n for general $n$ } It becomes $. )

Numerical calculation (?)

By the way, I would like to confirm that this formula (★) is really likely to hold by actually calculating the first number term of the power series. What should I do?

In this article, I’ll try programming with Python.

First of all, the right side should calculate the binomial coefficient. The Python library SciPy has a function comb that computes the binomial coefficient. If we use this to calculate the first 10 terms,

from scipy.special import comb
combinations = [comb(k + 2, k, exact=True) for k in range(10)]
print(combinations)

You just have to run the program. Result is

[1, 3, 6, 10, 15, 21, 28, 36, 45, 55]

have become.

What about the left side?

General expression for Taylor expansion around $t = 0$

f(t) = f(0) + f'(0)t + \frac{1}{2!}f''(0)t^2 + \frac{1}{3!}f'''( 0)t^3 + ... = \sum_{k=0}^\infty \frac{1}{k!} \frac{d^kf(0)}{dt^k}t^k

Recalling that, the coefficient is the $k$ derivative at $t = 0$ on the left side divided by $k!$. Let’s calculate this by programming in Python. This kind of mathematical processing can be done using the library SymPy.Similarly,ifyoudecidetocalculateuptothefirst10terms,youneedtocalculatethe10thderivative(!) **, which is very complicated by hand, but you can easily find it by writing a program. I will. The actual program looks like this: (I used scipy to calculate the factorial.)

from sympy import *
from scipy.special import factorial
t = Symbol('t')
P = 1 / (1-t) ** 3
coeffs = []
for k in range(10):
    dk = diff(P, t, k).subs([(t, 0)])
    coeff = dk / factorial(k, exact=True)
    coeffs.append(coeff)

print(coeffs)

The execution result is still

[1, 3, 6, 10, 15, 21, 28, 36, 45, 55]

It is confirmed that the coefficients on the left and right sides are the same until the first 10 terms of the power series expansion.
