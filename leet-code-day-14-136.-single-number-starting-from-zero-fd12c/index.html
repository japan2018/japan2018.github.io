Problem

136. Single Number

The difficulty level is easy. This issue is also from the Top 100 Liked Questions. There are so many Goods, so why? I thought that, but when I tried to solve it, it became so obvious that it may be better to try solving it myself before actually seeing the solution.

The problem is that a non-empty array of numbers is given, so please extract and return the two numbers that are not displayed.

Example 1:

Input: [2,2,1] Output: 1

In this example, there is only 1 in the array, so 1 is returned.

Example 2:

Input: [4,1,2,1,2] Output: 4

In this example, there is only one 4 in the array, so 4 is returned.

Solution

This time I tried to write it so that I know a little about information systems.

class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        ans = 0
        for n in nums:
            ans ^= n
        return ans

If you are not familiar with Python

ans ^= n

Since it may be because it is something like this, it is an assignment operator that represents exclusive OR (XOR).

As most people know, an XOR that returns 1 when only one of the two inputs is 1 is well suited for this problem.

In fact, when I looked into Discuss after solving it, there were many answers using XOR. If there seems to be a better answer, I will add it.
