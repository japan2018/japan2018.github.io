How to confirm missing values (Kaggle:House Prices)

Nov 30, 2019 Python Visualization 

#at first

One of the important things in analyzing data is checking the data content. This time, we will introduce a method for confirming missing values that can be performed by non-engineers.

#Load dataset Import pandas and load the dataset. This time we will use the data from train.csv from House Prices: Advanced Regression Techniques from kaggle.

House Prices: Advanced Regression Techniques https://www.kaggle.com/c/house-prices-advanced-regression-techniques

import pandas as pd
data = pd.read_csv('../train.csv')

#Display items with many missing values Set the data you want to check in df. In this case, look at the train.csv set above.

#How to check missing values
Register the dataset to df=data #df
total = df.isnull().sum()
percent = round(df.isnull().sum()/df.isnull().count()*100,2)

missing_data = pd.concat([total,percent],axis =1, keys=['Total','Ratio_of_NA(%)'])
type=pd.DataFrame(df[missing_data.index].dtypes, columns=['Types'])
missing_data=pd.concat([missing_data,type],axis=1)
missing_data=missing_data.sort_values('Total',ascending=False)
missing_data.head(20)

print(missing_data.head(20))
print()
print(set(missing_data['Types']))
print()
print("---Categorical col---")
print(missing_data[missing_data['Types']=="object"].index)
print()
print("---Numerical col---")
print(missing_data[missing_data['Types'] !="object"].index)

#Visualize missing values You can use the code above to see the percentage of missing values. But where are the missing values, such as a time series dataset? There are times when I want to know. In that case, use heatmap.

import matplotlib.pyplot as plt
import seaborn as sns
sns.set_style('whitegrid')
%matplotlib inline

df = data
plt.figure(figsize=(16,16)) # size adjustment
plt.title("Missing Value") # title
sns.heatmap(df.isnull(), cbar=False) # Heat map display
