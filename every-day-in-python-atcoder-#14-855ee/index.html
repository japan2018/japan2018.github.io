Thoughts ABC081-B ABC081-B divides by 2 when $A_i$ is even. When it is an odd number, I quit immediately.

n = int(input())
a = list(map(int,input().split()))
counter = []
for i in range(n):
    count = 0
    if a[i] %2 == 0:
        while a[i] %2 == 0:
            a[i] //= 2
            count += 1
        counter.append(count)
    else:
        print(0)
        quit()

print(min(counter))

ABC087-B ABC087-B has a triple loop with thought stop.

a = int(input())
b = int(input())
c = int(input())
x = int(input())

ans = 0
for i in range(a+1):
    for j in range(b+1):
        for k in range(c+1):
            price = 500 * i + 100 * j + 50 * k
            if price == x:
                ans += 1

print(ans)

ABC083-B ABC083-B is investigating all N because N is about $10^4$. The i of $1\leq i \leq n$ is set to str, and each digit is summed and divided by if.

n, a, b = map(int,input().split())

ans = 0
for i in range(n+1):
    i = str(i)
    k = 0
    for j in range(len(i)):
        k += int(i[j])
    if k <= b and k >= a:
        ans += int(i)

print(ans)

ABC088-B ABC088-B will do the best for each other, so take the largest remaining card. So, I sort them and add them one by one. You don’t have to separate them for each player, but you can take the difference for each.

n = int(input())
a = list(map(int,input().split()))

a.sort(reverse=True)
alice = 0
bob = 0
for i in range(n):
    if i %2 == 0:
        alice += a[i]
    else:
        bob += a[i]

print(alice-bob)
