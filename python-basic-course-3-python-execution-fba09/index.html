## Run the program

Now let’s actually write the program. The meaning of each code will be explained in the following chapters. First, let’s execute the program we actually wrote.

Create a file with the name odd_or_even.py and write the following program. Note the indentation that is a feature of Python. Indentation can be tabs or single-byte spaces, but be careful about the number when using spaces.

def odd_or_even():
    for i in range(1,6):
        if i %2 != 0:
            print("{0} is odd".format(i))
        else:
            print("{0} is even".format(i))
        
    
if __name__ == "__main__":
    odd_or_even()

If Python is installed correctly, when you run the program you should see something like this:

1 is odd 2 is even 3 is odd 4 is even 5 is odd

This program is a program to judge whether the value from 1 to 5 is odd or even. When translated into Japanese, the above program does the following:

  • For numbers in the range 1 to 5
  • If the remainder of the number divided by 2 is not 0, “* number * is odd” is displayed.
  • Otherwise, display “number is even”

From the next page, we’ll get into Python’s notation. But before that, remember the following for future study.

print

The print statement is the statement to output characters in the program. Run the following program to understand how to use the print statement.

print("display statement")
print("display statement {0}".format("string"))
print("display {0} {1} {2}".format(1,"a","abc"))

Comment

A comment is a comment in the source code that is inserted in human language (Japanese or English). You are free to insert the sentence as it will be ignored by the computer when it does its work. When someone else reads your program, the code alone may not tell you what the program means. If you include a comment in that case, it will help the other person understand the intention of this site. Also, if you have a program that you do not want to run temporarily, you may comment it out. To make a line a comment, add “#” at the beginning. The following program outputs “taro” and “saburo”, not “jiro”.

print("taro")
#print("jiro")
print("saburo")

If you want to comment out multiple lines at once, enclose the statement with three double quotation marks (or single quotation marks).

print("1")
#print("2")
print("3")
print("4")
'''
print("5")
print("6")
'''
print("7")
print("8")
"""
print("9")
print("10")
"""

Make good use of comments.

Japanese output

Execute the following program. I think I get an error.

print ( "Hello World")

If you want to output Japanese in Python, add the following code to the first line.

# -*- coding: utf-8 -*-
print ( "Hello World")

I think the output will be correct. If you want to input and output Japanese in Python, utf-8 should be specified. *Although described in comments, Python understands the character code setting.
