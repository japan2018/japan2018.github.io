[Python] Add a custom dictionary to MeCab

Oct 29, 2016 Python mecab 

#Environment
  • MacOS 10.11 (El Capitan)
  • pip install mecab-MeCab installed with python

Create a dictionary

vi user_dic.csv

Usage, -1,-1,1, noun, general, *,*,*,*,*,*,*,mydic
Instruction manual, -1,-1,1, noun, general, *,*,*,*,*,*,*,mydic
...

User dictionary compilation

$ /usr/local/Cellar/mecab/0.996/libexec/mecab/mecab-dict-index \
- d /usr/local/lib/mecab/dic/ipadic \
- u user.dic \
- f utf-8 \
- t utf-8 user.csv
  • Meaning of options –d DIR: directory containing system dictionary –u FILE: create user file FILE –f charset: CSV character code –t charset: Character code of binary dictionary

Since it is said that

/usr/local/Cellar/mecab/0.996/libexec/mecab/mecab-dict-index \
- d /usr/local/lib/mecab/dic/ipadic \
- u user_dic.dic \
- f utf-8 \
- t utf-8 \
user_dic.csv

user_dic.dic is created.

Add dictionary

vi /usr/local/etc/mecabrc

userdic = /Users/.../user_dic.dic

I was able to add.

Operation check

$ mecab
instruction manual
Instruction Manual Noun, General, *,*,*,*,*,*,*,mydic
EOS

It looks like it’s done.
