Object oriented

One of the merits of object-oriented is the use of encapsulated objects that group data and methods together. The second point is that you can extend or modify the program by inheritance or overriding. Since you can be aware of object-oriented just by creating a class, there are two possible advantages.

If you’re not a programmer, you’ll find that strange program files are created when you continue copying and pasting software that has Python functionality from a book. Not bad, but it’s the owner of the program file that is having trouble modifying or expanding. As you can see, programming has a low threshold, but it has a lot of depth.
