Brief introduction

(For the introduction of this content, since the “purpose of this course” part written on the blog was easy to understand, I will quote from that part) This course aims to master the’basic’ usage of Python modules that are frequently used in data science, such as the environment construction necessary for doing data science in Python, the basics of Python, the basics of the Python library used for data science, and other is.

The aim of this course is

Master the basics of using tools, libraries, and modules to process data required for data science in Python Data can be processed without using a spreadsheet tool such as Excel Can process data files such as image files Daily data processing (such as Excel) can be automated with Python Such a place. Although some statistics are mentioned, please note that it is not a “learning data science course” but a “learning Python for data science” course.

However, since there are plenty of “techniques that can be used in the field” and “frequently used data science techniques” throughout the course, you can study data science in a broad sense.

Anyway, I intend to write it in a way that is easy to understand. I don’t think many difficult words come out, and I’m pretty clumsy to explain them, so I don’t think I’ll stop there.

As for what is introduced,

  • python basics

  • Pandas (for manipulating and analyzing data (makes spreadsheets faster than you can in Excel))
  • matplotlib (for graph drawing)
  • Seaborn (like matplotlib, you can write graphs, but it’s easier and cleaner)
  • Other useful libraries, modules, etc.
