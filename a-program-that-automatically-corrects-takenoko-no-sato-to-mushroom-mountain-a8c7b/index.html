#Implementation If you’re a sensible reader, I’m sure you’ve already pushed the “like button” ** by the time you’ve read this far, but I’ll also introduce the implementation just in case.

Simply replacing the “Mushroom Mountain” and “Takenoko no Sato” strings will not work. First of all, ** “This mountain is dangerous in case of fire” ** (This mountain is dangerous in case of fire) ** “Satomi Shonen who was beloved by that girl” ** (Satomi Shonen who was beloved by that girl) When such a sentence comes in, there is a possibility that you will make a wrong conversion without considering the delimiter. Then, since it corresponds to profanity to Mushroom Mountain, it is necessary to properly extract proper names such as “Mushroom Mountain” and “Takenoko no Sato”. Secondly, we have to separate the process depending on whether we are complimenting or degrading. You can judge that you are surely “Mushroom Mountain” if you praise it, and Not “Mushroom Mountain” if you are indebted.

By implementing natural language processing, these are implemented nicely. There are various ways to perform natural language processing, but this time, use COTOHA API.

By the way, although there are many language processing functions provided by COTOHA API, I will use “proper noun extraction API” and “emotion estimation API” among them.

  1. Determine whether the proper noun API includes “Mushroom Mountain” and “Takenoko no Sato”
  2. In the included sentence, the emotion estimation API is used to determine whether it is negative or positive. ** If “Mushroom Mountain” is included and is negative, character string replacement , ** “Takenoko no Sato” Is included and is positive, string replacement Will be processed. This time, we determined whether score is positive or negative using Score=0.5 as a threshold.

import requests
import json
import sys

BASE_URL = "https://api.ce-cotoha.com/api/dev/"
CLIENT_ID = "ID obtained with COTOHA API"
CLIENT_SECRET = "PASS obtained with COTOHA API"


def auth(client_id, client_secret):
    token_url = "https://api.ce-cotoha.com/v1/oauth/accesstokens"
    headers = {
        "Content-Type": "application/json",
        "charset": "UTF-8"
    }

    data = {
        "grantType": "client_credentials",
        "clientId": client_id,
        "clientSecret": client_secret
    }
    r = requests.post(token_url,
                      headers=headers,
                      data=json.dumps(data))
    return r.json()["access_token"]

def ne(sentence, access_token):
    base_url = BASE_URL
    headers = {
        "Content-Type": "application/json",
        "charset": "UTF-8",
        "Authorization": "Bearer {}".format(access_token)
    }
    data = {
        "sentence": sentence,
    }
    r = requests.post(base_url + "nlp/v1/ne",
                      headers=headers,
                      data=json.dumps(data))
    return r.json()def sentiment(sentence, access_token):
    base_url = BASE_URL
    headers = {
        "Content-Type": "application/json",
        "charset": "UTF-8",
        "Authorization": "Bearer {}".format(access_token)
    }
    data = {
        "sentence": sentence,
    }
    r = requests.post(base_url + "nlp/v1/sentiment",
                      headers=headers,
                      data=json.dumps(data))
    return r.json()


if __name__ == "__main__":
    sentence = "きのこの山美味しい。神の食べ物か。"
    args = sys.argv
    if len(args) >= 2:
        sentence = str(args[1])

    access_token = auth(CLIENT_ID, CLIENT_SECRET)

    kinoko_flg=False
    kinoko_begin_pos=0
    takenoko_flg=False
    takenoko_begin_pos=0
    suginoko_flg=False

    ne_result  = ne(sentence,access_token)
    sentiment_result  = sentiment(sentence,access_token)

    for chunks in ne_result['result']:
      if chunks['form'] == "きのこの山":
        kinoko_flg = True
        kinoko_begin_pos = chunks['begin_pos']
      elif chunks['form'] == "たけのこの里":
        takenoko_flg = True    
        takenoko_begin_pos = chunks['begin_pos'] 
      elif chunks['form'] == "すぎのこ村":
        suginoko_flg = True    
  
    if kinoko_flg == True and sentiment_result['result']['sentiment'] == "Negative" and sentiment_result['result']['score'] > 0.5:
      print(sentence[:kinoko_begin_pos] + 'たけのこの里' + sentence[kinoko_begin_pos+5:])
    elif takenoko_flg ==True and sentiment_result['result']['sentiment'] == "Positive" and sentiment_result['result']['score'] > 0.5:
      print(sentence[:takenoko_begin_pos] + 'きのこの山' + sentence[takenoko_begin_pos+6:])
    elif suginoko_flg == True:
      print("誰だお前")
    else:
      print(sentence)
