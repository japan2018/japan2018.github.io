Send to client

Next is the part that sends the received comments to the client (presenter’s PC) via socket communication. The flow is as follows.

  1. Wait for Socket connection on 10023 port
  2. Check user information if connected
  3. If the received information matches the user information on the DB, continue the connection
  4. After that, send the received comments to the client as appropriate. Checks whether the id and password of the sent data match the hash of the id and password registered in the DB in advance. ~~** Nest deep? **~~

    print("SocketStart")
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:

Comment drawing

Draw the received comment on Tkinter canvas.

class CommentManager: # Manage drawing of comments
    def __init__(self, canvas):
        self.canvas_text_list = []
        self.canvas = canvas
        root.after(1, self.update)

    def add_text(self, comment): # add comment
    #x coordinate is generated at the screen edge and y coordinate is generated at random
        text = self.canvas.create_text(
            w, random.uniform(2.0, 18.0) * 100, text=comment, font=comment_font)
        self.canvas_text_list.append(text)

    def update(self): # move left
        new_list = []
        # Move comments in the list to the left. (It would be better to do it all at once using tags)
        for canvas_text in self.canvas_text_list:
            self.canvas.move(canvas_text, -15, 0)
            x, y = self.canvas.coords(canvas_text)
            if x >-10:
                new_list.append(canvas_text)
            else: # After moving to the left end
                self.canvas.delete(canvas_text)
        self.canvas_text_list = new_list
        root.after(20, self.update)
