Data science is arguably one of the hottest markets right now. Almost all companies are looking for or considering a data science role.

So it’s a great time to become a data scientist. Or if you’re already a data scientist and want to promote to a senior position, it’s a great time to hone your skills.

This article covers some of the most popular books on data science.

Introductory level

If you are just starting out on a data science adventure, give it a try

Data Science from Scratch. Yes, it’s a complete introduction to data science for beginners. You can get started without knowing Python. Translation: Japanese translation is data science starting from scratch-Basics and practices learned with Python

If you are a complete beginner, but want to know more about machine learning, Introduction to Machine Learning with Python is the recommended book. This book also does not assume you know Python. Translation: Japanese translation is machine learning with Python

#Intermediate level If you’ve already read one or two data science books, do one or two projects yourself, and are a little accustomed to working with data, here are some books that will help you grow further.

Python for Data Analysis are NumPds. It’s perfect to become familiar with standard Python libraries like this. It is a book that starts from a place that reminds me of the behavior of Python and completes in one book.

Translation: Japanese translation is Introduction to data analysis with Python

Python Data Science Handbook, 5ff5431b. An excellent introduction to all the standard Python libraries, including, Matplotlib, and Scikit-learn.

Translation: Python Data Science Handbook

Python Machine Learning is for intermediate level. It’s between home level. Suitable for both professionals and intermediate and higher level people. It begins gently and touches on the latest in machine learning and deep learning. Please have a look at this!

#Expert Level When approaching the professional level, it is often more meaningful to actually read a scientific paper than to read a book. However, in order to go beyond traditional statistics, learning deep learning, this book started slowly in the solution, providing very practical, out-of-the-box code to help you use deep learning. It offers a number of tips that are generally useful. It is also time to implement. Here are three great references that are currently standard:

Deep Learning with Python Written by the author of Keras, one of the popular machine learning libraries. This is a must read for deep learning.

Translation: Deep Learning with Python and Keras

Deep Learning is a deep learning algorithm. A great reference book. It doesn’t contain much code, but it has deep insights on how to tackle machine learning problems: Written by a pioneer of deep learning.

Translation: Japanese translation is deep learning

If you’re interested in math, Machine Learning: a Probabilistic Perspective is very popular. This is a masterpiece of the mathematics behind all machine learning methods. You probably can’t read it all at once, but it can be a great reference for your machine learning research.
