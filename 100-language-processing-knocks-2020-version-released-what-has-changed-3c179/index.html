100 language processing knocks 2020 version released! What has changed

Apr 8, 2020 Python NLP language processing 100 knocks 

#Introduction

100 knocks of language processing that have been popular for a long time as a collection of problems that you can enjoy learning the basics of natural language processing, its 2020 version was released on 4/6 ! This is the first revision in five years. 2015 version but those who are interested, those who feel sorry that the 15th edition Qiita article is no longer useful, 15th edition For the person who was doing halfway, but the 20-year edition is coming out and it seems to be heartbreaking, I will summarize what has changed. Of course, it is unofficial, so you may miss some changes.

Summary of revision

As of April 7, according to the official update history, the following three points seem to have changed significantly.

  • Added problem about deep neural network -Chapters 8, 9 and 10 are all newly created issues
  • Release of English version (up to number 39) -It is scheduled to be released sequentially after the 40th number (Author Twitter)
  • Old Chapter 6 (Handling English text) moved to English version -The corresponding English version has not been published. It seems that it is being created (Github).

As I will explain later, as far as I can compare the Japanese version**, the 2015 version has no changes in Chapters 1-5, Chapter 8 has changed tasks, Chapters 9 and 10 have been simplified, and the rest have been deleted* It feels like *. Other than that, the URL of the file to be downloaded has been completely changed (Is it because I am concerned about the character code and the link destination?).

What changed specifically

Old Chapter 1: The Preparatory Movement

  • no change
  • There seems to be no difference in the English version except that “Patatoka See” became “schooled”

Old Chapter 2: Unix Command Basics

  • Title changed to “Unix command”
  • tsv file changed. The number of lines has also increased.
  • No change in question text

Old Chapter 3: Regular Expressions

  • No change in question text
  • Japanese Wikipedia data updated
  • English version uses English Wikipedia data

Old Chapter 4: Morphological Analysis

  • Old 33rd “Sahen noun” was deleted, and new 37th “Top 10 words with high frequency of co-occurrence with “cat”” was added
  • Text file URL updated
  • English version is “POS tagging” and comes with a script to download Alice’s Adventures in Wonderland -Japanese version specifies MeCab, but English version does not specify POS tagger

Old Chapter 5: Dependency Analysis

  • No change in question text

Old Chapter 6: Processing English Text

  • Moved to English version as described in the overview

Old Chapter 7: Database

  • Delete -I think that it is a reasonable change because there is no NLP feeling only in the old Chapter 7

Old Chapter 8: Machine Learning

  • Move to New Chapter 6 -However, there are many small changes -Maybe you were aware of the introduction to the method using deep learning in the second half
  • Task changed from polarity analysis (binary classification) to categorical classification (multiclass classification) -When I try to open the download location of the new version, I get a warning that the certificate may have expired
  • The old 78th was performing split intersection verification, but the new version does holdout verification instead -I think it’s a reasonable change as the size of the dataset has increased -In line with this, in the new version, the data is divided into learning data, verification data, and evaluation data at number 50 -The verification data seems to be used only on No. 58. Is that all right?
  • Old 71st “Stop Word” is deleted
  • Old 72-75, 77 almost moved to new version -Features → There are small changes such as feature amount -Following the stop word, stemming was also omitted -Is it due to the flow of time…
  • Old number 76 “Labeling” removed, new number 55 “Create confusion matrix” added -Originally, the significance of existence was a delicate issue
  • The old 79th “precision rate-recall rate graph drawing” was deleted, instead the new 58th correct rate-regularization parameter graph is drawn
  • New 59th “Hyperparameter search” added

Old Chapters 9 and 10: Vector Space Method

  • Moved to New Chapter 7 “Word Vector” -In the old version, you learned the word vector by yourself, but in the new version you will use the already learned -Everything else is the same as the previous 10 chapters
  • Evaluation data was broken, but updated

New Chapter 8: Neural Networks

  • Create a single-layer (feedforward) neural network using PyTorch, TensorFlow, etc.
  • Matrix construction → prediction → loss and gradient calculation → mini-batch → GPU → multi-layer, etc.

New Chapter 9: RNN, CNN

  • Implement RNN/CNN (a simple model) that was once often used in natural language processing research
  • Number 89 is supposed to try transfer learning using a pre-trained language model such as BERT -The approach currently used in various tasks of natural language processing -It seems that the library will just work as the sample, but I think that is also an important experience

New Chapter 10: Machine Translation

  • Use existing tools to perform neural machine translation -Machine translation is a core field of natural language processing, and I think that using tools alone will be an important experience.
  • In 99, create a demo in which the result of machine translation is displayed on the browser -Old 69th “Web application creation” was a difficult problem for those who do not know anything, but I think that it became a reasonable form

Personal impressions

It reflects the latest trend of natural language processing research, and I think it has become more suitable as a collection of problems used for new recruit training for students in the laboratory!
