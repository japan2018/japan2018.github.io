ABC056-C

Thoughts The sum of non-negative integers up to i is $\frac{1}{2}n(n+1)$}}. This sum becomes $X$ when $\frac{1}{2}n(n+1)=X$, and when solving for $n$, $n=\frac{-1\pm{ It will be \sqrt{8X+1}}}{2}$. If you add the ceil symbol to the solution of this equation, it becomes $\lceil \frac{-1\pm{\sqrt{8X+1}}}{2} \rceil$. This is the answer.

import math
x = int(input())

f = (-1+math.sqrt(8*x+1)) / 2
print(math.ceil(f))

ABC066-C

Thoughts If you reverse every time you operate, you will die, so manage whether it is reversed with bool. We use deque because we only access the left and right sides of the data. All you have to do is implement the operation

from collections import deque
n = int(input())
a = list(input().split())

b = deque([])
r = False
for i in range(n):
    if r:
        b.appendleft(a[i])
    else:
        b.append(a[i])
    r = not r

if r:
    b = reversed(b)
ans =''.join(b)
print(ans)

Thoughts If you count the element of $A_i$, it will probably TLE, so we will count it by another method. It is implemented by a method that sorts and counts values. If there are four or more longest sides, the largest square that can be created will be the largest. Otherwise it will be a rectangle with two first sides and two second sides.

n = int(input())
a = list(map(int,input().split()))

a.sort(reverse=True)
c = 0
v = []
for i in range(n-1):
    if a[i] != a[i+1]:
        c += 1
        if c >= 2: # append only when there are two or more edges
            v.append([a[i],c])
        c = 0
    else:
        c += 1
if a.count(a[-1]) >=2:
    v.append([a[-1],a.count(a[-1])])
if len(v) == 0: no rectangle can be created if #v is empty

elif v[0][1] >= 4: become a # square
