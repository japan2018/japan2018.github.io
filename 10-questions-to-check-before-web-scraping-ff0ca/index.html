10 questions to check before web scraping

Dec 4, 2019 Python scraping api data analysis crawlers 

With [Web scraping](https://www.octoparse.jp/), you can get the information you need in seconds and the great value behind it. But before we do that, there are 10 questions to ask.

#1.Is Web scraping illegal?

[robot.txt](https://en.wikipedia.org/wiki/Robots_exclusion_standard If you are allowed to crawl according to the terms of the file, we recommend that you read the Terms of Service (ToS) of the target website in advance to assess the legal feasibility of your data project. Some sites have made clear statements that scraping without permission is prohibited. In that case, you must get permission.

#2. Decide which website you want to get data from

What is the purpose of collecting the data? Lead generation? Price monitoring? Or create a sales list? SEO analysis? Where is the high quality information? How to find the target data? Making decisions about information when choosing a data source is very important because it can have a significant impact on the results. You can get tips from introductory articles scraping various information from popular websites in the Help Center.

#3. Confirm that the acquired website has API provision

The website of the acquisition source is [API]( https://ja.wikipedia.org/wiki/%E3%82%A2%E3%83%97%E3%83%AA%E3%82%B1%E3%83%BC%E3%82%B7%E3 %83%A7%E3%83%B3%E3%83%97%E3%83%AD%E3%82%B0%E3%83%A9%E3%83%9F%E3%83%B3%E3%82 %B0%E3%82%A4%E3%83%B3%E3%82%BF%E3%83%95%E3%82%A7%E3%83%BC%E3%82%B9) If so, you can get the data directly using the provided API platform. You don’t have to take the time to scrape. See the following example for how to connect to the API platform.

#4. Clear time and financial budget

Scraping small amounts of data is quick and easy with free scraping tools or free Python scripts. However, when scraping different website structures or large amounts of data from multiple pages, business automation is required. Scraping can be done by spending time learning programming or outsourcing. In fact, many dedicated data service providers offer data collection services. Octoparse is one of them. Even if you turn on your computer, you can extract a large amount of data in the cloud just by eating.

#5. How to deal with websites that require you to log in and when filter links are set

The URL of the web page before and after setting the filter (or before entering/selecting parameters) may be different, so do not jump deep into it. #6. What to do if your website’s bot detection system is triggered and your IP address is very likely to be banned

If a web crawler frequently visits within a very short period of time (which is likely not human), the website will track and ban the local IP. The solution can make the scraping process as slow as possible unless you trigger a bot detection. However, if your goal is to get the latest data or get it faster, use the IP rotation feature.

#7.CAPTCHA support method

In Octoparse, just as you would normally do when browsing a website, CAPTCHA can be solved manually. However, it is better not to touch it from the beginning. Scrapin your website like you’re human, so that you don’t scrape it too much.

#8. Extracted data export format

You can export the data in the following formats: Excel, JASON, CSV, HTML, MySql, or [API]( Use the https://helpcenter.octoparse.jp/hc/ja/articles/360017791934-API?source=search&auth_token=eyJhbGciOiJIUzI1NiJ9.eyJhY2NvdW50X2lkIjo5MDc3MjYzLCJ1c2VyX2lkIjozOTUyNDkzNjQyNzQsInRpY2tldF9pZCI6OTQxLCJjaGFubmVsX2lkIjo2MywidHlwZSI6IlNFQVJDSCIsImV4cCI6MTU3ODAxNTI4NH0.WOZ-IR83jS4KbRxYvM21mEEFBYI338aV022wJyH5yhc) to export to its own system.

#9. What to do if your website changes and you lose data

If you need to keep up to date with the latest data, the crawler written in a programming language is no longer useful because of a change in the structure of your website. Rewriting a script is not a trivial task and is very tedious and time consuming. Unlike the cumbersome task of rewriting the code, you can keep your crawlers up to date simply by clicking the web page again in Octoparse’s built-in browser.

#10.Analysis of the collected data

It’s not the collection of data that has a big impact on the business, but the analysis of the data. How you can make decisions based on those data is very important.
