Show grass on Google Nest Hub

Dec 26, 2019 Python GitHub GooglePhotos smart speakers NestHub 

I bought the Pixel 4 and received the Google Nest Hub. It seems to be usable as a digital photo frame. Since I'm sorry, I will display the grass image on GitHub.

Google Nest Hub settings

Immediately after purchase, Nest Hub will display an art gallery prepared by Google. Change this to a setting that allows you to view your favorite image.

  1. Launch the Google Home app on your iOS or Android device
  • If you bought Nest Hub, you should have installed it
  1. Go to Nest Hub> Photo Frame
  • Change the display image setting to Google Photos
  1. Change “Personal Photo Organize” setting to “Real-time shared albums only”
  • Without this setting, GitHub grass images, etc. may be judged as unsuitable for the photo frame and may not be displayed.

Select the Google Photos album to be displayed in the photo frame. The album to save the grass image is automatically created by the script introduced later.

Get Google Photo API access token

Now we will run a python script that will generate a grass image and upload it to the Google Photo API. To do so, you’ll need to enable the Google Photos API, which is disabled by default, and get an OAuth 2.0 token.

This time, I got a token by referring to this Qiita article.

  • https://qiita.com/shin1ogawa/items/49a076f62e5f17f18fe5

You will use the Client ID, Client Secret and Refresh Token later. These are things that should never be made public, so be very careful about their management.

Script to create and upload grass images

I have prepared a python script for generating the GitHub grass image and uploading it to Google Photos in the following repository.

  • https://github.com/fjt7tdmi/grass-photo

I have referred to https://grass-graph.moshimo.works/ for generating the grass image.

Running the script will automatically create a Google Photos album for saving grass images. The default name is “GitHub grass”. Please configure Google Nest Hub to display this album.

Automation

The repository contains settings for GitHub Actions. If you set the following secret, it will automatically generate and upload grass images every day.

  • GITHUB_CLIENT_ID
  • GOOGLE_API_CLIENT_ID
  • GOOGLE_API_CLIENT_SECRET
  • GOOGLE_API_REFRESH_TOKEN

#For those who want to display grass images on Nest Hub The scripts I made are distributed under the Apache 2.0 license. You can use it by including the automation function by forking the GitHub repository.

  • https://github.com/fjt7tdmi/grass-photo

However, note the following points.

  • Use private repositories on GitHub to avoid leaking OAuth 2.0 tokens
    • I’m using GitHub’s secret, but this is not safe enough
    • If the token is leaked, data that is not related to the grass image can be viewed and operated by a third party.
  • Please be aware of the remaining capacity of Google Drive, as my script does not delete grass images.
    • API to delete images uploaded to Google Photos doesn’t seem to be provided at present (26/12/2019)
