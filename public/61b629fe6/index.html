Machine learning sequence

First, the skills required to learn machine learning are as follows. In practice, it requires a few more detailed steps, but at first it is enough to study the following.

    1. Visualization of data: Determine the preprocessing policy by grasping the overall feeling of the data
    1. Data pre-processing: Clean the data so that the prediction accuracy is high.
    1. Selection of algorithm: Decide appropriate algorithm for data
    1. Model learning: Let the computer learn the law of data
    1. Model validation: Confirm the prediction accuracy of the completed model

1) Data visualization

  • Check basic statistics
    • Average value
    • standard deviation
  • Aggregate data -Cross tab -Binning -Pivot Table -Group
  • Graph(Detailedversionishere) -Bar chart
    • Scatter plot -Histogram -Box plot
  • Heat map of correlation matrix

2) Data pre-processing

  • Handling of missing values
    • Removal -Complement
  • Align the scale of data * Skip the math part, so please skip it.
    • Standardization
    • Normalization
  • Categorical variable encoding -one-hot encoding -Label Encoding -For those who want to know the difference in encoding, click here
  • Table Join (DataFrame Join) -merge -concat

3) Selection of algorithm

Link is here

  • Regression -Linear regression (single regression, multiple regression) -Regression tree -Random forest regression
  • Classification -Logistic regression -Decision tree -Random forest -Support Vector Machine (SVM)
  • Can be used for both -Neural network (deep learning) -XGBoost -LightGBM

4) Learning the model

  • Validation -Holdout method -Cross validation
  • Parameter tuning -Grid search -Random search -(Applied) Bayesian Optimization

5) Model validation

  • Evaluation Index (Regression) -RMSE (Root Mean Squared Error) -RMSLE (Root Mean Squared Logarithmic Error) -MAE (Mean Absolute Error) -Coefficient of determination
  • Evaluation index (2 classes) -Confusion matrix (TP, TN, FP, FN) -Accuracy -Precision -Recall -F value -LogLoss (cross entropy error) -AUC (area of ROC curve)
  • Evaluation index (multi-class classification) -multi-class accuracy -multi-class logloss -mean-F1, macro-F1, micro-F1
