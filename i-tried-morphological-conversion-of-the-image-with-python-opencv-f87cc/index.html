## Introduction

With image processing, it is not always possible to prepare beautiful images. Images often contain noise. If there is noise, it may be removed using a method called “smoothing” or “morphological transformation”.

This time, we will use Python to perform morphological transformation of the image using OpenCV.

*For details on “smoothing”, refer to the following. I tried “smoothing” the image with Python + OpenCV

What is morphological conversion

Morphological transformation is mainly the process of “shrink” and “expansion” for binary images. Morphology is “morphology”, that is, processing related to shape.

A “binary image” is a binarized image. Binarization is the process of converting an image into two gradations, black and white. *For details on “binarization”, refer to the following. I tried “binarizing” an image with Python + OpenCV

Morphological transformation

Now, let’s introduce how to perform morphological transformation on images.

Import numpy in advance for use below.

import numpy as np

erosion

Erosion contracts white areas. As a result, the white areas become smaller and thinner and the black areas increase. This is an effective process for removing white noise in an image or separating multiple connected objects.

Now, let’s display the contracted image side by side with the original image.

kernel = np.ones((3, 3), np.uint8)

img_ero = cv2.erode(img_bw, kernel, iterations=1)
imgs = cv2.hconcat([img_bw, img_ero])
cv2_imshow(imgs)

You can see that the white part is smaller and thinner than the original image.

There are three arguments to cv2.erode. The first argument is the target image. The second argument is what is called the kernel size. When you decide one point of the image, it shows how much of the surrounding area is included. Think of it as the “box size.” The third argument is what is called an iteration. Iteration means “iteration”, and here you specify how many times the process is performed. In the above, the contraction process is performed only once.

Now, let’s display the image that has been shrunk by changing the kernel size value.

kernel_1 = np.ones((1, 1), np.uint8)
kernel_2 = np.ones((3, 3), np.uint8)
kernel_3 = np.ones((5, 5), np.uint8)
img_1 = cv2.erode(img_bw, kernel_1, iterations=1)
img_2 = cv2.erode(img_bw, kernel_2, iterations=1)
img_3 = cv2.erode(img_bw, kernel_3, iterations=1)

imgs = cv2.hconcat([img_1, img_2, img_3])
cv2_imshow(imgs)

You can see that the larger the kernel size is, the more it shrinks, and the smaller the white part becomes.

Next, let’s display an image that has undergone shrinkage processing with various iteration values.

kernel = np.ones((3, 3), np.uint8)

img_1 = cv2.erode(img_bw, kernel, iterations=1)
img_2 = cv2.erode(img_bw, kernel, iterations=3)
img_3 = cv2.erode(img_bw, kernel, iterations=5)
imgs = cv2.hconcat([img_1, img_2, img_3])
cv2_imshow(imgs)

Again, since the shrinking process is repeated for the number of iterations, it shrinks as the value increases.

dilation

Dilation expands the white area. This is the reverse process of contraction. In other words, as a result, black areas become smaller and thinner, and white areas increase.

Now, let’s display the dilated image side by side with the original image.

img_dil = cv2.dilate(img_bw, kernel, iterations = 1)imgs = cv2.hconcat([img_bw, img_dil])
cv2_imshow(imgs)

You can see that the white part is thicker and larger than the original image.

The arguments for cv2.dilate are similar to those for contraction cv2.erode.

Opening process (opening)

The opening process (opening) is the process of expanding after contraction. The white part can be made smaller by contraction, and the size can be restored by expansion. Therefore, it is an effective process for removing white noise.

Let’s display the image with noise and the image subjected to the opening process side by side.

img_opening = cv2.morphologyEx(img_p, cv2.MORPH_OPEN, kernel)
imgs = cv2.hconcat([img_p, img_opening])
cv2_imshow(imgs)

As a result of the opening process, you can see that the white noise has been removed.

Closing process (closing)

The closing process is the reverse of the opening process. The black part can be made smaller by expansion, and the size can be restored by contraction. Therefore, it is an effective process for removing black noise.

Let’s display the noise image and the closing image side by side.

img_closing = cv2.morphologyEx(img_s, cv2.MORPH_CLOSE, kernel)
imgs = cv2.hconcat([img_s, img_closing])
cv2_imshow(imgs)

As a result of the closing process, you can see that the black noise has been removed.

Morphological gradient

Morphological gradient is a process that takes the difference between an inflated image and an eroded image. This will result in the contour of the object.

Let’s display an image with a morphological gradient.

img_gradient = cv2.morphologyEx(img_bw, cv2.MORPH_GRADIENT, kernel)
cv2_imshow(img_gradient)

You can see that the outline of the object is taken.

Summary

This time, we used Python to perform morphological conversion of the image using OpenCV.

As morphological transformation, we introduced “shrinkage”, “expansion”, “opening process”, “closing process”, and “morphological gradient”. Especially, the opening process and closing process are effective for removing small noises. Consider opening processing for white noise and closing processing for black noise.

There is also a method called “smoothing” as a method of removing noise, so I think that you should try this as well.
