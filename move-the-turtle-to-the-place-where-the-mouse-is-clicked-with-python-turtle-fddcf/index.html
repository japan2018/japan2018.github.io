Move the turtle to the place where the mouse is clicked with Python turtle

Apr 15, 2020 Python Python3 turtle 

Move the turtle to the place where the mouse is clicked with #Python turtle

Execution image

Sample program

import turtle

kame = turtle.Turtle("turtle")

def moveKame(x, y):
    kame.goto(x, y)

kame.screen.onclick(moveKame)

turtle.done()

Explanation

# load the turtle module
import turtle

Generate and store a Turtle in the turtle module in a variable named # kame
kame = turtle.Turtle("turtle")

Define a function named #moveKame
#def means to generate a function
# moveKame can be any name
# x, y contains the numerical value of the coordinates clicked with the mouse
def moveKame(x, y):
    # Move the turtle
    kame.goto(x, y)

# Register a mouse click event. (The moveKame function is executed when the mouse is clicked.)
kame.screen.onclick(moveKame)

# Do not close the window immediately when running the program.
turtle.done()
