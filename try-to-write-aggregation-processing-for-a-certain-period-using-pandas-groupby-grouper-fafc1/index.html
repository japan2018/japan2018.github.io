Try to write aggregation processing for a certain period using pandas × groupby × Grouper

Dec 12, 2019 Python pandas 

I wrote this article from the motivation "I want to write a process to create a feature amount that aggregates a certain column for N days in pandas, what should I do?"

Sample code is excerpted from Google Colabratory linked below. GoogleColab-pandas × groupby × Grouper sample

In addition, the data here (kaggle Datasets-Bitcoin Historical Data) is used.

First of all, from the pre-processing of data, convert the Timestamp column to datetime type for the previous processing.

import pandas as pd

btc = pd.read_csv("/content/bitstampUSD_1-min_data_2012-01-01_to_2019-08-12.csv")
btc[btc.columns.values] = btc[btc.columns.values].ffill()

# https://stackoverflow.com/questions/19231871/convert-unix-time-to-readable-date-in-pandas-dataframe

The problem is the process of creating a feature quantity that aggregates a certain column every N days, but as the title suggests, pandas × groupby × Grouper is used. “Group by” is often used when doing aggregation processing, but this time “Grouper” is used because I want to summarize it every N days and every fixed period/section. Since it is time to summarize, if you specify the Timestamp column as the key of Grouper and specify freq='D’, the process of aggregating every day is completed.

By setting # freq='D', it can be summarized on a daily basis.
btc.groupby(pd.Grouper(key='Timestamp', freq='D')).mean()
