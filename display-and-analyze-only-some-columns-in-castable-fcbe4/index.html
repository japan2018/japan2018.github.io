Display and analyze only some columns in CASTable

Dec 17, 2019 Python SAS machine learning AI 

[SAS Viya](https://www.sas.com/ja_jp/software/viya.html)isanAIplatform.ItisavailablethroughlanguagessuchasPython,JavaandR.AtableobjectcalledCASTableisusedinSASViya(CASstandsforCloudAnalyticServices). This time, I will introduce a method to display information by extracting only some columns from CAS Table.

Get table from database

First, connect to SAS Viya.

import swat
conn = swat.CAS('server-name.mycompany.com', 5570,'username','password')

Then get the CASTable. This time I use CSV of IRIS data.

tbl = conn.loadtable('data/iris.csv', caslib='casuser').casTable

Get column

You can retrieve columns by specifying the key with tbl as a dict.

col = tbl['sepal_width']
col

The output looks like this:

CASColumn('DATA.IRIS', caslib='CASUSER(username)')['sepal_width'].sort_values(['sepal_length','sepal_width'], ascending=[False, True])

Display data

If you use the head method of a column, only the value for that column will be output.

col.head()

0 3.8
1 2.6
2 2.8
3 3.0
4 3.8
Name: sepal_width, dtype: float64

Get multiple columns

Similarly, you can get multiple columns by giving the key as an array.

widths = tbl[['sepal_width','petal_width','species']]

The contents will be as follows.

| | sepal_width | petal_width | species | |————|————-|———–|———- -| | 0 | 3.8 | 2.0 | virginica | | 1 | 2.6 | 2.3 | virginica | | 2 | 2.8 | 2.0 | virginica | | 3 | 3.0 | 2.3 | virginica | | 4 | 3.8 | 2.2 | virginica |

Summary

By acquiring some columns, you can narrow down the analysis to only the ones you need, even if you have a table with many columns. This can be used when there is too much numerical data and you do not know where to analyze it.
