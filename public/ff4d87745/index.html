## What is Python

Python is a programming language that appeared in 1991. It has the following features.

  • Free SoftwareandOpenSource
  • Cross Platform
  • [Interactive Shell](http://en.wikipedia.org/wiki/%E3%82%A4%E3%83%B3%E3%82%BF%E3%83%A9%E3%82%AF%E3(%83%86%E3%82%A3%E3%83%96%E3%82%B7%E3%82%A7%E3%83%AB)
  • Incorporates many useful ideas about programming languages in the past
  • Many libraries available for scientific calculation and machine learning

Due to these characteristics, it is mainly used in Europe and America. Google App Engine is adopted as a language that can be used with Java. YoutubeandDropbox are also written in Python. Surprisingly, the popular simulation game “Civilization 4” is also written in Python.

Perhaps anyone who is reading this page has heard that “Python is easy to learn”? There are several reasons why Python is said to be suitable for learning.

  • Simple because there are few reserved words
  • Instructions that perform the same processing can be executed with the same name, making it easy to understand
  • It is easy to understand what the code is doing by forcing indentation

“Forced indentation” is often mentioned as a major feature of Python.

◆Example: Factorial program written in C language

int factorial(int x)
{
    if (x == 0) {
        return 1;
    } else {
        return x * factorial(x-1);
    }
}

◆C language program that returns the same result, but it is difficult to understand

◆ Factorial program written in Python

def factorial(x):
    if x == 0:
        return 1
    else:
        return x * factorial(x-1)
