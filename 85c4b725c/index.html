It’s too interesting to get a number from 1 to 100, isn’t it too great?

Although it is reproduced in perl and ruby in the article, it seems that it is not reproduced in python that I usually use, so I tried it.

from decimal import *
import re
getcontext().prec = 298

data = Decimal(1000) / Decimal(999**2) #Calculation part
data = (str(data).split(".")[1])

print(re.findall(r'...', data)) # split + output part
