[Python] If you use GitPod, you can use IDE and jupyter on iPad, so it’s the best!

Nov 10, 2019 Python GitHub iPad JupyterLab Gitpod 

# TL;DR

**Using GitPod, you can use IDE and jupyter on iPad, so it’s the best! **

Features of #GitPod GitPod has the following features.

  • VS code-like IDE available
  • Available in chrome and firefox
  • Can be used by linking to an account with GitHub
  • Public repositories are free, private repositories are available for a fee (9$/month ~)

#This goal and procedure

on chrome on iPad Build a Python programming environment (IDE and jupyterlab). The general procedure is as follows.

  1. Create a repository on GitHub and put 3 files
  2. Access to GitPod
  3. Launch jupyterlab

Preparation on GitHub

Place the following 3 files in an appropriate repository. The first file is a docker file used on GitPod.

FROM python:3.7
RUN apt update -y && apt upgrade -y
RUN pip install pipenv

The second file is the GitPod configuration file. The reason for opening port 8888 is to use jupyterlab.


image:
  file: .gitpod.Dockerfile

ports:
- port: 8080
  onOpen: open-preview
- port: 8888
  onOpen: open-browser

The third file is Pipfile. This is not required because it is a file prepared assuming that you are building a virtual environment with pipenv. (It is recommended that you can use the local environment etc. as it is on GitPod if you use pipenv.) I think that there should be jupyterlab and pandas as the package.

[[source]]
name = "pypi"
url = "https://pypi.org/simple"
verify_ssl = true

[dev-packages]

[packages]
pandas = "*"
jupyterlab = "*"
matplotlib = "*"

[requires]
python_version = "3.7"

By now, I think the following 3 files (and README.md) are located directly under the repository.

(Repository root)
 |-.gitpod.Dockerfile
 |-.gitpod.yml
 |-Pipfile
 |-README.md

Access GitPod

To access GitPod, just add gitpod.io/# to the URL of the GitHub repository. For example, if your GitHub repository URL is

https://github.com/[user_name]/[repository_name]

The URL to access GitPod is as follows.

https://gitpod.io/#github.com/[user_name]/[repository_name]

It’s hard to type every time, so it’s convenient to put a link in README.md on GitHub.

The GitPod IDE takes about 10 minutes to start at the first time, but it will start in 1 to 2 minutes after the second time.

Launch jupyterlab.

Since there is a Pipfile in the repository, you can launch jupyterlab by hitting the following command.

$ pipenv install
$ pipenv shell
([repository_name])$ jupyter lab

Launch the pipenv shell,

You can also start and log in jupyterlab on your iPad.
