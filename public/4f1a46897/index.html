Practical robust control

Book Contents This book is a book where you can practically learn robust control as the title says. First, I will briefly explain the positioning of robust control. In the field of control, you will first learn classical control. In classical control, the controller is decided by trial and error based on the frequency response of the system such as Bode plot or Nyquist plot. Since it is difficult to deal with classical control when the order of the controlled object becomes large or it becomes a multi-input/output system, modern control theory solves these problems. However, modern control theory requires an accurate model of the controlled object. A model error exists between the actual controlled object and the model, and the controller designed based on the model adversely affects the control result. The controlled object may become unstable. In robust control, the controlled object is expressed as a nominal model (rough model) + model error, and a controller is designed for those systems. Therefore, it can be stabilized even if there is a model error. This book describes the concept of robust control, the method of expressing the controlled object with model error, and the robust stabilization problem. After that, learn the actual design procedure through programming using MATLAB with actual application examples. It also describes the limits of robust control and the implementation of a controller (digital controller) in an actual machine.

Watawata voltage impressions

  • I think most of the robust control can be learned with this book.
  • If you just want to try it out, programming the controller in Chapter 4 with MATLAB is sufficient. You can practice that robust control is powerful in Chapter 4 alone.
  • The theory and practice parts of robust control (programming with MATLAB) are arranged in a well-balanced manner, which is exactly “practical robust control”.
  • It is good to have an example of hard disk drive control. I think watawata voltage is an example in which the robust control scenario is well understood.
  • Because the program of MATLAB is loaded, it is easy for beginners to learn. In addition, the sample code can be downloaded from the home page of this manual.
