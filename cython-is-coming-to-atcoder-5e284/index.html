It’s fast when typed!

Looking at the Cython documentation, it seems like I need to write a .pyx file and compile it, and then import and use it in .py, so I need two source codes, and I didn’t know at first how to use it. Cython has Pure Python Mode, which AtCoder seems to use.

As a result, AC. 6 times faster in 107ms!!

from cython import longlong

def main():
    N: longlong
    Y: longlong
    N, Y = map(int, input().split())

    i: longlong
    j: longlong
    for i in range(N + 1):
        for j in range(N + 1-i):
            k: longlong = N-i-j
            if 10000 * i + 5000 * j + 1000 * k == Y:
                print('%d %d %d' %(i, j, k))
                return
    print('-1 -1 -1')

if __name__ =='__main__':
    main()
