Male death, female survival model

This time, let’s simply allocate men to death and women to survive. This should be predictive, as Titanic had higher mortality rates for men and higher survival rates for women.

use #pandas
import pandas as pd

# Read CSV
test = pd.read_csv('test.csv')

# Added Survived column
test["Survived"] = 0

# If a woman, replace it with 1 (survival)
test.loc[test["Sex"] =='female', "Survived"] = 1

#Only PassengerId and Survived for submission.
test = test.loc[:,['PassengerId','Survived']]

Output to #CSV (no index required)
test.to_csv('titanic1.csv',index=False)
