[Python3] Understand the basics of file operations

May 24, 2020 Python Beginner File Manipulation 

## Introduction

I will summarize the basic knowledge to operate files with Python.

Target: For those who have started learning Python Operating environment: Python3.8.3 (OS is MacOS)

Read and write files

Open file

open method
  • Specify the file to open and how to open (read/write).
  • File object is returned.
  • After manipulating the file, it is necessary to execute the close method (to release it from memory).
f = open("filename", "mode")
  • “mode” type

|mode | Description | |:——–|:————————————– —————–| |r | Read mode. ｜ |w | Overwrite mode. The existing contents are discarded. | |a | Append mode. Append to the end of the file. | |x | Write mode only for creating new files. If the file exists, an error is returned and writing is not done. | |r+ | Read/write mode. ｜

If omitted, it is processed as “r”.

  • “Mode” for handling file types

It can be specified in combination with the above-mentioned “mode”.

|mode | Description | |:——–|:————————————– —————–| |t | Text mode. The default mode for the open method. Convert the line feed code from the platform-specific symbol to “\n”. | |b | Binary mode. Use of binary mode is recommended except for text files. |

with keyword

Since the file object is always closed when you use the with keyword, you can write it more succinctly than implementing try ~ finally.

with open("filename") as f:
  f.read()

File object methods

f.read(size)
  • Only the value specified in size is returned as a character string (in text mode) or a bytes object (in binary mode).
  • If size is omitted or a negative number is specified, the entire contents of the file will be returned.
  • If the end of the file has been reached, an empty string ('') is returned.
>>>f.read(10)
"Hello Pyth"
f.readline()
  • Read only one line from the file and return it.
  • Includes a newline character (\n) at the end (excluding the last line that does not include a newline character).
>>> f.readline()
'Hello Python\n'
Reading multiple lines
  • You can perform file reading efficiently for loop processing.
  • Use list(f) or f.readlines() if you want to treat all lines in the file in list format.
>>> for line in f:
... print(line, end='')
...
Hello Python
Hello Java
Hello Ruby
f.write(string)
  • Write the contents of string to a file and return the number of written characters.
  • It is added every time you call the wite method until the file is closed.

>>> with open("sample.txt", "w") as f:
... f.write("Japan\n")
... f.write("USA\n")
... f.write("China\n")
...
6
Four
6
>>> with open("sample.txt", "r") as f:
... f.read()
...
'Japan\nUSA\nChina\n'
f.tell()

  • Returns an integer indicating the current open position of the file.

  • Returns “0” if the current position is the beginning

>>> with open("sample.txt", "r") as f:
... f.tell()
...
0
  • If you read 5 bytes, the current position also moves
>>> with open("sample.txt", "r") as f:
... f.read(5)
... f.tell()
...
'Japan'
Five
f.seek(offset, whence)
  • The file position is calculated by adding the offset value (offset) to the reference point (whence).
  • Whence specifies 0: start, 1: current position, 2: end. If omitted, 0: acts as the beginning.
>>> with open("sample.txt", "rb+") as f:
... f.write(b"0123456789abcdef")
... f.seek(3) # 3 bytes from the beginning
... f.read(1)
... f.seek(-1,2) # 1 byte from the end
... f.read(1)
...
16
3
b'3'
15
b'f'
