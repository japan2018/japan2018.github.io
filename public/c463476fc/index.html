What is nbdev

Perhaps succinctly, it’s a Python programming environment that completes your work in one Jupyter Notebook by including the role of an IDE. Created by fast.aiandcurrentlyindevelopmentfastaiv2 also uses nbdev.

At present, trial and error such as EDA and model construction are often done using Jupyter Notebook, and it exists as a role of “exploring”. However, it is common to use an IDE in the subsequent phases.

nbdev is trying to complete it on jupyter by providing the following features. ・Py file creation ・Preparation of README ・Visual diff ・Confirmation and correction of merge conflict

(Official borrowing)

Tutorial/How to use

The following is done on Mac, but the procedure is the same on Win.

Install ~ Until Jupyter opens

1- Install with the following command from the terminal. pip install nbdev

2- The official template is created, so create a Repository based on it.

3- Clone the created Repository. git clone [following URL]

4- Edit settings.ini to add the information of Repository to be linked. Uncomment the following items in settings.ini and edit them as in the entry example.

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/248638/f9f1e8d9-e3fd-e0f1-c5a1-63ceeb9cb735.png) ![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/248638/3a048935-9fc2-ee88-6726-4cf18d810294.png)

lib_name, user and description correspond to the created repository. It’s unclear what the keywords are, but please enter an appropriate string.

  • If you do not enter the copyright, you will be moss by the following procedure.

5- Type the following command from the terminal to create a directory for lib_name. nbdev_build_lib

(Pipfile and Pipfile.lock are displayed in the image above, but they are irrelevant so please ignore them)

6- Open Jupyter Notebook with jupyter notebook.

Create py file

Open 00_core.ipynb and create a module by the following procedure.

  1. Add #export to the beginning of the corresponding cell for the one converted to python module.
  2. Type the following two lines into the cell and execute. (Or run nbdev_build_lib on the terminal.) from nbdev.export import *''notebook2script()``
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/248638/a7a7075b-8631-3243-f8d3-2fb3c304738f.png)

As a result, the one specified in #export is described in core.py under the lib_name directory. Since core.py is updated every time you perform step 2, you can delete it by deleting existing cells in addition to adding it.
