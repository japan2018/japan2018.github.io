Main subject

When trying to do difficult things with TouchDesigner, python control becomes essential.

However, there are probably a few TouchDesigner users (including myself) who are familiar with python. .. .. maybe. ..

Therefore, I will introduce the tips for batch operation of operators by for statement that you will often use.

Operate many operators

For example, “substitute 1 for value0 of constant1,2…9” is as follows.

for i in range(1,10):
    op('constant' + str(i) ).par.value0 = 1

This is easy, isn’t it?

Explanation: Since op() is a string, i must be a string with str().

Assign to many parameters

For example, “assign 1 to value1,2,…9 of constant1” is as follows.

for i in range(1,10):
    script ='op(\'constant1\').par.value' + str(i) +'=1'
    exec(script)
print(a)

Use the exec function. (I didn’t know this and struggled…)

Tips: To use the item number of value as an iterator, first write the entire statement you want to execute in the string type and embed the iterator. Then execute it with the exec() function. Don’t forget to escape the quotation! !

Get the values of many parameters

For example, “Display the total value of 1,2,…10 of constant1” is as follows.

a = 0
for i in range(10):
    a += eval('op(\'constant1\').par.value' + str(i))
print(a)

It’s similar to the exec function above, but this time it uses eval, a function that evaluates a value.

You can also use the exec function as follows:

a = 0
for i in range(10):
    script ='a += op(\'constant1\').par.value' + str(i)
    exec(script)

Summary

In this way, when you want to turn TouchDesigner’s built-in function into a for statement, use exec or eval and it will work.
