Summary of how to write if statement (Scala, Java, Rust, C language, C++, Go language, PHP, Perl, Python, Ruby)

Nov 22, 2019 Ruby Python Go Scala Rust 

When you touch a variety of languages, the detailed characteristics of the language and the usage of symbols are getting messed up, regardless of the important characteristics of the language. How do you write that this is this language? It often happens. Even though I used to use a language that I used to use a lot, I sometimes feel shocked that I have forgotten it even after a short distance. So, I will make a memo like this.

Scala

if (x >= 30) {
    ...
} else if (x >= 0) {
    ...
} else {
    ...
}

// expression, not statement
val max = if (a> b) a else b
  • Conditional expression must be logical
  • Like C, { } can be omitted when there is only one expression in { } (dangling else problemYes)
  • Expression, not statement
  • Strictly speaking, the { ... } part in the above example can contain one expression, and the braces are part of the syntax of the block expression, not the specification of the if expression. If you put an expression other than a block expression, it will look the same as the curly braces omitted in C.
