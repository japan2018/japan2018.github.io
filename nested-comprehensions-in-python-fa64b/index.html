Nested comprehensions in Python

Dec 3, 2019 Python comprehension 

Conclusion: If you forget which one of the nested comprehensions is the inner, remember it is the same as when you wrote it down in the for loop. I forgot how many times I forgot so I took a note

Single comprehension

First a basic example.

>>> [x * 2 for x in [1,2,3,4,5]]
[2, 4, 6, 8, 10]

You can create a list by applying an expression to each element of the list by doing “[Expression using variable for variable in list]`.

>>> {x: x * 2 for x in [1,2,3,4,5]}
{1: 2, 2: 4, 3: 6, 4: 8, 5: 10}

You can also create a dictionary instead of a list by doing {expression with variable: expression with variable: for variable in list}.

Double comprehension

I always forget this order. .. ..

>>> [(x, y) for x in [1, 2] for y in ['a','b','c']]
[(1,'a'), (1,'b'), (1,'c'), (2,'a'), (2,'b'), (2,'c')]

In this way, .. for .. in .. can be doubled. In this case, what was written later becomes the inner loop. In this case, the y written later is the inner loop, while the outer x rotates slowly while it turns finely. As a way to remember

>>> results = []
>>> for x in [1, 2]:
        for y in ['a','b','c']:
            results.append((x, y))
>>> results
[(1,'a'), (1,'b'), (1,'c'), (2,'a'), (2,'b'), (2,'c')]

It is the same as writing a for loop in the same order as.

Expanding double list using double comprehension

That means that you can use the previous variable in the back loop. For example, if you want to expand a nested list,

>>> [item for sublist in [[1,2], [3,4], [5,6]] for item in sublist]
[1, 2, 3, 4, 5, 6]

Like At first glance, it’s the same as below. Somehow it feels like the previous for is inside, but it is outside.

>>> results = []
>>> for sublist in [[1,2], [3,4], [5,6]]:
        for item in sublist:
            results.append(item)
>>> results
[1, 2, 3, 4, 5, 6]

Applying double list expressions using double comprehensions

If you include the comprehension before the comprehension for, you can apply the expression to the inner element without changing the structure of the list.

>>> [[item * 2 for item in sublist] for sublist in [[1,2], [3,4], [5,6]]]
[[2, 4], [6, 8], [10, 12]]

I think this is intuitive because you just write the comprehension notation where you can write the expression. Just in case you write it in a for loop:

>>> results = []
>>> for sublist in [[1,2], [3,4], [5,6]]:
        subresults = []
        for item in sublist:
            subresults.append(item * 2)
        results.append(subresults)
>>> results
[[2, 4], [6, 8], [10, 12]]
