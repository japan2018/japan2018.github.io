#1.What is Kaggle

It is read as kaguru. Kaggle is a US-based community site for people involved in data science and machine learning around the world. The highlight of Kaggle is the competition. The competition is a competition in which participants compete for the power of data analysis. Finally, I would like to participate in the competition and compete, but there seems to be a tutorial, so I would like to proceed from the place where the tutorial is done first.

#2. Sign up for Kaggle

https://www.kaggle.com/ Register as a member on Kaggle’s site. Perhaps because I already have a Google account, “Welcome sudominoru” was displayed and I was already registered.

I uploaded photos from “Edit Profile” and registered “City”.

#3. Participate in the competition Those who want to try kaggle first try “Titanic Tutorial”. I will participate in this competition.

Select “Competitions” at the top of the screen Our Titanic Competition is a great first challenge to get started. (The Titanic Competition is a great first challenge to get started.) Titanic is displayed at the top. Click and select Titanic.

Click “Join Competition” to participate in the Titanic competition.

Please read and accept the competition rules (Please read and agree to the competition rules) Since this is this, I confirm the rules and then agree. I was able to participate.

#4. Try to write the code kaggle provides an environment for writing code. Let’s write it now.

From Notebooks, click your Work and then Create New Notebook.

Select the language and type (“Notebook” or “Script”). I chose “Python” as the language and “Notebook” as the Type. I think “Notebook” is like “Jupyter Notebook” and “Script” is an image of writing code with “Spyder”. I usually write code in “Spyder”, but I want to see it in Kaggle, so I want to write the code while writing the explanation, so I will proceed with “Notebook”.

A screen like “Jupyter Notebook” is displayed like the image above. In the input directory on the right side, there are training data (train.csv) and verification data (test.csv) used in this competition. “Gender_submission.csv” will be explained later, but it is sample data for submission of competition. If you run the sample code as it is, the file name will be output. Now you are ready to write the code.

#5. Try to submit the learning results

I mentioned earlier that “gender_submission.csv” is sample data for submission of competitions. Looking at the contents, there is the same PassengerId as “test.csv”, and the number of cases is also the same. The flow is as follows.

  1. Learn using “train.csv”
  2. Verify the learning results using “test.csv”. Output the verification result to “gender_submission.csv”
  3. Submit “gender_submission.csv”

As a test, let’s output “gender_submission.csv” as it is.

# Any results you write to the current directory are saved as output.

Load # gender_submission.csv
# Load gender_submission.csv
df_gender_submission = pd.read_csv('/kaggle/input/titanic/gender_submission.csv')
Write # gender_submission.csv in the current directory
# Write gender_submission.csv to the current directory
df_gender_submission.to_csv('gender_submission.csv', index=False)

Add the above code and click “Commit” in the upper right to execute.

After a while, the execution result will be displayed. Click Open Version.

The content of the result is displayed. You can see that “gender_submission.csv” was output to “Output Files” at the bottom of the screen. Click “Submit to Competition” to submit.

The result will be displayed after transitioning to “Leader board”. The score is “0.76555”. The correct answer rate was “76.5%”. You can transition to your own rank by “Jump to your position on the leaderboard”.

#6. Summary I understand how to use Kaggle. Next time, I would like to study at Titanic.
