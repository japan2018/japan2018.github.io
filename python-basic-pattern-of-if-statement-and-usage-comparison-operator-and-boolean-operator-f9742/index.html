##1. Basic syntax of if statement The if statement has four patterns, which are a combination of conditional branches “else” and “elif”.

① No conditional branch ② One conditional branch. True or False ③ Two or more conditional branches. Only the specified conditions are processed. ④ Two or more conditional branches. Also set processing in case of false (False)

####▼ ① No conditional branch If the conditions are not met, do nothing.

if A:
    AAA

“A”: Conditional expression “AAA”: Processing when A is true

  • Add “:” at the end of the conditional expression.
  • The pattern executed in the if statement lowers the indent. (In the case of a general editor, if you add “:” and start a new line, indentation will be added automatically)
A = 100

if A == 100:
    print ('A is 100.')

#output
#A is 100.

A = 90

if A == 100:
    print ('A is 100.')

####▼ ② Conditional branching (else only) Two branches. True or False. 
if A:
    AAA
else:
    BBB

“A”: Conditional expression “AAA”: Processing when A is true “Else:": When A does not hold “BBB”: Processing when A does not hold

*No conditional expression is required after else. *“Else:” is always double colon and set.

A = 90

if A == 100:
    print ('A is 100.')
else:
    print('A is not 100.')

#output
#A is not 100.

####▼ ③ Two or more conditional branches (without else) Process is executed only under the specified conditions. Describe the branch condition and processing with elif. └ abbreviation of elif = else if (other than that) 
if A:
    AAA
elif B:
    BBB
elif C:
    CCC

“A”: Conditional expression “AAA”: Processing when A is true “Elif B”: Second conditional expression ”BBB”: Processing when B holds ”Elif C”: third conditional expression ”CCC”: Processing when C holds

  • Multiple “elif” can be set. (Only 2 above)
  • Add “:” at the end of “elif conditional expression:” conditional expression *Confirm the conditions from the top, and when it becomes Pure, the process ends. *No processing if the conditions of “if” and “elif” are not met
A = 70

if A == 100:
    print ('A is 100.')
    
elif A == 80:
    print ('A is 80.')
    
elif A == 70:
    print ('A is 70.')

#output
#A is 70.

A = 70
B = 100

if A == 100:
    print ('A is 100.')
    
elif B == 100:
    print('B is 100.')
    
elif A == 70:
    print ('A is 70.')

#output
#B is 100.

It ends when the conditions are matched from above. └ Both “B == 100” and “A == 70” are true, but the process ends when “B == 100” above becomes true.


####▼ ④ Two or more conditional branches (with else) Also set processing in case of false (False) 
if A:
    AAA
elif B:
    BBB
elif C:
    CCC
else:
    DDD

“A”: Conditional expression “AAA”: Processing when A is true “Elif B”: Second conditional expression ”BBB”: Processing when B holds ”Elif C”: third conditional expression ”CCC”: Processing when C holds “Else:": If none of the conditions are met “DDD”: Processing when none of the conditions are met

  • Multiple “elif” can be set. (Only 2 above)
  • Add “:” at the end of “elif conditional expression:” conditional expression
A = 50

if A == 100:
    print ('A is 100.')
    
elif A >= 80:
    print ('A is over 80.')
    
elif A >= 70:
    print ('A is over 70.')
    
else:
    print ('A is less than 70.')

#output
#A is 70 or less.

##4. Examples of comparison operators

① “==” equal

There are two equals. If it is 1 or 3 or more, an error will occur.

(One equal is assigned to a variable)

A = 100

if A == 100:
    print ('A is 100.')
else:
    print ('A is not 100.')

#output
#A is 100.

###② “!=” is not equal

When “!” is added at the beginning, it means “not”.

A = 100

if A != 100:
   print ('A is not 100.')
else:
   print ('A is 100.')

#output
#A is 100.

Supplement: “!” Exclamation mark, exclamation mark, exclamation mark

###⑦ Contains “in” element

if A in B └ A is included in B. *A is shorter. B is longer.

A ='pen'

if A in B:
   print('Yes')
else:
   print('No')

#output

#output
# No

##5. Examples of Boolean operators Used when you want to multiply multiple conditions. Each operator can be used together.

① “and” and

if A and B:└ “A” “B” conditional expression

*Any number of conditions can be combined.

a = 1
b = 2

if a == 1 and b == 2:
    print('a=1, b=2')
else:
    print('a=1, not b=2')

#output
# a=1, b=2

a = 1
b = 2
c = 3

if a == 1 and b == 2 and c != 3:
    print('a=1 and b=2 and 3 is not 3')
else:
    print('a=1 and b=2 and 3 is not 3')

#output
# a=1 and b=2 and 3 is not other than 3

### ② "or" or `if A or B:` 
a = 1
b = 2
c = 3

if a == 1 and b == 2 or c != 3:
    print('a=1 and b=2 or 3 is not 3')
else:
    print('a=1 and b=2 or 3 is not other than 3')

#output
# a=1 and b=2 or 3 is other than 3

**▼ operator can be used together** 
a = 1
b = 2
c = 3

if a == 100 and b == 200 or c != 3:
    print('a=1 and b=2 or 3 is not 3')
else:
    print('a=1 and b=2 or 3 is not other than 3')

#output
# a=1 and b=2 or 3 is not 3

###③ Not “not” if not A: └ Attach to the beginning of the conditional expression

a = 100
b = 100

if not a == b:
    print('not a=b')
else:
    print('a=b')

#output
# a=b

a = 100
b = 100

if a != b:
    print('not a=b')
else:
    print('a=b')

#output
# a=b
