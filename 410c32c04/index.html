[Python] The world changed when I opened a large Python project (Django) with Sourcetrail (Linux)

Nov 22, 2019 Python Django sourcetrail 

It seems that a tool called Sourcetrail has been released and it seems interesting, so I tried it.

There was an article you have already touched, but it is charming. Ubuntu.

I used it, and the big repository that I can think of was Django, so I will try it with a combination of Python and Django.

Install SourceTrail

You can download it from Sourcetrail(GitHub) Release Page. Download 64-bit Linux tar file

Then decompress it with tar -xzf Sourcetrail_2019_4_61_Linux_64bit.tar.gz -C path/ to an arbitrary location. The version name is appropriate.

Enter the sourcetrail directory you answered and execute install.sh with root privileges to complete the installation.

Open for now

A screen like this will appear.

Open Django as a project

When you select New Project, the following screen will appear, so select it normally.

![Screenshot from 2019-11-22 19-32-16.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/140543/6739e3f7-3823-3005-(46fc-fe0681858c4c.png)

Select Source Group Types

There is only one Python, so let’s choose this. Note that Python support seems to be in beta phase.

Environmental setting

![Screenshot from 2019-11-22 19-37-32.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/140543/da7fb2c1-4eb6-f48b-(7127-1e6d79dd5626.png)

Default Python is different depending on the person, so specify it appropriately.

In Files & Directories to index (files and directories to index), specify django/django/ or the directory whose contents you want to check.

After confirming the next screen, let’s create it.

indexing

If a screen like this appears, the next step. It seems to be good indexing.

830Files is also available, so it will take some time. (It took about 40 minutes in my environment) It seems that one of the aims is to reduce the code reading effort in large-scale development, so I would like to see if it can be used properly in a repository as large as Django.
