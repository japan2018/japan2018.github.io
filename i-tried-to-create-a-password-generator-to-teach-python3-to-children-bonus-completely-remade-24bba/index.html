# It seems to be popular to change the password of my number card, so I made a password generator. Update: 2020/05/30

Actually, because I needed a password with an arbitrary number of digits, 1000 or more, at work… Since it’s just a matter of course, I also studied Python and made a password generator with the following specifications.

Password generator specifications

  1. Password with any number of digits
  2. Mix English lowercase letters (or choose)
  3. Mix English capital letters (or choose)
  4. Mix symbols (or choose)
  5. 3 consecutive characters are repelled (123, abc, 987, XYZ, 121, etc. are bad!)
  6. Make sure to include the symbol if it is marked
  7. Allows you to exclude characters that are difficult to enter, such as symbols and 0s and Os, and those that are likely to be mistaken (the illustrated characters include characters that I dislike as passwords w).
  8. Create a specified number of passwords for the above rules
  9. Output to screen

Brief explanation

I haven’t done much. Changed to use symbol weight in random.choices(population,weights=weights,k=password_length)). (Because I thought that the symbol in the password was Osugi!) Since the weighting is done with make_weights(population), please change this if you think it is out of balance.

From the perspective of “teaching children,” this program is more difficult than Othello. Because the concept of character code comes out. It’s incomprehensible to say, “A on the screen you’re looking at is hexadecimal 0x41.”

import random ,string ,re

class password_generator_class:
    def __init__(self):
        self.punctuation = "!""#$%&'()*+,-./:;<=>?@[\\]^_`{|}~"

            if yndsw == False and (akey.lower() =='n' or akey.lower() =='y' ):
                return akey.lower()

        # Make a list and delete unnecessary symbols
        list_population = list(population)
        for c in specific_characters:
            list_population.remove(c)
        # Return to string
        new_population = "".join(list_population)
        return new_population

    # Create weights
    def make_weights(self,current_population):
        list_weights = list()
        for c in current_population:
            list_weights.append(2 if c in self.punctuation else 6)
        return list_weights

    # Password validity
    def check_validity(self,tmp_password,punctation_check):
        for x in range(len(tmp_password)-2):
            a = ord(tmp_password[x])
            b = ord(tmp_password[x+1])
            c = ord(tmp_password[x+2])
            if (-1 <= int(a-b) <= 1) and (-1 <= int(b-c) <= 1 ):
                return 1
            start_punctation = re.search( "[{}]".format(self.punctuation) ,tmp_password)
            if punctation_check == "y" and start_punctation == None:
                return -1
        return 0
