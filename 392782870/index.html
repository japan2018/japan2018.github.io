[Python] I installed the IoT platform Rimotte

Nov 16, 2019 Python WebRTC automation artificial intelligence IoT 

### What is Limotte

Rimotte is an All-In-One platform for always-on apps such as IoT. It is used by using the “Station” that is installed and used on Windows and a web browser that runs on Android, iOS, Windows, MacOS, Linux, etc. Not limited to IoT, it can be used for applications that are used without being connected to the Internet, and applications such as measurement, monitoring, automation, and AI can be easily realized. In general, IoT is often realized by combining multiple services, and it tends to take a lot of time to select the technology to use and the learning cost tends to be high. In Limotte, all the functions necessary for realizing the application such as server, database, communication, execution management, maintenance are included in the station, and the operation/setting, screen creation, program development environment, etc. are unified. It has been implemented with a user interface. This provides an environment where you can do what you want to do even if you don’t know much about your expertise.

Characteristic

  1. Can be used for free
  2. With the All-In-One platform, you can get started with a Windows computer
  3. Direct connection with peer-to-peer using WebRTC technology enables high quality data to be transferred with little delay time
  4. Not only sensors and remote controllers, but also video and audio functions are extensive and can be distributed.
  5. There is a market called “Rimotte Store” where you can download and use apps and functions. From small personal computers of around 610,000 yen to tablets and high-performance desktops, you can select and use it according to your needs.
  6. The application screen can be created by arranging the display parts prepared in advance as if they were drawing software and setting the options.
  7. Suitable for small to medium sized applications, the program is written in Python

Installing Limotte Station

  1. On the homepage of Rimotte Technologies, Inc., click “Download Station Now” to get “Remotte Installer.exe”.
  2. Double-click to launch the installer.
  3. When the welcome screen is displayed, click “Next”.
  4. The license agreement is displayed. Read the text to the bottom, and if there are no problems, check the checkbox “I accept the terms of the license agreement” and click “Next”.
  5. Click “Next” with the default installation destination.
  6. Click “Install” on the installation start screen. After this, if the “Visual C++ Redistributable” Ununun dialog is displayed, install it according to the displayed contents.
  7. When it is installed successfully, click “Close” to complete!
  8. If an orange icon like the one surrounded by a red frame below is registered in the task tray, it’s okay. This is the end of this post.
  9. If the above icon is not registered, open the Windows menu at the bottom left of the screen and click “Start/Stop Services” in the “Remotte Station” folder. Next, when the “Remotte Service Manager” window appears, press the “Start” button.

Once the service has started successfully, the Stop button will now be active, so close and close this window and make sure you see the orange icon in the task tray.
