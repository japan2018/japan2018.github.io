Classification of irises by neural network

Although the book explained about neural networks using handwritten characters of the keras dataset, I dared to implement classification using sklearn’s iris dataset.

from keras import models, layers
from keras.utils.np_utils import to_categorical
from sklearn import datasets
from sklearn.model_selection import train_test_split

# Loading toy data
iris = datasets.load_iris()

#Check directory contents, define'data' and'target' respectively
dir(iris)
x = iris.data
y = iris.target

#Check data shape
print([x.shape, y.shape])
print(iris.fearture_names)
print(x[0])

x.shape = (150, 4), y.shape = (150,)

Check x element in iris.feature_names.

Confirm that 4 columns of x correspond to [‘sepal length (cm)’,‘sepal width (cm)’,‘petal length (cm)’, and’petal width (cm)'].

y is 0 to 2 and supports 3 types of irises.

def normalization(x):
    mean = x.mean(axis = 0)
    std = x.std(axis = 0)
    x -= mean
    x /= std
    return x

x = normalization(x)
y = to_categorical(y)

By subtracting the average from x and dividing by the standard deviation, each element is normalized so that “center is 0 and standard deviation is 1”. y is also to_categorical and vectorization of 0 or 1 is performed.

x_train, x_test, y_train, y_test =
train_test_split(x, y, train_size = 0.7, random_state = 3)

print([x_train.shape, x_test.shape, y_train.shape, y_test.shape])

7:3 for training and test. Since Shuffle is Default and True, set only random_state.

The elements are [[105, 4), (45, 4), (105, 3), (45, 3)], respectively, and it was confirmed that they could be properly divided.

model = models.Sequential()

model.add(layers.Dense(64, activation ='relu', input_shape = (x.shape[1], )))
model.add(layers.Dense(64, activation ='relu'))
model.add(layers.Dense(3, activation ='softmax'))

model.summary()

model.compile(optimizer ='rmsprop', loss ='categorical_crossentropy', metrics = ['accuracy'])

ReLU is used for the activation function of each layer and softmax is used for the activation function of the last layer. (See Table 4-1 on p.119)

Visualize each layer with model.summary()

The optimizer has selected this from the description that rmsprop is sufficient.

model.fit(x_train, y_train, epochs = 100, batch_size = 1, verbose=0)

result = model.evaluate(x_test, y_test)

print(result)
#[0.15816927485995821, 0.9555555582046509]

epochs is 100 for the time being, batch_size is 1 as a matter of course.

As mentioned above, although it was a simple structure, 96% accuracy was obtained as a result.
