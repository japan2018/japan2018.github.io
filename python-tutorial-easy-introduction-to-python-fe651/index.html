Use Python as a calculator

Numerical value

Number type

  • Integer has type int
  • Numbers with fractions have type float (float)
  • Division always returns type float
>>> 2 + 2
Four
>>> 50-5 * 6
20
>>> (50-5 * 6) / 4
5.0
>>> 8/5
1.6
>>> 17/3
5.66666666666667

arithmetic operations

  • Use the // operator if you want to perform integer division by rounding down (if you want to discard the remainder)
  • Use the % operator if you want to get only the remainder
>>> 17 // 3
Five
>>> 17% 5
2
>>> 5 * 3 + 2
17
  • Power can be obtained by using the ** operator
>>> 5 ** 2
twenty five
>>> 2 ** 7
128
  • If the operand types are mixed, integers are converted to floating point numbers
>>> 3 * 3.75 / 1.5
7.5
>>> 7.0 / 2
3.5

variables

  • The equal sign (=) is used to assign a value to a variable
>>> width = 20
>>> height = 5 * 9
>>> width * height
900
  • An error occurs if you try to use a variable without “defining” (assigning a value)
#If you try to access an undefined variable, NameError will occur because the variable is not defined.
>>> n
Traceback (most recent call last):
File "<stdin>", line 1, in <module>
NameError: name'n' is not defined
  • In interactive mode, the last displayed value is assigned to the variable “_” (underscore)
>>> tax = 12.5 / 100
>>> price = 100.50
>>> price * tax
12.5625
>>> price + _
113.0625
>>> round(_, 2)
113.06

Other numeric types

  • Various numeric types such as Decimal and Rational are supported
  • Complex numbers are also supported, use the suffix “j” or “J” to indicate their imaginary part (like 3+5j)

string

Quotation marks

  • You can use single quotes (’…') and double quotes (”…") in quotes and both will give the same result -There is no need to escape double quotes within single quotes (single quotes need to be escaped like \') -You don’t need to escape single quotes inside double quotes
  • Backslash (\) can escape quote characters
  • Backslash is entered and displayed as a backslash on Japanese keyboards and fonts
>>>'spam eggs' # single quote
'spam eggs'
>>>'doesn\'t' # Escape single quotes with \ or...
"doesn't"
>>> "doesn't" # ... use double quotes
"doesn't"
>>>'"Yes," he said.'
""Yes," he said.'
>>> "\"Yes,\" he said."
""Yes," he said.'
>>>'"Isn\'t," she said.'
'"Isn\'t," she said.'
  • In the interpreter, strings are enclosed in quotation marks, special characters are output with \ escaped
  • Display quotes are double quotes only if the string itself contains single quotes and no double quotes, otherwise it is single quotes.
>>>'"Isn\'t," she said.'
'"Isn\'t," she said.'
>>> print('"Isn\'t," she said.')
"Isn't," she said.
>>> s ='First line.\nSecond line.'
>>> If you print without s # print(), \n will be included in the output
'First line.\nSecond line.'
>>> print(s) # print() causes a line break at \n
First line.
Second line.

raw characters

  • Use raw string to prevent the characters prefixed with “" from being interpreted as special characters
>>> print('C:\some\name') # \n is a newline
C:\some
ame
>>> print(r'C:\some\name') # Note the r before the quotes
C:\some\name

Triple quote

  • It is possible to write a string literal over multiple lines -Use triple quotes (""”…””” or’'’…''') -Put a \ at the end of the line to avoid automatically including end-of-line characters in the string
# Like this
print("""\
Usage: thingy [OPTIONS]
- h Display this usage message
- H hostname Hostname to connect to
""")

# You will get output similar to the following (note that the first line break is not included)
Usage: thingy [OPTIONS]
- h Display this usage message
- H hostname Hostname to connect to

String concatenation

  • Strings can be concatenated with the + operator and repeated with the * operator
Repeat # un three times and add ium at the end
>>> 3 *'un' +'ium'
'unununium'
  • Enumerated string literals (quoted ones) are automatically concatenated
  • Useful when you want to split a long string -Valid only between literals, not variables or expressions
>>>'Py''thon'
'Python'
>>> text = ('In the parentheses, enter the character string
'Let's put it in and connect it. ')
>>> text
'Let's put the long string in parentheses and connect them. '
  • Use + to connect variables and literals, and to connect variables
>>> prefix ='Py'
>>> prefix +'thon'
'Python'

index

  • Character strings can be indexed (specified by serial number) -First character is 0 -The character here is a character string of length 1.
>>> word ='Python'
>>> word[0] # character at position 0
'P'
>>> word[5] # character at position 5
'n'
  • Negative number can be used for index, indicating counting from the right
  • -0 is the same as 0, so negative indices start at -1
>>> word ='Python'
>>> word[-1] # last character
'n'
>>> word[-2] # penultimate character
'o'
>>> word[-6]
'P'
  • Specifying too large an index results in an error
>>> word[42] # word is 6 characters
Traceback (most recent call last):
File "<stdin>", line 1, in <module>
IndexError: string index out of range

Slicing

  • You can get individual characters by using index, while you can get substring by slicing
>>> word ='Python'
>>> word[0:2] # Characters from position 0 to 2 (inclusive of 0) (excluding 2)
'Py'
>>> word[2:5] # Characters from position 2 to 5 (inclusive of 2) (excluding 5)
'tho'
  • Note that the start point is always included and the end point is always excluded
  • This makes s[:i] + s[i:] always equivalent to s
>>> word ='Python'
>>> word[:2] + word[2:]
'Python'
>>> word[:4] + word[4:]
'Python'
  • If the first character is omitted, the default value is 0
  • Default value when second character is omitted is string size
>>> word ='Python'
>>> word[:2] # Characters from the first character to position 2 (not including 2)
'Py'
>>> word[4:] # Characters from position 4 (including 4) to the end
'on'
>>> word[-2:] # Characters from position-2 (inclusive) to the end
'on'
  • How to learn slicing -** An index is a number that indicates the “between” the letters. -** The left edge of the first character is 0 ** -** Index n is to the right of the last character in a string of n characters ** -** The slicing [i:j] from index i to j consists of all characters between boundary i and boundary j ** -** For non-negative indexes, the slicing length is the difference between the two indexes (word[1:3] has a length of 2)**
 +---+---+---+---+---+---+
 | P | y | t | h | o | n |
 +---+---+---+---+---+---+
 0 1 2 3 4 5 6
- 6 -5 -4 -3 -2 -1
  • With slicing, even if you specify an index outside the range, it will be processed well
>>> word ='Python'
>>> word[4:42]
'on'
>>> word[42:]
''

list

Overview

  • Python has several types for complex data, which can be used to group other types of values
  • The most versatile is a list, and put comma-separated values (items) in the inside of square brackets
  • The list can contain items of different types, but usually all have the same type
>>> squares = [1, 4, 9, 16, 25]
>>> squares
[1, 4, 9, 16, 25]

Index and slicing

  • Like strings (like all other sequence types), lists can be indexed and sliced
>>> squares = [1, 4, 9, 16, 25]
>>> squares[0] # indexing returns items
1
>>> squares[-1]
twenty five
>>> squares[-3:] # Slicing creates and returns a new list
[9, 16, 25]
  • Substitution to slicing is also possible
  • You can change the length of the list and clear all the contents of the list
>>> letters = ['a','b','c','d','e','f','g']
>>> letters
['a','b','c','d','e','f','g']
>>> # replace some values
>>> letters[2:5] = ['C','D','E']
>>> letters
['a','b','C','D','E','f','g']
>>> # remove these
>>> letters[2:5] = []
>>> letters
['a','b','f','g']
>>> # Clear the list. Replace all elements with empty list
>>> letters[:] = []
>>> letters
[]
  • Slicing operations always return a new list with the requested elements
  • The following slicing is a movement to make a new shallow copy and return it
>>> squares = [1, 4, 9, 16, 25]
>>> squares[:]
[1, 4, 9, 16, 25]

List concatenation

  • List also supports operations such as concatenation
>>> squares = [1, 4, 9, 16, 25]
>>> squares + [36, 49, 64, 81, 100]
[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

Mutable

  • The character string was immutable, but the list is mutable, that is, the contents can be exchanged.
>>> cubes = [1, 8, 27, 65, 125] # Something is wrong
>>> 4 ** 3 # 4 raised to the third power is 64. Not 65!
64
>>> cubes[3] = 64 # replace wrong values
>>> cubes
[1, 8, 27, 64, 125]

Add item

  • You can add items to the end of the list by using the append() method
>>> cubes = [1, 8, 27, 64, 125]
>>> cubes.append(216) # 6 cubed added
>>> cubes.append(7 ** 3) # 7 cubed added
>>> cubes
[1, 8, 27, 64, 125, 216, 343]

List nesting

  • Lists can be nested
>>> a = ['a','b','c']
>>> n = [1, 2, 3]
>>> x = [a, n]
>>> x
[['a','b','c'], [1, 2, 3]]
>>> x[0]
['a','b','c']
>>> x[0][1]
'b'

the term

literal

  • A thing that simply describes the actual value of a number or string, such as a constant

Sequence type

  • Consists of multiple elements, with elements arranged in order (list, etc.)

shallow copy

  • A copy of a pointer that does not involve copying the object
