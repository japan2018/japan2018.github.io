Is it so easy! ? How to write Python

Some people may think, “That said, Python is a programming language, so you often learn it, right?” Certainly, there are a certain number of things to learn because it’s a programming language, but Python is still super easy to get started with. First, try downloading Python on the official website. https://www.python.jp/ On the above site, click “Download” Python 3.7.2 to install it. For a detailed explanation of here, please googling with “Python installation method”. Anyway, if you can install Python, please start the following program

print('Hello World!')

Yes, the output is already “Hello World!”. Python has the advantage of outputting Hello World at the ultimate speed. By the way, many programming languages require some type of output or variables to be “typed”, but the nice thing about Python is that it types arbitrarily (good or bad). For good or bad, suppose you wrote the following code in Python.

a = 2
b ='hello'
print(a)
print(b)
#2
#hello

The number 2 is assigned to a, and the character string hello is assigned to b. This is not a mistake by itself, so if you print each with a print statement, it will output what you put in a variable properly. And since the contents of the variable are judged to be numeric type (int type) and string type (str type), it is not necessary to declare them like other languages, but this is the cause. There are times when bugs occur, so I want you to gradually remember it and feel that “Oh, Python has some weak points.” Anyway, even from a point like this, even non-programmers can do programming tricks suddenly, so it’s very fun, and I recommend it because it gives you a versatility like “I SUGEEEE”.
