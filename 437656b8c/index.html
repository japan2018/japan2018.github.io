[Python] Jupyter Notebook Magic Command My Summary

Nov 25, 2019 Python Beginner Jupyter 

### Prerequisites
  • macOS Catalina 10.15.1
  • anaconda 4.17.12
  • jupyter-notebook 5.0.0
  • Python 3.6.9

What is a magic command

Magic Commands is a mechanism provided by the IPython kernel. In addition to the functions related to the operation of Notebook, it provides utility functions such as directory movement and file list display.

Magic command basics

Enter ``%’’ at the beginning and then enter the command. The command to output the current directory is as follows.

%pwd

The magic command %pwd'' seems to be the same as !pwd’’ using the shell, but the command execution using ``!’’ depends on the OS shell. On the other hand, the magic command depends on the function provided by the IPython kernel.

If the magic command changes any value, you can assign the return value to a variable as if you were calling a Python function.

curr_dir = %pwd
curr_dir

Magic commands that are easy to use

  • %time

    A magic command that measures the execution time of Python.

    %time sum(range(10000))
CPU times: user 226 μs, sys: 0 ns, total: 226 μs
Wall time: 230 μs
Output Description
Wall time Time taken from the start to the end of the program
CPU times: user user CPU time. The time taken to execute the program itself
sys system CPU time. Time required for OS system call

  • %timeit

    A magic command that summarizes and returns the measured values of multiple attempts. In the following cases, the time when 1000 times of repeated processing is tried 7 times is output.

    %timeit sum(range(10000))
224 µs ± 21.8 µs per loop (mean ± std.dev. of 7 runs, 1000 loops each)

    The number of loops and the number of trials can be specified as options.

    # 2000 loops, 5 trials
%timeit -n 2000 -r 5 sum(range(10000))
215 µs ± 17.9 µs per loop (mean ± std.dev. of 5 runs, 2000 loops each)

    When measuring with %timeit for multiple lines of Python code, continue the two leading % as shown below to make %%. (Cell magics)

    %%timeit -n 1000 -r 3
  
for i in range(1000):
  i*2
  
75.2 µs ± 9.58 µs per loop (mean ± std.dev. of 3 runs, 1000 loops each)

  • %history

    Magic command to get the execution history of code cells as a list.

    # Get the last 5 history
%history -l 5

  • %ls

    A magic command that behaves like the UNIX command ls. Unlike !ls, %ls determines the OS type and uses the command to execute internally. (ls command for macOS, dir command for Windows)

  • %autosave

    You can change the frequency of Auto Save. (Default 120 seconds)

    # Execute Auto Save once every 60 seconds.
%autosave 60

  • %matplotlib

    Magic command for setting related to Matplotlib.

    –%matplotlib inline

    If inline is specified, the graph will be drawn immediately below the code cell.

    -%matplotlib tk

    When tk is specified, the interactive graph is output in another window.

    –%matplotlib notebook

    If notebook is specified, an interactive graph will be output just below the code cell.

    -To change the specified output method, restart the kernel.
