What is #Python/Tkinter?

Source: Free encyclopedia "Wikipedia"

Tkinter is a standard library (widget toolkit) for building and operating GUI from Python.

The Tk part of Tcl/Tk is made available to Python, and the usage is designed to match Tcl/Tk as much as possible.
This makes it possible to easily create an application with a GUI screen from Python, which is a scripting language.

##window

First, import the module…

With #Python2...
import Tkinter #tkinter body
import ttk #tkinter extension (?)
With #Python3...

from tkinter import ttk #tkinter extension (?)

Widget

###button

def callback():
    print("Hello!!")

button = ttk.Button(root, text="[text to display]", command=callback)

In command, specify the name of the function to be executed when the button is pressed. If you want to specify arguments,

def callback(hogehoge):
    print(hogehoge)

button = ttk.Button(root, text="[text to display]", command=lambda: callback("Hello!!!"))

Widget placement

In tkinter, it is necessary to place the widget after defining it. There are three placement methods.

Packing method (pack)

With the packing method (pack), widgets are placed in empty areas of the window. (vocabulary…)

[Widget].pack ([options])

[Widget] is a widget instance, (If you’re crazy about instances and functions, please read other Python introductory articles.) The following can be specified in Options: ・Side You can set which direction to place. Possible values: “Right” right, “left” left, “bottom” bottom, “top” top ・Fill You can set which direction to reclaim. Possible values: “Both”-both directions “X”-x direction (vertical?) “Y”-y direction (sideways?)
