A problem

station_b = s.count('B')
if station_a != 3 and station_b != 3:
    print('Yes')

B problem

Problem

Way of thinking You just have to count how many A+B pairs are in N. The number of pairs of $A+B *A$ does not include the remainder. So calculate the remainder with $N %(A + B)$ and add it. However, the remainder may be larger than A, so add min(a,(n %(a + b))).

n, a, b =map(int,input().split())
p = n // (a + b)
ans = a * p + min(a,(n %(a + b)))
print(ans)

C problem

Problem 1WA Thoughts I calculated it normally. The reason for 1WA is that the stop of the for statement is set to 1000 instead of 1001. Because of this, either case couldn’t handle the case where the answer was 1000. Everyone should be careful of the for stop.

1009*0.10 will be 100 when rounded, so you have to set it to 1010

import math
a, b = map(int,input().split())
ans = []
for i in range(1010): # I made 1000 here
    price_8 = math.floor(i * 0.08)
    price_10 = math.floor(i * 0.1)
    if price_8 == a and price_10 == b:
        ans.append(i)

if len(ans) != 0:
    print(min(ans))
else:
    print(-1)
