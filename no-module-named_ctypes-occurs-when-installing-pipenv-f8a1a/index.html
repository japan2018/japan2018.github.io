No module named’_ctypes’ occurs when installing pipenv

Apr 18, 2020 Python Python3 Pipenv 

## 環境

OS: Amazon Linux 2 Python: 3.8.1

事象

pyenvでpython3.8.1をinstall後、pip install pipenvを実行すると以下のエラーが発生し、installできない。

対処方法

以下で解決

pyenv uninstall 3.8.1
sudo yum install libffi-devel
pyenv install 3.8.1

この状態で

pip install pipenv

すればinstallできます！
