[Python] Ohm’s Law with Python

Nov 10, 2019 Python Python3 

#Introduction

While studying Python, I found that complex numbers could be easily calculated, so I tried to see if Ohm’s law used in electrical engineering could be implemented in Python really easily.

Ohm’s Law (DC)

It’s one of the most important laws in electrical engineering and the first to learn.

V = IR
  • $V$: voltage, $I$: current, $R$: resistance The above Ohm’s law is written in Python. Set to calculate for empty parameter among $V$, $I$, and $R$.
#V = IR for DC circuit
print('Empty a parameter to calculate')

V = float(input('Input V :') or 0)
I = float(input('Input I :') or 0)
R = float(input('Input R :') or 0)

if not V:
    V = R * I
elif not I:
    I = V / R
elif not R:
    R = V / I
else:
    print('Empty a parameter to calculate')

print('V =', V,'[V]')
print('I =', I,'[A]')
print('R =', R,'[Ω]')

It is the first input part, but it is supposed to calculate the parameter entered with an empty. Therefore, if you convert the input as it is as follows, an error will occur during conversion with an empty input parameter.

V = float(input('Input V :'))

For that reason, I wrote it so that it will be input once with str type, and will be converted only when the input is not empty. I really wish I could write something a little easier… I checked the initial value of input or something that can not be set, but I could not find such an argument orz

2019/11/10 update In the comment, I learned that the following notation can be done. By doing this, you could convert it to a number without an if statement. it’s amazing!

V = float(input('Input V :') or 0)

The or operator is said to be true or false for the value on the left and returns the result. Deep, I see… https://docs.python.org/ja/3/library/stdtypes.html#boolean-operations-and-or-not

Ohm’s Law (Exchange)

This is the main topic from here. I didn’t use any complex numbers for DC, but for AC I need to calculate complex numbers with impedance (Z) added.

v = iz
  • $v$: voltage, $i$: current, $z$: impedance
z = \sqrt{(r^2+x^2)}
  • $r$: resistance, $x$: reactance

Where it differs from DC, $x$ (reactance) is added to the parameters. This reactance represents the imaginary component in the AC circuit.

Now, let’s write Ohm’s law for AC circuits in Python. Try to calculate the empty parameter among $v$, $i$, $r$ and $x$.

# v = iz for AC circuit

print('Empty a parameter to calculate')

v = complex(input('Input v[V] :') or 0)
i = complex(input('Input i[A] :') or 0)
r = complex(input('Input r[Ω] :') or 0)
x = complex(input('Input x[jΩ] :') or 0)*1j

if not v:
    z = (r ** 2 + (x / 1j) ** 2) ** (1/2)
    v = z * i
elif not i:
    z = (r ** 2 + (x / 1j) ** 2) ** (1/2)
    i = v / z
elif not r:
    z = v / i
    r = (z ** 2-(x / 1j) ** 2) ** (1/2)
elif not x:
    z = v / i
    x = (z ** 2-r ** 2) ** (1/2) *1j
else:
    print('Empty a parameter to calculate')

print('v =', v,'[V]')
print('i =', i,'[A]')
print('r =', r,'[Ω]')
print('x =', x,'[jΩ]')
print('z =', z,'[Ω]')

I was surprised to find that it was difficult to input real numbers and convert from real numbers to complex numbers internally. It is easy if you know that you can convert from real number to imaginary number with “*1j”.

The result is output as follows.

Input x[jΩ]:
v = (10+0j) [V]
i = (2+0j) [A]
r = (3+0j) [Ω]
x = 4j [jΩ]
z = (5+0j) [Ω]
