[Python] Let’s execute a module periodically using schedule

Dec 30, 2019 Python schedule 

Introduces the schedule module which is useful for making the module run periodically. schedule is a useful module when you want to perform the same task at regular intervals (every few minutes, hours, or days). It is a convenient module for situations such as collecting information by web scraping.

After summarizing the usage of the basic schedule module, I will introduce a module that executes scraping regularly as an implementation example.

schedule installation

The schedule module can be obtained with the pip command. Execute the following command on the command prompt to install.

pip install schedule

#Sample code

import schedule
import time

# Execute job function
def job():
    print("run job")


#1 register job execution every minute
schedule.every(1).minutes.do(job)

#1 register job execution every hour
schedule.every(1).hours.do(job)

#AM11:00 job execution registration
schedule.every().day.at("11:00").do(job)

# Register job execution on Sunday
schedule.every().sunday.do(job)

# Register the job execution on Wednesday 13:15
schedule.every().wednesday.at("13:15").do(job)

# Job execution monitoring, execute job function at specified time
while True:
    schedule.run_pending()
    time.sleep(1)

This is a sample module of schedule. The job function is executed at a certain interval, and “job execution” is displayed in the module. schedule.every is a description that registers the job to be executed and the execution interval. You can run it every few minutes, every few hours, or even at a specific date and time.

# Run job every 1 minute
schedule.every(1).minutes.do(job)

The job registered with schedule.every above is executed by schedule.run_pending() below. Normally, just calling ```schedule.run_pending()`’’ will end the module once the job is executed, so it is necessary to put it in an infinite loop state with a while statement. .. By setting the infinite rule state, it is possible to continue performing the same processing at regular intervals.

while True:
    schedule.run_pending()
    time.sleep(1)

Regular collection of Yahoo news using # schedule Using the schedule module introduced this time, I created a module that executes scraping periodically. It is a module that accesses Yahoo News every hour and acquires the news title and URL.

from urllib.request import urlopen
from urllib.error import HTTPError
from urllib.error import URLError
from bs4 import BeautifulSoup
import re
import schedule
import time

def job():
    try:
        html = urlopen('https://news.yahoo.co.jp/topics')
    except HTTPError as e:
        print(e)
    except URLError as e:
        print(e)

    else:

# Access Yahoo topics and collect news information
        bs = BeautifulSoup(html.read(),'lxml')
        newsList = bs.find('div', {'class':'topicsListAllMain'}).find_all('a')

# Get and display the news title and URL from the acquired List
        for news in newsList:
            if re.match('^(https://)', news.attrs['href']):
                print(news.get_text())
                print(news.attrs['href'])

#1 run the job every hour
schedule.every(1).hours.do(job)

while True:
    schedule.run_pending()
    time.sleep(1)
