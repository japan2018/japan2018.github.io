#Definition of symbol

The specified characters are treated as symbols (characters that represent variables).

a = sym.Symbol('a')

If you want to define them all together, do the following.

a, b, c, x, y = sym.symbols("a b c x y")

Treat the specified character as a function.

f = sym.Function('f')
g = sym.Function('g')

Factor

# Factorization
expr = x**2 + 2*x + 1
sym.factor(expr)

$$\left(x + 1\right)^{2}$$

#Solve equation

#Equalities in English are equation or equality
eq = sym.Eq(expr)
eq

$$x^{2}-12 x + 8 = 0$$

# You can specify the right side like this
eq = sym.Eq(x**2-12*x, -8)
eq

$$x^{2}-12 x = -8$$

#Solve equation
sym.solve(eq)

$$\left [- 2 \sqrt{7} + 6, \quad 2 \sqrt{7} + 6\right ]$$

#Allows to handle expressions using algebra
eq = sym.Eq(a * x ** 2 + b * x + c)
eq

$$a x^{2} + b x + c = 0$$

#solve about x
sym.solve(eq, x)

$$\left [\frac{- b + \sqrt{- 4 ac + b^{2}}}{2 a}, \quad-\frac{b + \sqrt{- 4 ac + b^{2} }}{2 a}\right ]$$

# Simultaneous equations
eq1 = 3 * x + 5 * y-29
eq2 = x + y-7

sym.solve([eq1, eq2])

$$ x :3, y :4$$

#Differential

expr = 2 * x ** 2 + 5 * x-3
expr

$$2 x^{2} + 5 x-3$$

#Differential
sym.Derivative(expr)

$$\frac{d}{d x} \left(2 x^{2} + 5 x-3\right)$$

#Calculate the derivative
sym.Derivative(expr).doit()

$$4 x + 5$$

# Even if you write it like this
sym.diff(expr)

$$4 x + 5$$

Notation as an equation using #sym.Eq
sym.Eq(sym.Derivative(expr), sym.diff(expr))

$$\frac{d}{d x} \left(2 x^{2} + 5 x-3\right) = 4 x + 5$$

# Same meaning as above, but if there is only one variable it can be omitted as above.
#If there are two or more variables, you must specify what differentiates themsym.Eq(sym.Derivative(expr, x), sym.diff(expr, x))

$$\frac{d}{d x} \left(2 x^{2} + 5 x - 3\right) = 4 x + 5$$

# ２階微分
sym.Eq(sym.Derivative(expr, x, 2), sym.diff(expr, x, 2))

$$\frac{d^{2}}{d x^{2}} \left(2 x^{2} + 5 x - 3\right) = 4$$

# xについて微分して x=1を代入
sym.diff(expr).subs(x, 1)

$$9$$

常微分方程式

$2 f’(x) + 5 f(x) = 0$ を解く

# 常微分方程式
eq = sym.Eq(2 * f(x).diff(x,1) + 5 * f(x))
eq

$$5 f{\left (x \right )} + 2 \frac{d}{d x} f{\left (x \right )} = 0$$

# 一般解
ans = sym.dsolve(eq)
print(ans)
ans

Eq(f(x), C1*exp(-5*x/2))

$$f{\left (x \right )} = C_{1} e^{- \frac{5 x}{2}}$$

# 特殊解
ans = sym.dsolve(eq, ics={f(0):1})
print(ans)
ans

Eq(f(x), exp(-5*x/2))

$$f{\left (x \right )} = e^{- \frac{5 x}{2}}$$

plot(ans.rhs, (x, -10, 10)) # rhs は右辺(Right-hand side)の意味


<sympy.plotting.plot.Plot at 0x7faf54dd4320>

# 特殊解で x = 2 のとき

ans.subs(x, 2)

# evalf というメソッドを使えば浮動小数点まで展開してくれる
