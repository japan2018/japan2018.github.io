An index that indicates how far the data is from the average

Unbiased distribution

The sample variance is a value obtained by further calculating the variance using the sample mean, but this value causes an underestimate bias.
Therefore, the one without bias becomes the universal variance.

\sigma^2=\frac{1}{N-1}\sum_{i=1}^{n} (x_i-\mu)^2

numpy.var(data, ddof = 1)

After that, unbiased distribution is used. #standard deviation The square root of the variance

\begin{align}
\sigma&=\sqrt{\sigma^2}\\
&=\frac{1}{N-1}\sum_{i=1}^{n} (x_i-μ)^2
\end{align}

numpy.std(data, ddof=1)

#Covariance

  • When the covariance is greater than 0
    → If one variable takes a large value, the other also becomes large
    → there is a positive correlation.
  • When covariance is smaller than 0
    → If one variable takes a large value, the other becomes small
    → there is a negative correlation.
Cov(x,y)=\frac{1}{N}\sum_{i=1}^{n-1} (x_i-\mu_x)(y_i-\mu_y)

#Probability density Probability in continuous variable 1. For continuous variables, the probability of a particular value is always 0. That’s because some values have an infinite number of decimal points. For example, a person’s height cannot be exactly 160 centimeters. However, it can be calculated by the “probability of a person between 159 cm and 160 cm.” The probability is the “probability density”. The probability density from e.g. 0 to the maximum value is 1.

c.f. The probability of discrete variables 2 is the probability that many people will learn at school. (P(x)=1/4)

In particular, when considering the probability that a variable X that takes a real value takes x<=X<=x+⊿x, when ⊿x→0, P(x) is called the probability density of x.

random variable

When calculating the probability, the variable to be calculated is called a random variable. The probability that x=2 is 1/3. 2 at this time is the establishment variable.

Probability density function of normal distribution

N(x|\mu, \sigma^2)=\frac{1}{\sqrt{2\pi\sigma^2}}e^{-{\frac{(x-\mu)^2}{2 \sigma^2}}}

Example: When random variable x=3, mean=4, standard deviation=0.8

>>>x = 3
>>>mu = 4
>>>sigma = 0.8
>>>1 / (numpy.sqrt(2 * sp.pi * sigma**2)) * numpy.exp(- ((x-mu)**2) / (2 * sigma**2))
>>>0.228

You can also easily do it with the function below.

>>>stats.norm.pdf(loc = 4, scale = 0.8, x = 3)
>>>0.228

Cumulative distribution function, lower probability, percentage point

F(x)=P(X\leq x)

A function represented as. That is, “a function that calculates the probability of being less than a certain value”. The value obtained here is called the lower probability. Also, x at this time is called a percentage point. In case of normal distribution, it is calculated by the integral calculation below. Also, use the scipy.stats.hoge.cdf function

P(X\leq x)=\int_{-\infty}^{x}\frac{1}{\sqrt{2\pi\sigma^2}}e^{-{\frac{(x-\mu )^2}{2\sigma^2}}}dx

>>>import scipy as sp
>>>from scipy import stats
>>>stats.norm.cdf(loc = 4, scale = 0.8, x = 3) #loc is mean, scale is standard deviation
>>>0.106

Percentage function-ppf function

Percentage point with a lower probability of 2.5%

Pearson residual

Interpreted as “ordinary residual divided by the standard deviation of the distribution.” Example: Binomial distribution

  • When p=0.5, it becomes 0 or 1, but it is half, so the probability of guessing is low. The gap at this time is recognized as a “small gap” in the Pearson residual.
  • When p=0.9, the probability of guessing should be high. If the guess is wrong at this time, it is recognized as a “large deviation” in the Pearson residual.
\begin{align}
Pearson \quad residuals &= \frac{y-N\hat{p}}{\sqrt{N\hat{p}\quad(1-\hat{p}\quad)}}\\
&=\frac{y-\hat{p}}{\sqrt{\hat{p}\quad(1-\hat{p}\quad)}}
\end{align}
\\
\hat{p}\quad represents the estimated success probability.

The sum of squared Pearson residuals is the Pearson chi-square statistic.
