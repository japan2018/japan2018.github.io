Overview of this document

The machine learning algorithms described in this document are as follows.

In Chapter 2, Linear regression, regularization, logistic regression, SVC (Support Vector Machine Classification) Decision tree, random forest Naive Bayes

In Chapter 3, Principal component analysis, k-means, Gaussian mixture model

In Chapter 4, Gradient boosting decision tree, elbow method and silhouette analysis, t-SNE, Abnormality detection Novelty Detection, Outlier Detection

And in Chapter 5, Machine learning system construction flow and performance evaluation

Is explained.

Excerpts of about 6 pages of the image on paper. The atmosphere is as follows. *Although two pages are listed, they are not two consecutive pages.

For each algorithm, a simple implementation example is given, followed by a description of the operating principle of the algorithm (algorithm, mind).
