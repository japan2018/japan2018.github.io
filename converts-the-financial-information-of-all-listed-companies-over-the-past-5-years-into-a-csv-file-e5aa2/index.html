Converts the financial information of all listed companies over the past 5 years into a CSV file

Mar 19, 2020 Python Zhu Dong financial analysis xbrl EDINET 

## Introduction

In the previous article, I wrote about how to obtain financial statement information from XBRL format data obtained from the EDINET of the Financial Services Agency.

Get information on the financial statements of listed companies from XBRL.

After that, EDINET released a web API called EDINET API, which made it possible to automatically download XBRL files.

This time, I downloaded a XBRL file of all listed companies for the past 5 years and created a program to convert it into a CSV file.

The CSV file of the processing result is below. http://lkzf.info/xbrl/data/summary-join.csv The financial information is written line by line as shown below. The closing figures are the consolidated closing values.

Simple application example

I made a frequency distribution table of gross profit margin (gross margin) from a CSV file using spreadsheet software (LibreOffice Calc). I think there are various other uses as well.

CSV file creation procedure

Financial analysis sample app

I also made a web application that uses financial data from CSV files for financial analysis. You can access the application from the following URL. http://lkzf.info/xbrl/chart/ We have confirmed the operation on Chrome and FireFox. I used Plotly.js to draw the graph.
