↓Problems that have been taken up as a topic in past articles! (At this time, I solved with a triple for sentence!) [Python] ABC133B (top right triangle problem) [At Coder]

There is no problem with the triple for sentence, This time I tried to make it a double for sentence.

def LI(): return list(map(int,input().split()))
while 1:
    n,x = LI()
    if n==x==0:
        break
    ans = 0
    for i in range(1,n+1):
        for j in range(1,n+1):
            k = x-i-j # where k is [1,n]
            if 1<=k<=n and i<j<k:
                ans += 1
    print(ans)

** (Added May 1, 2020) ** Now I write code like this.

import itertools
def LI(): return list(map(int,input().split()))
while 1:
    n,x = LI()
    if n==x==0:
        break
    ans = 0
    for i,j,k in itertools.product(range(1,n+1),repeat=3):
        if i<j<k and i+j+k==x:
            ans += 1
    print(ans)

If you use itertools.product, nesting will not be deep! So, even a triple for sentence is easy to understand.

import itertools
def LI(): return list(map(int,input().split()))
while 1:
    n,x = LI()
    if n==x==0:
        break
    ans = 0
    for i,j,k in itertools.combinations(range(1,n+1),3):
        if i+j+k==x:
            ans += 1
    print(ans)

def I(): return int(input())
N = I()
ans = 0
if N >=105:
    for i in range(105,N+1,2):
        count_yakusuu = 0
        for j in range(1,int(i**0.5)+1):
            if i%j==0:
                count_yakusuu += 2
        if count_yakusuu==8:
            ans += 1
print(ans)

I came up with two solutions!
One is the usual greedy method. The other is DP.

Solution 1 (greedy method)
Ask for all possible answers and overwrite the `ans`!

```python:test.py
S = input()
ans,temp = 0,0
for x in S:
    if x in'ACGT':
        temp += 1
    else:
        temp = 0
    ans = max(ans,temp)
print(ans)

Solution 2 (DP) With the second argument of enumerate (default 0), The index starts from that number!

S = input()
dp = [0]*(len(S)+1) #1_indexed
for i,x in enumerate(S,1):
    if x in'ACGT':
        dp[i] = dp[i-1]+1
print(max(dp))

** (Added May 26, 2020) ** Solution 3 (full search for substrings) Using the duplication combination itertools.combinations_with_replacement, You can search all substrings! ! ! Wow! ! !

import itertools
S = input()
ans = 0
for i,j in itertools.combinations_with_replacement(range(len(S)),2):
    if all([x in'ACGT' for x in S[i:j+1]]):
        ans = max(ans,j+1-i)
print(ans)

N,M = LI()
A = [LI() for _ in range(N)]
temp = [[0]*(M*(M-1)//2) for _ in range(N)]
for i,x in enumerate(A):
