I tried VS Code Jupyter notebook

Nov 21, 2019 Python pandas Jupyter VSCode 

I tried #VSCode's Jupyter notebook.

There was an article of VS Code Jupyter notebook in the magazine, so I tried it. I made a Jupyter notebook experience very easily, so I will make a note of my footsteps.

Assumption

  • Tried on Mac.
  • Homebrew has python installed as python3.
  • When I put python3, pip3 is also included

###python3

$ which python3
/usr/local/bin/python3

$python3 --version
Python 3.7.5

###pip3

$ which pip3
/usr/local/bin/pip3

$ pip3 --version
pip 19.3.1 from /usr/local/lib/python3.7/site-packages/pip (python 3.7)

Add the module to be used on Mac

A notebook is also included with the ipython module

$ pip3 install "ipython[notebook]"

↑ I was able to do it smoothly on Mac, but with Windows7+python3.8.0, it failed in the module called pywinpty. It worked fine with Windows7+python3.7.4.
