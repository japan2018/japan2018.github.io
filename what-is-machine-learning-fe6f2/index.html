## Differences between machine learning, statistics and data science

Machine Learning (ML), Statistics, Data Science. All of these words are often heard due to the rapid spread of AI in recent years, but what is the difference between them? There is no definite answer, but to put it simply, firstly Data Science is a field that combines specialized fields such as statistics, computer science, and business. I think it will be easier to understand if you look at the figure below. Currently, data scientists are active in various fields and are being adopted by various companies, from insurance companies to human resources companies to consulting firms. ▽ All areas have the common point of conducting analysis using data and specialized knowledge and explaining the analysis results. On the other hand, machine learning can be thought of as the intersection of statistics and computer science in data science. The points of solving problems using data of statistics and solving problems using algorithms of computer science also apply to machine learning. On the other hand, in machine learning, we focus on **how well the results apply, whereas in statistics we focus on mathematical correctness and in computer science how fast and accurately we process. The main focus is on what you can do.

Machine learning classification

Machine learning methods are broadly classified into three types: “supervised learning”, “unsupervised learning”, and “reinforcement learning”. Both “supervised learning” and “unsupervised learning” are common in that they let machines learn the given data and obtain output. On the other hand, in “supervised learning”, the data given is accompanied by a label indicating whether or not the answer is correct in advance. Since the information on whether or not the answer is correct is not included, the classification that the machine judges by itself is the final output. In “reinforcement learning”, the output (reward) given from the input data is given an evaluation (reward), and learning is promoted accordingly. “Reinforcement learning” has relatively limited applications such as robot walking and shogi games. Supervised/unsupervised learning is further divided into “classification problems” and “regression problems”. The “classification problem” outputs 1 or 0 classification results as output, whereas the “regression problem” outputs continuous numerical data as output. There are various models for calculating output data from input data.

Supervised learning example

  • Whether credit card payment is fraudulent based on payment amount and withdrawal account information

Example of unsupervised learning

  • Classify municipalities into groups based on population, land use ratio, etc.
