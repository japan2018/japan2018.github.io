Specify default value for function argument

Default value can be set in the argument. In the following example, default values are set in two of the three arguments of the showName() method, and the method can be called by specifying only one or two arguments.

def showName(name1, name2 ='Taro', name3 ='Kota'):
    print(name1)
    print(name2)
    print(name3)

print('---')
showName('Hanako','Sachiko','Kazuko') # (1)
print('---')
showName('Yuta','Ryo') # (2)
print('---')
showName('Chisato') # (3)
print('---')
showName('Hiroki', name3 ='Yuji') # (4)

Execution result

Hanako Sachiko Kazuko

Yuta Ryo Kota

Chisato Taro Kota

Hiroki Taro Yuji

(1) When three arguments are specified, the default value is not used (2) When only two arguments are specified, the third argument is processed with the default value (3) If only one argument is set, the second and third arguments will be processed with default values (4) If you want to retain the default value only for the second argument and execute the method processing by specifying the first and third arguments, specify the argument explicitly like this good

Specify multiple return values

When specifying a return value in the return statement, you can return multiple return values by separating them with commas. Since the return type of such a method is tuple, it can be used in the for statement as well.

Description example
def twice(num1, num2):
    num1 *= 2
    num2 *= 2
    return num1, num2

for n in twice(10, 25)
    print(n)

Execution result

20 50

map method

It takes another method and iterable data as an argument, and returns an iterator that applies the processing of that method to each element of the data.

Description example
def twice(num):
    return num*2

a = [1, 2, 3]

b = map(twice, a)

for n in b:
    print(b)

Execution result

2 4 6

Numerical representation

By adding 0b to the beginning, the binary number is expressed, 0o is the octal number, and 0x is the hexadecimal number.

Description example
a = 0b1100 # 12 in binary
b = 0o17 # octal 15
c = 0x2B # Hex 43

Lists and tuples

It is a data type that can store a variable number of data, and the stored data is ordered. The list can increase or decrease the number of data, whereas the tuple cannot change the content after creating the data.

List Tuple
Add data ○ ×
Data retrieval ○ ○
Insert data ○ ×
Delete data ○ ×
Description example
#List operations
a = [0, 1, 2, 3, 4]
a.append(6) # add data at the end
print(a)

a.extend([7, 8, 9]) # Add list data at the end
print(a)

a.insert(5, 10) # Add data at index 5 position
print(a)

del a[5] # Delete the data of index 5
print(a)

b = a.pip(7) # Take out the data of index 7 and assign to b
print(a)
print(b)

c = a.pip() # fetch the last data and assign to c
print(a)
print(c)

a.remove(2) # remove data with value 2
print(a)

Execution result

[0, 1, 2, 3, 4, 6] [0, 1, 2, 3, 4, 6, 7, 8, 9] [0, 1, 2, 3, 4, 10, 6, 7, 8, 9] [0, 1, 2, 3, 4, 6, 7, 8, 9] [0, 1, 2, 3, 4, 6, 7, 9] 8 [0, 1, 2, 3, 4, 6, 7] 9 [0, 1, 3, 4, 6, 7]

In addition to this, count() which counts the total number of data, sort() which sorts the data, reverse() which reverses the order of the data, copy() which copies the data, data Methods such as clear() which delete all of the can be used only on lists.
