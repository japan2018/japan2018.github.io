#How to make?

  1. Get HTML data from Stackoverflow
  2. Morphological analysis
  3. Calculate the frequency of words
  4. Display the most frequently used items as candidates

This time, we will use only the data for one screen of the following HTML page tagged with the keyword “Python” because it is the first part. https://stackoverflow.com/questions/tagged/python

Now let’s get to the code.

Entire code

The whole code looks like this.

from enum import Enum, unique
from typing import List, Tuple, Set, Dict
import requests

from bs4 import BeautifulSoup as bs
from textblob import TextBlob

URL ='https://stackoverflow.com/questions/tagged/python'
PARSER = "html.parser"
FILTER_BY_COUNT = 2


@unique
class PartOfSpeechToLearn(Enum):
    JJ ='Adjective'
    VB ='Verb'
    NN ='Noun'
    RB ='Adverb'


if __name__ =='__main__':
    # Get HTML data and remove html tags
    res = requests.get(URL)
    raw_html = bs(res.text, PARSER)
    texts_without_html: str = raw_html.text

    # morphological analysis
    morph = TextBlob(texts_without_html)
    word_and_tag: List[Tuple[str, str]] = morph.tags

    # Filter words to create a book for vocab
    part_of_speech_to_learn = tuple(pos.name for pos in PartOfSpeechToLearn)
    words_to_learn: Set[str] = {
        wt[0]
        for wt in word_and_tag
        if wt[1] in part_of_speech_to_learn
    }
    words_filtered_by_count: Dict[str, int] = {
        word: morph.words.count(word)
        for word in words_to_learn
        if morph.words.count(word) >FILTER_BY_COUNT
    }

    # Show 50 words that are most frequently used
    words_in_descending_order: List[Tuple[str, int]] = sorted(
        words_filtered_by_count.items(),
        key=lambda x: x[1],
        reverse=True
    )
    for i, word_and_count in enumerate(words_in_descending_order[:50]):
        print(f'rank:{i} word: {word_and_count}')

Set the part of speech to get for the word book

This time we will acquire the following four parts of speech.

  1. Adjective (adjective) Part of speech that modifies a noun such as big or funny
  2. Verb (verb) Part of speech that expresses movement, such as run or write
  3. Noun (Noun) Apple, etc.
  4. Adverb (Adverb) Part of speech that modifies an adjective or verb such as really or surely
@unique
class PartOfSpeechToLearn(Enum):
    JJ ='Adjective'
    VB ='Verb'
    NN ='Noun'
    RB ='Adverb'

Morphological analysis is performed to limit the part of speech included in the word book. TextBlob is used for morphological analysis. For TextBlob, I referred to here.

Filter for word book

After filtering only the part of speech you want to include in the vocabulary, while filtering by the frequency (FILTER_BY_COUNT=2), create a dictionary in which the word is the key and the number of occurrences is the value.

改善点と改善方法

微妙な理由は以下の3点です。(続編執筆中に修正 更新4/23 11:57)

  1. プログラムのみで使用する単語が存在する
  2. case insensitiveになっていない
  3. stackoverflowのページにオーバフィットしている

1の改善方法としては前処理をもう少し高度にしていく必要があります。 3に関しては、stackoverflowの個別質問ページや、その他技術ブログ、技術ニュースなども混ぜて取得データを増やしていく必要がありそうです。
