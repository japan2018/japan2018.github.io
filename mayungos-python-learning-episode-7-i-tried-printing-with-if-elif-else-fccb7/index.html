Mayungo’s Python learning episode 7: I tried printing with if, elif, else

Dec 3, 2019 Python For beginners Getting started with python 

This time we will work on if, elif, and else.

The same content is also shown in the video, so please see it if you like.

When writing a program, there are situations where you want to make different actions depending on the conditions.

Now put 1 in the variable num.

If num is 1, 1 is output. When they are the same, they are written as ==.

num = 1

if num == 1:
    print(1)

1

Since I have not touched on the case where num is other than 1 here, if you set it to 2, nothing will appear.

Create another condition to output the condition of 1 without changing the condition of 1.

Here we use elif.

num = 2

if num == 1:
    print(1)
elif num == 2:
    print(2)

2

There is no problem in writing as many elif as there are.

num = 3

if num == 1:
    print(1)
elif num == 2:
    print(2)
elif num == 3:
    print(3)

3

However, there are times when it is not clear if you write each one.

At that time, use else.

num = 4

if num == 1:
    print(1)
elif num == 2:
    print(2)
elif num == 3:
    print(3)
else:
    print("others")

others

This way, if the condition is not touched by if or elif, the same operation will be performed uniformly.

Up to this point, it was done with numbers, but of course you can do it with letters.

Even at this time, the rules of if, elif, else and == etc. remain the same.

ans = "yes"

print ("Please subscribe to the channel")
if ans == "yes":
    print("thanks")
else:
    print("please")

Please subscribe to the channel
thanks

If, elif, and else will come out frequently in the future, let’s get used to it now.
