Problem

121. Best Time to Buy and Sell Stock

If you’ve been a competitive professional, you’ve probably seen similar problems. Algorithm and data structure for capturing programming contest Especially in this book.

Given an array containing the stock prices for each date. Since only one transaction is allowed, the problem is to design the algorithm that maximizes the profit. Please note that you cannot sell before you buy.

Input: [7,1,5,3,6,4] Output: 5 Explanation: Buy on day 2 (price = 1) and sell on day 5 (price = 6), profit = 6-1 = 5. Not 7-1 = 6, as selling price needs to be larger than buying price.

You can get maximum profit by buying when the stock price is 1 and selling when the stock price is 6. And it returns 5 which is the profit at that time.

Input: [7,6,4,3,1] Output: 0 Explanation: In this case, no transaction is done, i.e. max profit = 0.

In this case, there are no profitable combinations, so 0 is returned.

Solution

As an easy-to-understand thing, I wondered if there was a way to lick the given array from the front, compare the minimum value with the maximum value, and update it.

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        mx, mn = 0, float('inf')
        for price in prices:
            mn = min(mn, price)
            pro = price-mn
            mx = max(mx, pro)
        return mx
