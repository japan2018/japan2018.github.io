There are only 3 basic programs! ?

No matter what language you use, there are only three basics you need to write a program.

| Basics | Description: | |:——————–|:————————– ———–| | Action (control statement) | Take a step forward | | Judgment (conditional sentence) | Do you hit the wall? | | Repeat | Repeat a certain number of times, repeat forever |

Writing like a program looks like this. When the program hits a wall, it turns to the right and keeps going until then. It will be a program that will last forever because it repeats forever. Yes, this is the end of studying the basics of the program.

Repeat forever{
Hit the wall?
Yes: Turn your body to the right
No: Go forward
}

Python is basically indented

If you are learning Python, it is necessary to explain only here in advance.

Python has enforced a language structuring called indentation. As a result, it is possible to write programs that are easy to read and not redundant even as a creator. In Python, indent to indicate that the block of the program matched the condition. In the following program, if the condition is “31st December in 2020”, the message “Because it is the end of the year, the next day is 1st January 2021” is displayed, and the year is incremented by 1 and the month is set to 1 The program to reset.

if year == 2020 and month == 12 and day == 31:
    print ("Because it is the end of the year, the next day will be January 1, 2021")
    year += 1
    month = 1
    day = 1

If you match the conditions if you write as follows, the message “Because it is the end of the year, the next day will be January 1, 2021” will be displayed and the year will be incremented by 1 regardless of whether it matches or not. This is a program in which the day is reset to 1.

if year == 2020 and month == 12 and day == 31:
    print ("Because it is the end of the year, the next day will be January 1, 2021")
year += 1
month = 1
