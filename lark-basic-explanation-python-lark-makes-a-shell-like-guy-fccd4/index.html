The library you are using Lark-a modern parsing library for Python

Creating a grammar rule file

First, create a text file that describes the grammar rules to be passed to the parser generator. In this case, A script is one or more chunks connected by a join, A chunk is one or more sentences connected by a pipe… And so on.

script: [chunk join] chunk

chunk: [(sentence pipe)+] sentence

sentence: command [space option] [(space arg)+] [/\n+/]

command: chars
        | ([chars] subshell [chars])+
option: "-" (chars|subshell)+
arg: chars
    | "'" allchars "'"
    | ([chars] subshell [chars])+

subshell: "(" script ")" [/\n+/]

chars :/[^\+\|\s\(\)']+/[/\n+/]
allchars :/[^']+/[/\n+/]
space :" "
join :[space] "+" [space]
pipe :[space] "|" [space]

Creating a parser

How to create a parser. All you have to do is load the text file you just created and pass it to Lark.

from lark import Lark

with open("Grammar.lark",encoding="utf-8") as grammar:
    LP = Lark(grammar.read(),start="script")

In the second argument start= of Lark(), write the outermost component (probably).

Try parsing rand -s3 a b c + say def | replace e E with this parser. You can use the pretty() method of the parsed result tree for high readability, so use this.

tree = LP.parse("rand -s3 a b c + say def | replace e E")
print(tree.pretty())

Creating a transformer

Now, let’s create a part that executes processing according to the state of the analyzed tree.

from lark import Transformer

class mytransformer(Transformer):
    def __init__(self): # Initialize variables used appropriately
        self.var1 = None
        self.var2 = []

    def sentence(self,tree): # If you take the second argument, you can use a tree deeper than sentence!
        print("sentence") # example

    def command(self,tree):
        print("command")

    def option(self,tree):
        print("option")

Processing such as arg and chunk has been omitted. The sentence, command, option, etc. defined in mytransformer are called depth-first. So in this example command -> option -> sentence and so on.

How to use a transformer,

tree = LP.parse("rand -s3 a b c + say def | replace e E")
mytransformer().transform(tree)
