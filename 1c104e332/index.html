Commonly used libraries

Next, I tried to identify frequently used libraries.

Sequence related

In python, the pop and append of the array are slow, so if you get an incorrect answer at the execution time, describe as follows and use the deck (double-ended queue).

from collections import deque
l = deque
l.append(1)
l.pop()

Next, the story when copying the array

# If the array is one-dimensional
list2 = list1[:]
# For 2D and above
import copy
list2 = copy.deepcopy(list1)

In the case of one dimension, if “list2=list1” is set, list1 is rewritten if list2 itself is rewritten because of passing by reference. In the case of 2 dimensions or more, deep copy is used because it cannot be described like this.

data structure

Heaps (priority queues) are often used in competitive professionals. (For example, when calculating the median) The heap has the following properties

  • Addition (push) and removal (pop) of elements can be done in O(logN) time
  • Minimum value (maximum value) search can be done with O(1)

Since the specification is such that the minimum value comes to the top, if you want to bring the maximum value to the top, multiply by -1 and save it in the heap.

You can handle the heap as follows.

import heapq
heap = []
heapq.heappush(heap, 1)
heapq.heappop(heap)

dictionary

When using a normal dictionary in python, it is necessary to check if a value corresponding to the key exists. The default dict saves you such trouble. (I feel that there are many scenes to use even if it is not a competitive professional)

Use like

from collections import defaultdict
d = defaultdict(int)
If you want to set the default when there is no # key, write it like this
d = defaultdict(lambda:'Initial Value')
