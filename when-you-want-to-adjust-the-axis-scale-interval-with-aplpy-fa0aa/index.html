When you want to adjust the axis scale interval with APLpy

Feb 12, 2020 Python Astronomy APLpy 

Note that I was a little addicted to the method of adjusting the axis scale interval with APLpy.

For example, if you want to adjust the display interval of the main scale on the x-axis,

fig.ticks.set_xspacing(0.0005)

You can do it. However, the argument here must be a multiple of the minimum unit of label format. Now

fig.ticklabels.set_xformat(hh:mm:ss.ss)

If so, the minimum unit is 0.001s or 0.015 arcsec, and therefore 0.015/3600 degree. Based on this, all you have to do is to put n*0.015/3600 into set_xspacing, where n is an integer.
