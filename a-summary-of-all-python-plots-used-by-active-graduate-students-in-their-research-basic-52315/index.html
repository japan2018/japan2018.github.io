def fun(x):
    y = 2*x # write your function here
    return y
X = np.linspace(1,10,num=100) #1 to 10 is divided into num=100 equal columns
Y1 = fun(X)
Y2 = 20*np.sin(X) #y=20sin(x)
plt.plot(X,Y1,color='b',label='your original')#Plot Y1
plt.plot(X,Y2,color='r', label='sin')#Plot Y2
plt.legend(loc='lower right',prop={'size':15})#This is the legend

The size can be changed with prop={‘size’:15}. The position of the legend can be changed with loc='lower right’. It can be placed at almost any position as shown below. Click here for details

A sample looks like this. You can attach a legend with plt.legend().

import matplotlib.pyplot as plt # actually plot
import numpy as np # Data cleanup
def fun(x):
    y = 2**x # write your function here
    return y

X = np.linspace(1,10,num=100) #1 to 10 is divided into num=100 equal columns
Y1 = fun(X)
Y2 = np.exp(X)
plt.plot(X,Y1,color='b',label=r'$y=2^x$')#plot
plt.plot(X,Y2,color='r', label=r'$y=e^x$')
plt.legend(loc='lower right',prop={'size':18}) #Add legend
plt.xlabel('xaxis',size='20') #add label to x-axis, size='to whatever size'
plt.ylabel('yaxis',size='20') # Add label to y-axis
plt.xlim((0,10)) #x range: 0-10
plt.ylim((0,100)) #y range: 0-20
plt.tick_params(labelsize=15) #labelsize=Your favorite size

This is an option to make it a dashed line or display data points instead of just a line.

  • linestyle='your style’

To use. Here’s an example.

Y5=5*X
