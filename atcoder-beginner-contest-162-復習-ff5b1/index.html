In this problem, we consider a gcd with a combination of $K^N$, so you can see that counting all cannot be done*. Therefore, let’s think about how many values of **gcd** are. Also, since the possible values of gcd are 1~k from the definition of gcd, we search for the sequence $A_1,A_2,…,A_n$ where gcd is x($1\leqq x\leqq k$). Here, the **necessary and sufficient condition for gcd to be a multiple of x is **$A_1,A_2,…,A_n$ is a multiple of x**, so $(k//x)^ There are n$ streets. However, **necessary and sufficient condition for gcd to be exactly x** is a multiple of x and not 2x,3x,…**. Therefore, to find the number of cases where gcd becomes x, we must subtract the cases where gcd becomes 2x, 3x, … from $(k//x)^n$ ways. However, if we move x by 1 → k, we will check if 2x, 3x, … is less than or equal to k and subtract the number in that case. **2x,3x, In case of…, the number is not always fixed**. An effective idea here is to consider x in the reverse order of k→1 instead of 1→k**. In this case, it can be calculated by first calculating the number of sequences in which gcd is 2y, 3y, … and subtracting from the candidates of sequences in which gcd is y. Also, since we want to consider y for which $x=l \times y(l \geqq 1)$, it is easy to see that **y is a divisor of x**. From the above, with respect to the number of candidates in the sequence when gcd becomes a divisor $x_1,x_2,…,x_m$ excluding y itself, it is sufficient to subtract only the number of candidates in the sequence in which gcd is y. Implementing these, it becomes as follows. Also, as an answer, we have to find the remainder of $10^9+7$ and specify $10^9+7$ as the third argument of pow when initializing the memo array. It should be noted that the calculation can be speeded up by asking too much.

def make_divisors(n):
    divisors=[]
    for i in range(1, int(n**0.5)+1):
        if n%i==0:
            divisors.append(i)
            if i!=n//i:
                divisors.append(n//i)
    divisors.sort()
    return divisors
