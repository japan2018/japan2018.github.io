What is Python

A programming language developed by Dutch Guido Van Rossum in 1991.

Easy programming with less code Moreover, the code is easy to read.

The use of “artificial intelligence (AI)”, “web development”, and “education field” is spreading.

Advantages of Python

As a basic merit

  • Simple and easy to remember
  • Offside rules with indentation
  • No compilation required
  • Large library

These include:

Simple and easy to remember

The grammar is simple and there is only the minimum necessary A language that is easy to read and write. Because there are not many ways to write it, Code written by others will also be relatively readable.

Offside rules with indentation

It is to specify blocks such as syntax with indentation. This makes the code the same no matter who writes it, The program is easy to write and easy to read.

Offside rules

It is a rule that indicates the range (block) of a lump of sentences by indentation.

No compilation required

The compilation process is often time-consuming, with many errors. Python is an interpreted language, so no need to compile You can check the operation immediately.

#Disadvantages of Python The disadvantages are as follows.

  • Execution speed is slow
  • Indentation required

Python is said to have a particularly slow execution speed among the interpreted languages. It is not suitable for core systems, systems and advanced game development for companies.

In addition, if the indent is not properly made from the offside rule It may cause an error.
